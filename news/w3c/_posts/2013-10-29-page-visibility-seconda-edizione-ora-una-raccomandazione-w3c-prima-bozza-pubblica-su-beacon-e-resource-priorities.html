--- title: "Page Visibility (Seconda Edizione) è ora una Raccomandazione W3C, prima Bozza Pubblica su Beacon e Resource Priorities" date: 2013-10-29 layout: page categories: ---

Il  Gruppo di Lavoro su Web Performance ha pubblicato oggi una Raccomandazione W3C e due prime Bozze di Lavoro:

  • Una Raccomandazione W3C su  Page Visibility (Seconda Edizione). Questa specifica fornisce un mezzo per gli sviluppatori di siti web per determinare da codice lo stato di visibilità corrente della pagina allo scopo di sviluppare applicazioni web ottimali ed efficienti in termini di CPU e potenza.
  • Una prima Bozza pubblica su Beacon. Questa specifica definisce un mezzo interoperabile per sviluppatori di siti web per trasferire dati in maniera asincrona da uno user agent a un web server, in cui lo user agent ha eventualmente l'onere del trasferimento.
  • Una prima Bozza su Resource Priorities. Questa specifica definisce un mezzo per sviluppatori di siti web di dare, da codice, suggerimenti allo User Agent sulla priorità di download di una risorsa. Ciò permetterà allo user agent di gestire con maggiore efficienza l'ordine con il quale le risorse vengono scaricate.
