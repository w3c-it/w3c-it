--- title: "Secure Browsing: Working Draft" date: 2007-03-02 layout: page categories: w3c --- Il Web Security Context Working Group ha rilasciato il First Public Working Draft di Web Security Experience, Indicators and Trust: Scope and Use Cases. Il draft descrive quali tecnologie possono essere usate e quali proposte valutate per produrre un lavoro tecnico di gruppo che produca interfacce più sicure ed usabili, per permettere agli utenti Web di prendere decisioni affidabili e sicure durante la navigazione del Web. Maggiori informazioni sulla pagina della Security Activity. (Archivio News inglese, italiano)
