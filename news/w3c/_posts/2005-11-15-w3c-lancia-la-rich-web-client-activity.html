--- title: "W3C lancia la Rich Web Client Activity" date: 2005-11-15 layout: page categories: --- W3C è lieto di annunciare il lancio della Rich Web Client Activity per lo sviluppo delle Web Application lato-client. Il Web APIs Working Group presieduto da Robin Berjon (Expway) documenterà e costruirà una applicazione standard per la programmazione delle interfacce (APIs) quale XMLHttpRequest, un componente AJAX. Il Web Application Formats Working Group presieduto da Art Barstow (Nokia) svilupperà linguaggi dichiarativi per Web Application, costruendoli sulle tecnologie esistenti sul mercato, ove possibile. Il Compound Document Formats Working Group presieduto da Kevin Kelly (IBM) si trasferisce a questa Activity, continuando lo sviluppo di framework per documenti combinati, come i profili che utilizzano gli attuali formati W3C. La partecipazione a questi gruppi è aperta ai Membri W3C. (Archivio News inglese, italiano)
