--- title: "Pubblicati Last Call di XQuery, XSLT e XPath" date: 2003-11-12 categories: ---

L' XML Query Working Group e l' XSL Working Group hanno pubblicato nove Working Draft, sei dei quali sono nello stato di Last Call fino al 15 febbraio 2004. Si invita a inviare commenti su tutti i documenti. Per ulteriori informazioni: XML home page.

 

I ducumenti pubblicati sono:

  • XQuery 1.0: An XML Query Language - Last Call
  • XSL Transformations (XSLT) Version 2.0 - Last Call
  • XML Path Language (XPath) 2.0 - Last Call
  • XQuery 1.0 and XPath 2.0 Data Model - Last Call
  • XQuery 1.0 and XPath 2.0 Functions and Operators - Last Call
  • XQuery 1.0 and XPath 2.0 Formal Semantics
  • XSLT 2.0 and XQuery 1.0 Serialization - Last Call
  • XML Query (XQuery) Requirements
  • XML Query Use Cases
