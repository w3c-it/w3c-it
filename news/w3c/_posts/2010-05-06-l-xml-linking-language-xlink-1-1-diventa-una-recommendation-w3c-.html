--- title: "L' XML Linking Language (XLink) 1.1 diventa una Recommendation W3C " date: 2010-05-06 layout: page categories: w3c ---

L' XML Core Working Group ha pubblicato una Recommendation W3C su XML Linking Language (XLink) Version 1.1. Questa specifica definisce l' XML Linking Language (XLink) Version 1.1, che permette l'inserimento in documenti XML di elementi che consentono di creare e descrivere link tra risorse. Si utilizza la sintassi XML per creare strutture che possono descrivere link simili agli hyperlink semplici e unidirezionali dell' HTML di oggi, così come altri link più sofisticati. Changes from XLink 1.0 includono: xlink:type non è più necessario per i collegamenti semplici, sono usati IRI invece degli URI, e la specifica include un XML Schema di esempio e grammatiche RELAX NG non-normativi. Per saperne di più si consulti Extensible Markup Language (XML) Activity.
