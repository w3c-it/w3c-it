--- title: "Call for Review: Canonical XML 1.1 Proposed Recommendation" date: 2008-01-29 layout: page categories: w3c --- Lo XML Core Working Group ha pubblicato le Proposed Recommendation di Canonical XML 1.1. La specifica stabilisce un metodo per determinare se due documenti siano identici, o se un'applicazione non abbia modificato un documento, eccetto per le trasformazioni permesse da XML 1.0 e dai Namespace in XML. Canonical XML 1.1 è una revisione di Canonical XML 1.0 progettata per occuparsi dei problemi relativi ereditarietà degli attributi nel namespace di XML quando si rendono canonici sottoinsiemi di documenti, inclusi i requirement da non ereditare xml:id, e per trattare in modo appropriato xml:base URI path processing. Commenti sono graditi fino al 7 marzo. Ulteriori informazioni su W3C's XML Activity. (Archivio News inglese, italiano)
