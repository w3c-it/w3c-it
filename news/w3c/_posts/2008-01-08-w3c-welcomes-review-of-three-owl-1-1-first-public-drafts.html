--- title: "W3C Welcomes Review of Three OWL 1.1 First Public Drafts" date: 2008-01-08 categories: --- The OWL Working Group has published the First Public Working Draft of three Web Ontology Language (OWL) 1.1 specifications: Structural Specification and Functional-Style Syntax, Model-Theoretic Semantics, and Mapping to RDF Graphs. OWL is used to define Semantic Web vocabularies. Together, these new specifications extend the W3C OWL Web Ontology Language 1.0 with a small but useful set of features that have been requested by users, for which effective reasoning algorithms are now available, and that OWL tool developers are willing to support. The three specifications cover, respectively, the syntax, semantics, and mapping to RDF of OWL 1.1 ontologies. Learn more about the W3C Semantic Web Activity. (Archivio News inglese, italiano)
