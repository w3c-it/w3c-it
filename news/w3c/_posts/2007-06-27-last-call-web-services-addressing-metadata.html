--- title: "Last Call: Web Services Addressing Metadata" date: 2007-06-27 categories: --- Il Web Services Addressing Working Group ha rilasciato un Last Call Working Draft di Web Services Addressing 1.0 - Metadata per preparare la specifica da presentare, per richiedere lo stato di Proposed Recommendation. La specifica è usata per indicare supporto per Web Services Addressing 1.0 mechanisms usando il Web Services Policy 1.5 framework e definisce come esprimere le proprietà WS-Addressing WSDL . I commenti sono benvenuti fino all'11 luglio. Maggiori informazioni sulla pagina dei Web service. (Archivio News inglese, italiano)
