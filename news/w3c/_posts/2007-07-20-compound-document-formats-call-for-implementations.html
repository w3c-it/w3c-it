--- title: "Compound Document Formats: Call for Implementations" date: 2007-07-20 layout: page categories: --- Il Compound Document Formats Working Group ha rilasciato 4 Candidate Recommendation: Compound Document by Reference Framework 1.0, WICD Core 1.0, WICD Full 1.0, e WICD Mobile 1.0. Feedback dagli implementatori è benvenuto fino al 1. dicembre. È disponibile un implementation report ed una test suite è in fase di sviluppo. Il Web Integration Compound Document (WICD, pronunciato come la parola inglese "wicked") è un dispositivo indipendente Compound Document profile basato su XHTML, CSS e SVG. Il draft descrive presentazione, linking e comportamento in fase di navigazione quando documenti multipli sono combinati. Maggiori informazioni sulla Rich Web Clients home page.
