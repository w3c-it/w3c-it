--- title: "Last Call: Accessible Rich Internet Applications (WAI-ARIA) Working Draft" date: 2010-09-16 layout: page categories: w3c ---

Il Protocols and Formats Working Group (PFWG) ha pubblicato un Last Call Working Draft di WAI-ARIA, la specifica tecnica Accessible Rich Internet Applications per rendere accessibili alle persone con disabilità contenuti Web dinamici e interattivi. Il PFWG ha anche pubblicato Working Draft aggiornati della WAI-ARIA User Agent Implementation Guide che offre una guida su come browser e altri user agent dovrebbero esporre le funzioni WAI-ARIA alle API di piattaforme accessibili; WAI-ARIA Authoring Practices che descrive come gli sviluppatori di contenuti Web possono sviluppare ricche applicazioni Web accessibili usando le WAI-ARIA; e WAI-ARIA Primer che fornisce un'introduzione tecnica.

Questi documenti WAI-ARIA sono introdotti nel WAI-ARIA Overview. Sono benvenuti commenti fino al 29 Ottobre 2010. Per maggiori informazioni si consulti WAI-ARIA review announcement for details, e Web Accessibility Initiative (WAI).
