Il formato di file WOFF 1.0 diventa una recommendation W3C

13 dicembre 2012 | Archivio

Il WebFonts Working Group ha pubblicato una recommendation W3C sul formato file WOFF 1.0. Questo documento specifica il formato di packaging per i caratteri nei file WOFF. Questo formato é stato progettato per fornire una compressione di dati carattere leggera e facile da implementare, adatta all'uso con regole CSS di tipo @font-face. Ogni file composto da caratteri con licenza originale nel formato TrueType/OpenType/OpenFont può essere compresso nel formato WOFF ad uso Web. I programmi utente che lo utilizzano, decodificano il file WOFF in modo che i caratteri restituiti siano visualizzati identicamente a quelli originali. Per saperne di più vai a Fonts Activity.
