--- title: "W3C setta un nuovo Standard per Internazionalizzare i Web Content" date: 2007-04-03 layout: page categories: w3c --- Il World Wide Web Consortium rilascia oggi Internationalization Tag Set (ITS) Version 1.0 come W3C Recommendation. I creatori di contenuti XML possono usare il set ITS di elementi ed attributi per preparare schemi e documenti per la localizzazione e per internazionalizzarli per un pubblico globale che lavora con le disposizioni di documento e che diventa molto più semplice "se state generando un nuovo schema o state lavorando per fissarne uno," ha dichiarato Richard Ishida (W3C). Le implementazioni fornite con DTDs, XML Schema e Relax NG, possono essere usate con vocabolari nuovi o esistenti come XHTML, DocBook, e OpenDocument.
