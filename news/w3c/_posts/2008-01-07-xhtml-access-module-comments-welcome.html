--- title: "XHTML Access Module; Comments Welcome" date: 2008-01-07 layout: page categories: --- The XHTML2 Working Group has published the First Public Working Draft of XHTML Access Module. This document is intended to help make XHTML-family markup languages more effective at supporting the needs of the accessibility community. It does so by providing a generic mechanism for defining the relationship between document components and well-known accessibility taxonomies. Learn more about the HTML Activity.
