--- title: "Last Call: Mobile Web Best Practices" date: 2006-01-13 layout: page categories: --- Il Mobile Web Best Practices Working Group ha rilasciato il Last Call Working Draft di Mobile Web Best Practices 1.0. Il draft descrive come produrre Web content e Web site dedicati a dispositivi mobili e small-screen. Scritto per tutti i partecipanti della catena distributiva del mobile, il documento è progettato per migliorare la user experience. I commenti sono benvenuti fino al 17 febbraio. Maggiori informazioni sulla pagina della W3C Mobile Web Initiative, sforzo congiunto di fornitori di authoring tool, content provider, produttori di mocrotelefoni, browser vendor ed operatori del mobile.
