--- title: "Last Call: Web Security Context: User Interface Guidelines" date: 2009-02-26 layout: page categories: w3c --- Il Web Security Context Working Group ha pubblicato la seconda Last Call Working Draft di Web Security Context: User Interface Guidelines. Questa specifica si occupa delle decisioni di fiducia che gli utenti devono fare online e dei modi per supportarli nel prendere decisioni sicure e informate quando possibile. Per raggiungere questo obiettivo, questa specifica include raccomandazioni sulla presentazione delle informazioni sull'identità da parte degli user agent nel Web. Include anche raccomandazioni per la gestione degli errori nei protocolli di sicurezza. Questa seconda Last Call Working Draft incorpora feedback raccolti durante il periodo della prima Last Call, sia da parte del pubblico che da parte degli implementatori che hanno partecipato al Working Group. I commenti sono bene accetti fino al 19 marzo 2009. Ulteriori informazioni su Security Activity. (Archivio News Inglese, Italiano)
