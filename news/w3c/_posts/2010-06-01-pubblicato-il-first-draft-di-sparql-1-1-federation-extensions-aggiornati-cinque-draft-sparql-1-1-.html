--- title: "Pubblicato il First Draft di SPARQL 1.1 Federation Extensions; aggiornati cinque Draft SPARQL 1.1 " date: 2010-06-01 categories: ---

Lo SPARQL Working Group ha pubblicato un First Public Working Draft di SPARQL 1.1 Federation Extensions, che definisce le estensioni allo SPARQL Query Language per consentire l'esecuzione di query SPARQL distribuite. Il gruppo ha inoltre pubblicato 5 aggiornamenti, elencati di seguito. Il gruppo è in attesa di feedback, in particolare sulle open issues identificate in ogni documento. Per saperne di più si consulti il Semantic Web.

  • SPARQL 1.1 Query aggiunge al linguaggio di query SPARQL un supporto per aggregati, subquery, espressioni con proiezioni, e la negazione .
  • SPARQL 1.1 Update definisce un linguaggio per l'aggiornamento di grafi RDF.
  • SPARQL 1.1 Service Description definisce un vocabolario e il meccanismo di scoperta per descrivere le potenzialità di un endpoint SPARQL.
  • SPARQL 1.1 Uniform HTTP Protocol for Managing RDF Graphs descrive l' uso del protocollo HTTP per la gestione di grafi RDF con nome su un server HTTP.
  • SPARQL 1.1 Entailment Regimes definisce le condizioni alle quali query SPARQL  possono essere usate con i regimi di implicazione come RDF, RDF Schema, OWL, o RIF.
