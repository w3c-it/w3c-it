--- title: "Last Call: CSS3 Paged Media e CSS Print Profile" date: 2003-12-18 layout: page categories: w3c ---

Il CSS Working Group ha pubblicato due parti del linguaggio Cascading Style Sheets (CSS) come Last Call Working Draft. Si attendono commenti entro il 31 gennaio 2004. CSS3 Paged Media Module aggiunge paginazione, margini, intestazioni e piè di pagina, riferimenti incrociati con numeri di pagina. CSS Print Profile lavora con XHTML-Print per stampare su dispositivi di basso costo. Per ulteriori informazioni: CSS home page.
