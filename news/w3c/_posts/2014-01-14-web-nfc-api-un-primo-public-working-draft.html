--- title: "Web NFC API è un Primo Public Working Draft" date: 2014-01-14 categories: ---

Il NFC Working Group ha pubblicato un PrimoPublic Working Draft del Web NFC API. Near Field Communication (NFC) è uno standard internazionale (ISO/IEC 18092) che specifica un interfaccia e un protocollo per semplici interconnessioni wireless di dispositivi accoppiati vicini che funzionano a 13.56 MHz. NFC abilita le comunicazioni wireless tra due dispositivi a distanza ravvicinata, generalmente meno di pochi centimetri, secondo tre gruppi di scenari; tenendo un dispositivo vicino a un tag wireless per scambiare informazioni o dati digitali, tenendo due dispositivi l'uno vicino all'altro per scambiare alcune informazioni o dati tra loro, facendo pagamenti tenendo un telefono cellulare vicino al punto del terminale di vendita invece che passare smart card.
