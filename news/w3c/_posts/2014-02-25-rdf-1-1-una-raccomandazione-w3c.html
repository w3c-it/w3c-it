--- title: "RDF 1.1 è una Raccomandazione W3C" date: 2014-02-25 layout: page categories: ---

L' RDF Working Group oggi ha pubblicato un set di otto Raccomandazioni su Resource Description Framework (RDF):

  • “RDF 1.1 Concetti e Sintassi Astratta” definisce una sintassi astratta (un modello di dati) che serve a collegare tutte le lingue e le specifiche basate su RDF. La sintassi astratta ha due strutture di dati fondamentali: i grafi RDF sono set di soggetto-predicato-oggetto tripli, dove gli elementi possono essere IRI, nodi vuoti o datatyped letterali. Essi sono usati per esprimere descrizioni di risorse. I set di dati RDF sono usati per organizzare collezioni di grafi RDF e comprendono un grafo di default e zero o più grafi con nome.
  • “RDF 1.1 Semantics” descrive una semantica precisa per il Resource Description Framework 1.1 e RDF Schema, e definisce un certo numero di distinti regimi implicanti e modelli corrispondenti di implicazione.
  • “RDF Schema 1.1″ fornisce un vocabolario data-modelling per dati RDF. RDF Schema è un'estensione del vocabolario di base RDF.
  • “RDF 1.1 Turtle: definisce una sintassi testuale per RDF chiamata Turtle che consente ad un grafo RDF di essere completamente scritto in un formato di testo compatto e naturale, con abbreviazioni per modelli e datatype di uso comune. Turtle fornisce livelli di compatibilità col formato N-Triples così come con il modello di sintassi triplo della Raccomandazione SPARQL W3C.
  • “RDF 1.1 TriG RDF Lingua Dataset” definisce una sintassi testuale per RDF chiamata TriG che consente ad un set di dati RDF di essere completamente scritto in un formato di testo compatto e naturale, con abbreviazioni per modelli e datatype di uso comune. TriG è un'estensione del formato Turtle.
  • “RDF 1.1 N-Triples” è un formato di testo normale, line-based, per la codifica di un grafo RDF.
  • “RDF 1.1 N-Quads” è un formato di testo normale, line-based, per la codifica di un set di dati RDF.
  • “RDF 1.1 XML Syntax” definisce una sintassi XML per RDF chiamata RDF/XML in termini di Namespaces in XML, l'XML Information Set e XML Base.

Inoltre, il Gruppo di Lavoro ha anche pubblicato quattro Note del Gruppo di Lavoro:

  • “RDF 1.1 Primer” fornisce un'introduzione a livello di tutorial all'RDF 1.1.
  • I documenti RDF 1.1 Concetti, Semantica, Schema, e Sintassi XML sostituiscono la famiglia RDF di Raccomandazioni pubblicate nel 2004. “Cosa c'è di nuovo nell' RDF 1.1″ fornisce un riepilogo dei cambiamenti tra le due versioni dell'RDF.
  • “RDF 1.1: Sulla Semantica dei Set di Dati dell' RDF” presenta alcuni problemi da affrontare quando si definisce una semantica formale per i set di dati, così come sono stati discussi nel Gruppo di Lavoro RDF 1.1.
  • “RDF 1.1 Casi di Test” elenca le suite di test e i report di implementazione per la Semantica RDF 1.1 così come i vari formati di serializzazione.
