--- title: "UAAG 2.0 Requirements: Working Draft" date: 2007-10-31 layout: page categories: --- L'User Agent Accessibility Guidelines (UAAG) Working Group ha rilasciato il First Public Working Draft di User Agent Accessibility Guidelines 2.0 Requirements, che definisce un nuovo lavoro pianificato sulla seconda generazione di UAAG. UAAG fornisce una guida per la progettazione di browser Web, media player, tecnologie assistive ed altri 'user agent' per rendere accessibili ed incrementare l'accessibilità dei Web content per persone con disabilità. UAAG è parte di una serie di linee guida sull'accessibilità descritte in Essential Components of Web Accessibility. Maggiori informazioni sulla pagina UAAG Overview e sulla WAI home page.
