--- title: "Note: Device Description Repository Requirements 1.0" date: 2007-12-17 categories: --- The Mobile Web Initiative Device Description Working Group has published the Group Note of Device Description Repository Requirements 1.0. This document describes the use cases for a Device Description Repository (DDR). Each use case is analyzed in order to determine the behavior expected of a DDR in order to realize it. These expected behaviors are captured as high-level requirements, which when normalized across all use cases, lead to a discrete set of DDR requirements.
