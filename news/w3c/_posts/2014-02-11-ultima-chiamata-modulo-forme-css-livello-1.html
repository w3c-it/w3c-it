--- title: "Ultima Chiamata: Modulo Forme CSS Livello 1" date: 2014-02-11 categories: ---

Il Cascading Style Sheets (CSS) Working Group ha pubblicato un'Ultima CHiamata per lo Working Draft su Modulo Forme CSS Livello 1. Le Forme CSS descrivono forme geometriche da usare nei CSS. Per il Livello 1, le Forme CSS possono essere applicate al fluttuare. Una forma cerchio fluttuante farà in modo che il contenuto inline si avvolgerà attorno al cerchio  invece che al box di delimitazione dell'oggetto fluttuante. Il CSS è un linguaggio di descrizione del rendering di documenti strutturati (come HTML e XML) su schermo, carta, discorsi, etc. I commenti sono benvenuti fino al 4 marzo. Per saperne di più circa la Style Activity.
