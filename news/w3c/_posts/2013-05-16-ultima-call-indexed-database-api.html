--- title: "Ultima Call:Indexed Database API" date: 2013-05-16 layout: page categories: ---

Il Gruppo di Lavoro su Web Applications  ha pubblicato un'ultima Call per la bozza di Lavoro su Indexed Database API. Questo documento definisce API per la creazione di un database di record contenenti valori semplici e oggetti gerarchici. Ogni record consiste di una chiave e più valori. Inoltre, il database mantiene gli indici dei record che conserva. Uno sviluppatore di applicazioni usa direttamente un API per localizzare i record sia a partire dalla chiave sia usando un indice. Un linguaggio di interrogazione può poi essere  sovrapposto a questa API. Un database indicizzato può essere implementato usando una struttura dati persistente di tipo B-tree. I commenti sono benvenuti fino al 7 Giugno. Per saperne di più vai a  Rich Web Client Activity.
