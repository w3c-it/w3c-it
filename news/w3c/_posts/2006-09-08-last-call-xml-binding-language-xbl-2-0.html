--- title: "Last Call: XML Binding Language (XBL) 2.0" date: 2006-09-08 categories: --- Il Web Application Formats Working Group ha rilasciato un Last Call Working Draft di XML Binding Language (XBL) 2.0, una tecnologia per estendere l'apparenza e il comportamento di elementi nei formati Web come HTML. I commenti sono benvenuti fino al 7 dicembre. Con XBL, gli elementi possono essere mappati per script, event handlers, CSS, e modelli di contenuti più complessi. I contenuti possono essere riordinato ed impacchettati in modo tale che complessi stili CSS possano essere applicati ad un semplice markup HTML o XHTML. XBL può essere utilzzata sia per implementare nuove interfacce DOM, sia con altre specifiche, per implementare tag set arbitrari come congegni.
