--- title: "Secure Browsing: Working Draft" date: 2007-05-25 layout: page categories: --- Il Web Security Context Working Group ha rilasciato un Working Draft aggiornato di Web Security Experience, Indicators and Trust: Scope and Use Cases. Il lavoro del gruppo punta a realizzare una interfaccia sicura ed usabile per garantire agli utenti Web di fare scelte sicure ed affidabili. Questo draft valuta l'utilizzo delle caratteristiche di sicurezza odierne e le elabora seguendo il progetto del lavoro tecnico del gruppo. Maggiori informazioni sulla pagina della Security Activity.
