--- title: "Working Draft: XML Schema 1.1 Structures" date: 2006-03-30 layout: page categories: w3c --- L'XML Schema Working Group ha rilasciato un Working Draft aggiornato di XML Schema 1.1 Part 1: Structures. XML Schema definisce vocabolari di markup condivisi, la struttura di documenti XML che utilizzano questi vocabolari e fa in modo di agganciare le semantiche associate con essi. Questo draft apporta cambiamenti a XML 1.1, union types, context, forme canoniche dei valori, Simple Type Definition e alla gestione degli spazi bianchi. Maggiori informazioni sulla XML home page.
