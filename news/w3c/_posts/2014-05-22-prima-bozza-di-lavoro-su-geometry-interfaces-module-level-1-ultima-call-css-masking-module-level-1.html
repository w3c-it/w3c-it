--- title: "Prima Bozza di Lavoro su Geometry Interfaces Module Level 1, Ultima Call: CSS Masking Module Level 1" date: 2014-05-22 layout: page categories: ---

Il Cascading Style Sheets (CSS) Working Group e l'SVG Working Group hanno pubblicato oggu due documenti:

  • Una prima Bozza pubblica di Lavoro su Geometry Interfaces Module Level 1. Questa specifica descrive diverse interfacce geometriche per la rappresentazione di punti,quad, rettangoli e matrici di trasformazione. Le interfacce SVG stanno sostituendo le interfacce in genere in favore delle più comuni interfacce SVG, Canvas 2D Context e Trasformazioni CSS.
  • Una ultima Call per una Bozza di Lavoro su CSS Masking Module Level 1. CSS Masking fornisce due modalità per nasocndere parzialmente o totalmente elementi visuali: masking e clipping. Il Masking descrive come usare un altro elemento grafico o un'immagine  come luminanza o maschera tipo alfa. Il Clipping descrive la porzione visibile degli elementi visuali. La regione può essere descritta usando alcuni elementi grafici SVG  o forme di base. Qualunque cosa al di fuori della regione non viene rappresentata. I commenti sono benvenuti entro il 19 Giugno 2014.

Per saperne di più vai a  Style Activity  e a Graphics Activity.
