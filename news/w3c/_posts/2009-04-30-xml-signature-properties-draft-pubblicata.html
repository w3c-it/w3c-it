--- title: "XML Signature Properties Draft pubblicata" date: 2009-04-30 layout: page categories: w3c --- L'XML Security Working Group ha pubblicato una Working Draft di XML Signature Properties. Questo documento delinea le proposte per lo standard di XML Signature Properties sintassi e regole di processing ed un namespace associato a queste proprietà. Lo scopo è di permettere la composizione con qualsiasi versione di XML Signature usando l'elemento XML SignatureProperties. Ulteriori informazioni su Security Activity.
