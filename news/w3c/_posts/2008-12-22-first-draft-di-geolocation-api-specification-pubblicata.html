--- title: "First Draft di Geolocation API Specification Pubblicata" date: 2008-12-22 layout: page categories: w3c --- Il Geolocation Working Group ha pubblicato la First Public Working Draft di Geolocation API Specification. Questa specifica definisce un'API che fornisce accesso tramite script a informazioni su posizione geografica associata al dispositivo host. L'API definisce un interfaccia di alto livello per infomazioni sulla posizione in base al dispositivo host, come latitudine e longitudine. L'API in se è agnostica per quanto riguarda le fonti di informazione sulla posizione. Fonti tipiche di informazioni di posizione includono Global Positioning System (GPS) e posizione inferita da segnali di rete come indirizzo IP, RFID, indirizzo MAC WiFi e Bluetooth, e ID delle celle GSM/CDMA. Ulteriori informazioni su Ubiquitous Web Applications Activity. (Archivio News Inglese, Italiano)
