Pubblicato il Draft Using WAI-ARIA in HTML

14 Febbraio 2013

L'HTML Working Group ha pubblicato il Primo Working Draft Pubblico Using WAI-ARIA in HTML. Questo documento è una guida pratica per sviluppatori su come aggiungere informazioni sull'accessibilità ad elementi HTML usando la specifica Accessible Rich Internet Applications [WAI-ARIA], che definisce un modo per creare contenuti e applicazioni Web più accessibili a persone con disabilità. Questo documento illustra come usare WAI-ARIA in [HTML5], il che aiuta particolarmente nella creazione di contenuto dinamico e di controlli avanzati di interfacce utente sviluppati con Ajax, HTML, JavaScript, e simili tecnologie. Per saperne di più vai a HTML Activity.
