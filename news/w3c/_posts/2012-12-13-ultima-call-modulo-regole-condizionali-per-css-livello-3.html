Ultima call: Modulo Regole Condizionali per CSS Livello 3

13 dicembre 2012 | Archivio

Il Cascading Style Sheets (CSS) Working Group ha pubblicato la bozza di lavoro 'ultima call' per il Modulo Regole Condizionali per CSS Livello 3. Questo modulo contiene le caratteristiche di CSS per elaborare parti di fogli di stile in modo condizionale, condizionatamente alle capacità del processore o al documento cui il foglio di stile si applica. Esso include ed estende le funzionalità del CSS di livello 2, che è costruito sul CSS di livello 1. Le principali estensioni rispetto al livello 2 riguardano la possibilità di annidare certe at-rules entro ‘@media’, e l'aggiunta della regola ‘@supports’ per l'elaborazione condizionale. I commenti sono benvenuti fino al 10 gennaio. Per saperne di più vai a Style Activity.
