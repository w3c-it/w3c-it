Pubblicato il Drafts Streams API, HTML Templates

14 Febbraio 2013

Il Web Applications Working Group ha pubblicato due First Public Working Drafts oggi:

  • Streams API, che fornisce un'API per rappresentare i dati binari nelle applicazioni Web come uno Stream di oggetti, oltre che come costruire e leggere tali contenuti da programma. Questa specifica definisce gli oggetti da usare nelle applicazioni Web aventi thread per la lettura sincrona di uno Stream. Questa API è progettata per essare usata insieme ad altre API e altri elementi sulla piattaforma Web, in particolare: File, XMLHttpRequest, postMessage e Web Workers.
  • HTML Templates, che descrive un metodo per dichiarare sottoalberi DOM inerti in HTML e gestirli per istanziare frammenti di documento con identici contenuti.

Per saperne di più vai a Rich Web Client Activity.
