--- title: "Accessible Way-Finding Using Web Technologies: Simposio Online Call for Papers" date: 2014-09-17 layout: page categories: w3c ---

Il Research and Development Working Group (RDWG) terrà un simposio on-line il 3 dicembre 2014 per esplorare le tecnologie, le applicazioni, i concetti e le soluzioni che aiutano le persone con disabilità a orientarsi, esplorare e navigare attraverso il mondo fisico.  La Call for Papers è aperta fino al 20 October 2014. Per saperne di più vai a Accessible Way-Finding Using Web Technologies Symposium e a  Web Accessibility Initiative (WAI).
