Pubblicato il Draft CSS Overflow Module Livello 3

18 Aprile 2013

Il Cascading Style Sheets (CSS) Working Group ha pubblicato un Working Draft sul CSS Overflow Module Livello 3. Questo modulo contiene le caratteristiche dei CSS relative ai nuovi meccanismi di gestione dell'overflow nei visual media (ad es.: schermo o carta). Nei media interattivi, descrive le caratteristiche che consentono di gestire l'overflow, dovuto alla dimensione fissa di un container, con la paginazione (mostrando una pagina alla volta). Inoltre descrive le caratteristiche, applicabili a tutti i visual media, che consentono di distribuire i contenuti di un elemento su frammenti multipli, permettendo ai contenuti di fluire attraverso regioni multiple o di avere stili differenti per differenti frammenti. Per saperne di più vai al link Style Activity.
