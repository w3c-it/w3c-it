--- title: "XSLT 2.0, XML Query e XPath 2.0 sono W3C Candidate Recommendation" date: 2005-11-03 layout: page categories: --- W3C è lieto di annunciare otto nuove Candidate Recommendation per XSLT, XML Query e XPath. I commenti sono benvenuti fino al 28 febbraio. XSLT trasforma i documenti in differenti markup o formati. Importante per database, motori di ricerca e object repository, XML Query può effettuare ricerche, query e join tra collezioni di documenti. Entrambe, XSLT 2 e XQuery utilizzano espressioni XPath ed operano su istanze XPath Data Model. Maggiori informazioni sulla press release e sulla XML home page. (Archivio News inglese, italiano)
  • XSL Transformations (XSLT) Version 2.0:
    Trasforma istanze data model (XML and non-XML) in altri documenti, inclusi XSL-FO per la stampa
  • XQuery 1.0: An XML Query Language:
    Una sintassi XML-aware per ricercare collezioni di dati strutturati e semi-strutturati, sia localmente che nel Web
  • XML Syntax for XQuery 1.0 (XQueryX):
    Una rappresentazione precisa in XML del linguaggio XML Query, adattabile per elaborazione e introspezione
  • XML Path Language (XPath) 2.0:
    Sintassi di espressione per indirizzare alle diverse parti dei documenti XML
  • XQuery 1.0 and XPath 2.0 Data Model (XDM):
    Rappresentazione ed accesso per entrambi i codici XML e non-XML
  • XQuery 1.0 and XPath 2.0 Functions and Operators:
    Funzione che si può richiamare nelle espressioni XPath e le operazioni che si possono effettuare sui tipi di dati XPath 2.0
  • XSLT 2.0 and XQuery 1.0 Serialization:
    Come ottenere i risultati di valutazione XSLT 2.0 e XML Query in XML, HTML, o come semplice testo
  • XQuery 1.0 and XPath 2.0 Formal Semantics:
    Il type system utilizzato in XQuery e XSLT 2 via XPath definito in modo preciso per gli implementatori
  • Pubblicati anche i Working Drafts:
    • XQuery 1.0 and XPath 2.0 Full-Text:
      Semplifica il recupero full-text per XPath, XSLT e XML Query
    • XQuery 1.0 and XPath 2.0 Full-Text Use Cases:
      Esempi di ricerche full-text tra collezioni di data model
