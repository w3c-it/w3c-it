--- title: "Prime Bozze Pubbliche: W3C HTML Form HTTP Extensions, W3C HTML JSON Form Submission" date: 2014-05-29 layout: page categories: ---

L' HTML Working Group ha pubblicato oggi due prime Bozze di Lavoro.

  • W3C HTML Form HTTP Extensions. Questa è una appendice alla specifica sulle form HTML5 che estende le possibilità di configurare richieste HTTP tramite markup HTML.
  • W3C HTML JSON form submission.  Questa specifica definisce una nuova form che codifica un algoritmo che permette la trasmissione dei dati della form come JSON. Invece di catturare i dati della form essenzialmente come un array di coppie chiave-valore che rappresentano il pane e il burro delle attuali codifiche delle form, fa invece affidamento su una semplice sintassi nome attributo che rende possibile catturare strutture dati ricche direttamente come JSON.
