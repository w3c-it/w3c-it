--- title: "W3C XML Schema Definition Language (XSD): Component Designators Draft Pubblicata" date: 2008-09-10 layout: page categories: --- Il XML Schema Working Group ha pubblicato una Working Draft di W3C XML Schema Definition Language (XSD): Component Designators. Questo documento definisce un sistema di designazione delle XML Schema conmponents. Le Schema components sono i mattoni che costituiscono il modello astratto di dati dello schema. Esse sono specificate da XML Schema Part 1: Structures and XML Schema Part 2: Datatypes. La specifica XSD divide il problema di costruire designatori per schema component in due parti: definire un designatore per uno schema assemblato, e definire un designatore per un particolare schema component o particolari schema components, in base ad uno schema designato. Ulteriori informazioni su Extensible Markup Language (XML) Activity. (Archivio News Inglese, Italiano)
