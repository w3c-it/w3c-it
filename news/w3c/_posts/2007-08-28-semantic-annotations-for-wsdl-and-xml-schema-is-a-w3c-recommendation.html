--- title: "Semantic Annotations for WSDL and XML Schema Is a W3C Recommendation" date: 2007-08-28 layout: page categories: --- Il World Wide Web Consortium ha rilasciato oggi Semantic Annotations for WSDL and XML Schema (SAWSDL) come Recommendation W3C. Con questi attributi, le annotazioni semantiche possono essere aggiunte ai componenti Web Services Description Language (WSDL) per utilizzarle nella classificazione, scoperta, matching, composizione e invocazione dei Web service. Il documento che accompagna la specifica, Usage Guide è una Working Group Note che mostra attraverso esempi come associare le annotazioni semantiche con un Web service. Maggiori informazioni sulla pagina del SAWSDL Working Group e sulla Web service home page. (Archivio News inglese, italiano)
