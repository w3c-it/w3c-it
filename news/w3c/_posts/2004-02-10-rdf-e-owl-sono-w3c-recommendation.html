--- title: "RDF e OWL Sono W3C Recommendation" date: 2004-02-10 layout: page categories: w3c ---

Il World Wide Web Consortium oggi ha pubblicato Resource Description Framework (RDF) e OWL Web Ontology Language (OWL) come W3C Recommendation. RDF è utilizzato per rappresentare informazione e scambiare conoscenza nel Web. OWL è utilizzato per pubblicare e condividere insiemi di termini chiamati ontologie, supportando ricerche avanzate su Web, agenti software e gestione della conoscenza. Per ulteriori informazioni si veda il comunicato stampa in inglese e ledichiarazioni e si visiti la Semantic Web home page.

 

Resource Description Framework (RDF):

  • RDF/XML Syntax Specification (Revised) - Aggiornato per XML, namespace, Infoset, e XML Base
  • RDF Vocabulary Description Language 1.0: RDF Schema - Descrive come usare RDF per costruire vocabolari RDF. Definisce un vocabolario di base e convenzioni per l' uso da parte di applicazioni di Semantic Web
  • RDF Semantics - Teoria matematica formale per ragionare su dati RDF
  • RDF Primer - Un' introduzione per tutti
  • RDF Test Cases - Casi di test eleborabili dalla macchina
  • Resource Description Framework (RDF): Concepts and Abstract Syntax - Sintassi, progetto, obiettivi, concetti, significato di documenti RDF, normalizzazione di caratteri e trattamento di URI reference

Web Ontology Language (OWL):

  • Overview - Una semplice introduzione
  • Guide - Spiega OWL attraverso un esempio esteso. Fornisce un glossario
  • Reference - Una descrizione compatta, informale delle primitive di modellazione di OWL
  • Semantics and Abstract Syntax - Definizione normativa del linguaggio OWL
  • Test Cases - Casi di test che illustrano il corretto utilizzo di OWL, il significato formale di costrutti, e soluzione di problemi. Specifica la conformità
  • Use Cases and Requirements - Scenari di utilizzo, obiettivi e requisiti per un linguagggio di ontologia per il Web
