--- title: "Ultima Call: JSON-LD 1.0 Elaborazione di Algoritmi e API" date: 2013-05-16 layout: page categories: w3c ---

Il Gruppo di Lavoro RDF  ha pubblicato un'ultima Call per la definizione della Bozza di lavoro su JSON-LD 1.0 Processing Algorithms and API. Questa specifica definisce un API ( Application Programming Interface ) e un set di algoritmi per la trasformazione da programma di documenti JSON-LD. Ristrutturare i dati in accordo a tale trasformazione spesso semplifica drasticamente il loro uso. I commenti sono benvenuti fino al 6 Giugno. Per saperne di più vai a Semantic Web Activity.
