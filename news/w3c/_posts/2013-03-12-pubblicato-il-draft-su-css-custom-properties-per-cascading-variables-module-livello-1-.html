Pubblicato il Draft su CSS Custom Properties per Cascading Variables Module Livello 1

12 Marzo 2013

Il Cascading Style Sheets (CSS) Working Group ha pubblicato un Working Draft su CSS Custom Properties per Cascading Variables Module Livello 1. Questo modulo introduce le cascading variables come un nuovo tipo di valore primitivo accettato da tutte le properties CSS e le custom properties per definirle. Per saperne di più vai al link Style Activity.
