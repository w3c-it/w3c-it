--- title: "XMLHttpRequest Object for Ajax: Working Draft" date: 2007-10-26 layout: page categories: ---

Il Web API Working Group ha rilasciato un Working Draft aggiornato di The XMLHttpRequest Object. Componente base di Ajax, l'oggetto XMLHttpRequest è un'interfaccia che permette agli script di compiere le funzioni di un client HTTP, come inviare o caricare dati da un sito Web remoto. Maggiori informazioni sulla pagina della Rich Web Clients Activity. (Archivio News inglese, italiano)
