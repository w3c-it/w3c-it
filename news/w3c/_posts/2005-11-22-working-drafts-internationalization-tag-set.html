--- title: "Working Drafts: Internationalization Tag Set" date: 2005-11-22 layout: page categories: --- L'Internationalization Tag Set Working Group ha pubblicato un primo Public Working Draft dell' Internationalization Tag Set (ITS) ed un Working Draft aggiornato dei suoi requisiti. Organizzato per categorie di dati, questo set di elementi ed attributi supporta l'internazionalizzazione e la localizzazione di schemi e documenti. Le implementazioni sono fornite per DTDs, XML Schema e Relax NG, e per i vocabolari esistenti quali XHTML, DocBook ed OpenDocument. Maggiori informazioni sulla Internationalization home page.
