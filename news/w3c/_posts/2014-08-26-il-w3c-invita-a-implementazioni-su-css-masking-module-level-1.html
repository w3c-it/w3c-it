--- title: "Il W3C Invita a Implementazioni su CSS Masking Module Level 1" date: 2014-08-26 categories: ---

Il Cascading Style Sheets (CSS) Working Group e l' SVG Working Group esortano implementazioni della Candidate Recommendation su CSS Masking Module Level 1. CSS Masking fornisce due mezzi per  nascondere parzialmente o completamente porzioni di elementi visivi: mascheramento e ritaglio. Il Mascheramento descrive come utilizzare un altro elemento grafico o un''immagine come una maschera di luminanza o una alfa-mask. Il Clipping descrive l'area visibile di elementi visivi. La regione può essere descritto utilizzando alcuni elementi grafici SVG o forme di base. Qualsiasi cosa al di fuori di questa regione non viene renderizzata. Il CSS è un linguaggio per descrivere il rendering di documenti strutturati (come HTML e XML) sullo schermo, sulla carta, nel parlato, ecc. Per saperne di piàù vai a Style Activity e a Graphics Activity.
