--- title: "Working Draft: Selectors API" date: 2006-06-14 layout: page categories: w3c --- Il 25 maggio, il Web APIs Working Group ha rilasciato il Primo Public Working Draft di Selectors API. Il draft definisce i metodi per identificare elementi in un documento, con l'intenzione di effettuare script od operazioni Document Object Model (DOM) su essi. I selettori, definiti nella specifica CSS3 Selectors, sono utilizzati per identificare gli elementi. Maggiori informazioni sulla Web APIs Working Group home page. (Archivio News inglese, italiano)
