Il W3C Incoraggia le Implementazioni dell'Efficient XML Interchange (EXI) Profile

16 Aprile 2013

L'Efficient XML Interchange Working Group incoraggia implementazioni della Candidate Recommendation sull''Efficient XML Interchange (EXI) Profile. Questo documento descrive un profilo della specifica EXI 1.0 per dispositivi con limitate capacità di memoria. Per saperne di più vai al link Extensible Markup Language (XML) Activity.
