--- title: "Pubblicata Bozza di Lavoro su WebCrypto Key " date: 2013-08-22 layout: page categories: ---

Il Gruppo di Lavoro Web Cryptography  ha pubblicato una Bozza di Lavoro su WebCrypto Key Discovery. Questa specifica descrive un API JavaScript per scoprire le chiavi crittografiche cosidette, nominative, origin-specific e pre-provisioned in uso con le Web Cryptography API.  Le chavi Pre-provisioned sono chiavi che sono state rese disponibili allo user agent oltre che per mezzo della generazione, anche della derivazione, e tramite funzioni di importazione delle Web Cryptography API. Le chiavi Origin-specific sono chiavi che sono disponibili solo per una origine specifica. Le chiavi nominative sono identificate da un nome che si assume noto all'origine in oggetto e fornito insieme con la sua stessa chiave.Per saperne di più vai a  Security Activity.
