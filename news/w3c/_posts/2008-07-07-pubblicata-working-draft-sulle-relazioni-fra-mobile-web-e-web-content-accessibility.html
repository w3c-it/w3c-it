--- title: "Pubblicata Working Draft sulle relazioni fra Mobile Web e Web Content Accessibility" date: 2008-07-07 layout: page categories: --- Il Mobile Web Best Practices Working Group ed il WAI Education and Outreach Working Group hanno pubblicato una Working Draft aggiornata di Relationship between Mobile Web Best Practices (MWBP) and Web Content Accessibility Guidelines (WCAG). Controlla l'announcement email. I gruppi incoraggiano a cominciare a leggere Web Content Accessibility and Mobile Web: Making a Web Site Accessible Both for People with Disabilities and for Mobile Devices, che mostra come gli obiettivi del progetto per accessibilita' e mobile access si sovrappongano. Un terzo documento, Experiences Shared by People with Disabilities and by People Using Mobile Devices, fornisce esempi di barriere che le persone (senza disabilita') affrontano quando interagiscono con il Web content attraverso dispositivi mobili, e barriere simili per persone con disabilita' che usano computer desktop. Ulteriori informazioni su Mobile Web Initiative e su Web Accessibility Initiative (WAI). (Archivio News Inglese, Italiano)
