--- title: "Pubblicate 2 nuove bozze dal Gruppo di Lavoro System Applications " date: 2013-05-16 layout: page categories: ---

Il Gruppo di Lavoro su System Applications ha pubblicato oggi 2 Bozze di Lavoro:

  • The app: URI scheme. Questa specifica definisce l'app:schemi URI  e regole per de-referenziare una app: URI, che può essere usata per indirizzare risorse dentro un package (esempio, un package di applicazione). Il modello di de-referenziazione si riferisce alla semantica HTTP  per restituire le risorse in un modo simile a una richiesta HTTP GET. Facendo questo permette che lo schema URI possa essere usato con altre tecnologie che si basano su risposte HTTP per funzionare come richiesto, come nel caso XHR.
  • Messaging API. Questa specifica definisce un System Level API che offre una interfaccia semplice per avere accesso ai servizi di messageria mobile. Un uso tipico del Messaging API si ha nell'implementazione di un'applicazione client di messagistica che permette all'utente di inviare messaggi SMS e MMS così come di accedere e gestire i messaggi SMS e MMS ricevuti.

Per saperne di più vai a  Ubiquitous Web Applications Activity.
