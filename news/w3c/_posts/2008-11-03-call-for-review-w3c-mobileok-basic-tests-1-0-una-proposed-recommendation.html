--- title: "Call for Review: W3C mobileOK Basic Tests 1.0 è una Proposed Recommendation" date: 2008-11-03 categories: --- Il Mobile Web Best Practices Working Group ha pubblicato la Proposed Recommendation di W3C mobileOK Basic Tests 1.0.Questo documento definisce l'insieme di test machine-verifiable che forniscono le basi per la conformità di W3C mobileOK Basic ed si basa su W3C Mobile Web Best Practices. I contenuti che superano i test hanno fatto passi avanti per fornire una esperienza più funzionale per l'utente di dispositivi mobili basici le cui capacità corrispondono almeno al Default Delivery Context. I commenti sono bene accetti fino al primo dicembre 2008. Ulteriori informazioni su Mobile Web Initiative. (Archivio News Inglese, Italiano)
