--- title: "Ultima Chiamata: Modulo CSS Background e Bordi Livello 3" date: 2014-02-04 layout: page categories: w3c ---

Il Cascading Style Sheets (CSS) Working Group ha pubblicato un'Ultima Chiamata per un Working Draft su CSS Backgrounds and Borders Module Level 3 con il proposito di aggiornare la precedente Raccomandazione Candidata. Questo modulo sostituisce le due bozze precedenti: CSS3 Backgrounds e CSS3 Border. CSS è un linguaggio per la descrizione del rendering di documenti strutturati (come HTML e XML) su schermo, carta, in discorsi, etc. Questo draft contiene le funzionalità del CSS livello 3 riguardo ai bordi e ai background. Esso comprende ed estende le funzionalità del CSS livello 2, che si basa sul CSS livello 1. Le principali estensioni rispetto al livello 2 sono i bordi consistenti in immagini, box con background multipli, box con angoli arrotondati e box con ombreggiatura. I commenti sono benvenuti fino al 3 Marzo 2014. Per saperne di più circa la Style Activity.
