--- title: "XQuery Scripting Extension and Version 1.1 Requirements: Working Drafts" date: 2007-03-23 layout: page categories: --- Responding to requests from Working Group participants and the wider community, the XML Query Working Group released two First Public Working Drafts. XQuery Scripting Extension 1.0 Requirements describes goals and requirements for making XQuery 1.0 functional as a scripting language. XML Query (XQuery) 1.1 Requirements describes compatibility and functionality requirements for extending XML Query 1.0. Visit the XML home page. (Archivio News inglese, italiano)
