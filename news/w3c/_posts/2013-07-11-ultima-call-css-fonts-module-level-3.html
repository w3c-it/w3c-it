--- title: "Ultima Call: CSS Fonts Module Level 3" date: 2013-07-11 layout: page categories: ---

Il Cascading Style Sheets (CSS) Working Group ha pubblicato l'ultima call per il Working Draft relativa a CSS Fonts Module Level 3. Questo modulo CSS3 indica come specificare le proprietà dei font e come caricare dinamicamente delle font resource. I contenuti di questa specifica sono il risultato del consolidamento del contenuto precedentemente suddiviso nei moduli CSS3 Fonts e CSS3 Web Fonts. La descrizione del caricamento dei font è stata spostata nel modulo CSS3 Font Load Events. Si possono inviare commenti fino al 22 Agosto 2013.
