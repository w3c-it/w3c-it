--- title: "Pubblicato Draft su CSS Box Alignment Module Level 3 " date: 2013-05-14 categories: ---

Il Cascading Style Sheets (CSS) Working Group ha pubblicato un Working Draft su CSS Box Alignment Module Level 3. Questo modulo contiene le caratteristiche di CSS relative all'allineamento delle box e dei loro container nei vari modelli di layout delle CSS box: block layout, table layout, flex layout  e grid layout. Maggiori infromazioni su Style Activity.
