--- title: "Open Mobile Web Test Suite: Call for Contributions" date: 2007-07-30 categories: --- Il Mobile Web Test Suites Working Group sta lanciando una Open Mobile Web Test Suite costruita dalla comunità per descrivere il supporto per le tecnologie nei browser mobile Web disponibili oggi. Gli sviluppatori di Mobile Web possono sottoporre i test case (come descritto nelle submissions guidelines) che illustrano le pratiche di authoring. Le presentazioni contribuiranno a migliorare la comprensione dei limiti degli user agent odierni, ed aiuteranno a tracciare la strada per migliori browser mobile Web futuri. Maggiori informazioni sul Call for Contributions e sulla Mobile Web Initiative home page. (Archivio News inglese, italiano)
