--- title: "Note: Semantic Web Primer per Object-Oriented Software Developers" date: 2006-03-10 categories: --- Il Semantic Web Best Practices and Deployment Working Group ha pubblicato A Semantic Web Primer for Object-Oriented Software Developers come Working Group Note. Prodotta dal Gruppo Software Engineering Task Force, la Nota mostra come i processi di sviluppo possano utilizzare il Semantic Web come una piattaforma per la creazione, la condivisione ed il riuso di un domain model. Vengono mostrati RDF Schema e OWL usati in tandem con linguaggi di mainstream object-oriented. Maggiori informazioni sulla Semantic Web home page. (Archivio News inglese, italiano)
