--- title: "Interest Group Note: RDF e iCalendar Data" date: 2005-09-29 layout: page categories: --- Dan Connolly e Libby Miller, componenti del Semantic Web Interest Group, hanno pubblicato RDF Calendar - an application of the Resource Description Framework to iCalendar Data come Interest Group Note. La Note è un report sullo stato dei lavori per l'integrazione di calendar data con altri dati Semantic Web utilizzati in reti sociali, syndicated content, e multimedia. Maggiori informazioni sulla Semantic Web home page. (Archivio News inglese, italiano)
