--- title: "A MathML for CSS profile: Working Draft" date: 2007-04-27 layout: page categories: w3c --- Il Math Working Group ha pubblicato il First Public Working Draft di A MathML for CSS profile. Questo subset di MathML 3.0 può essere utilizzato per catturare la struttura della formule matematiche in modo particolarmente adattabile per facilitare la formattazione CSS. Coordinato con il lavoro su CSS Level 3, il profilo dovrebbe facilitare l'adozione di MathML nei browser Web e nei CSS formatter. Maggiori informazioni sulla Math home page.
