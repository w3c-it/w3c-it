--- title: "Tre Specifications pubblicate dal Web Applications Working Group" date: 2014-06-12 layout: page categories: w3c ---

Il Web Applications Working Group ha pubblicato oggi tre documenti: 

  • Una prima Public Working Draft relativa a DOM Level 3 KeyboardEvent key Values. Questa specifica definisce i valori per l'attributo KeyboardEvent.key,definito come parte del Document Object Model (DOM) Level 3 Events Specification. L'attributo Key contiene informazioni sui caratteri generati da un key event.
  • Una prima Public Working Draft in merito ai DOM Level 3 KeyboardEvent code Values. Questa specifica definisce i valori per l'attributo KeyboardEvent.code attribute, definito come parte del Document Object Model (DOM) Level 3 Events Specification. Il code value riporta informazioni sul key event che può essere utilizzato per identificare la physical key che deve essere utilizzata dall'utente.
  • Una Working Draft sui UI Events. Questi specifica estende gli eventi e le features definite nel DOM Events Level 3.

Maggiori informazioni su Rich Web Client Activity.
