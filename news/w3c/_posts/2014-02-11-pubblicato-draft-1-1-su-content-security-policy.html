--- title: "Pubblicato Draft 1.1 su Content Security Policy" date: 2014-02-11 layout: page categories: ---

Il Web Application Security Working Group ha pubblicato un Working Draft su Content Security Policy 1.1. Questo documento definisce un sistema di linguaggio usato per dichiarare una serie di restrizioni di contenuto per una risorsa web e un meccanismo per trasmettere il sistema da un server a un client dove il sistema viene applicato. Per saperne di più circa la Security Activity.
