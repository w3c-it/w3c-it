--- title: "Notes: XML Query Requirements and Use Cases" date: 2007-03-23 categories: --- The XML Query Working Group published two Working Group Notes, XML Query (XQuery) Requirements and XML Query Use Cases. The documents are a record of the development of XQuery and its associated specifications. Part of the XML family, the Recommendations are used for data mining, document transformation, and enterprise computing. Visit the XML home page. (Archivio News inglese, italiano)
