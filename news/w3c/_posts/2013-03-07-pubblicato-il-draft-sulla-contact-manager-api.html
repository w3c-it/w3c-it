--- title: "Pubblicato il draft sulla Contact Manager API" date: 2013-03-07 layout: page categories: w3c ---

Il System Applications Working Group ha pubblicato il primo working draft pubblico sulla Contacts Manager API. Questa specifica definisce una System Level API che offre un'interfaccia semplice per gestire i contatti degli utenti salvati nella rubrica del sistema. Un tipico caso d'uso della Contact API è l'implementazione di un'applicazione per gestire la suddetta rubrica. Ulteriori informazioni sulla Ubiquitous Web Applications Activity.
