--- title: "Ultima Call: State Chart XML (SCXML): State Machine Notation for Control Abstraction" date: 2014-05-29 layout: page categories: ---

Il Voice Browser Working Group ha pubblicato una Last Call Working Draft su State Chart XML (SCXML): State Machine Notation for Control Abstraction. Questo documento descrive SCXML, ovvero il “State Chart extensible Markup Language”. SCXML fornisce un generico ambiente di esecuzione  stato-macchina basato su CCXML e Harel State Tables. I commenti sono benvenuti fino al 26 Giugno. Per saperne di più vai a Voice Browser Activity.
