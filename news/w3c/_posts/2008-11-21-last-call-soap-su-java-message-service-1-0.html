--- title: "Last Call: SOAP su Java Message Service 1.0" date: 2008-11-21 layout: page categories: --- Il SOAP-JMS Binding Working Group ha pubblicato la Last Call Working Draft di SOAP over Java Message Service 1.0. Questo documento scpecifica come SOAP debba legarsi ad un sistema di messaggi che supporta Java Message Service (JMS). La specifica aiuta ad assicurare l'interoperabilità fra le implementazioni di diversi Web service. Dovrebbe anche permettere ai clienti di implementare i loro Web service come parte della loro infrastrutturafor, e di farlo interagire con i Web service forniti dai vendor. I commenti sono bene accetti entro il 13 gennaio. Ulteriori informazioni su Web Services Activity. (Archivio News Inglese, Italiano)
