--- title: "Pubblicato il Last Call Working Draft di CSS Media Queries" date: 2002-01-23 layout: page categories: w3c ---

Il CSS Working Group ha rilasciato un Last Call Working Draft di Media queries. Il draft propone un registry di media types per descrivere il tipo di device a cui uno style sheet si applica, e fornisce espressioni per limitare l'ambito di uno style sheet. Commenti sono benvenuti. Disponibile l'home page CSS.
