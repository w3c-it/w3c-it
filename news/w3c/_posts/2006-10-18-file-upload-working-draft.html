--- title: "File Upload: Working Draft" date: 2006-10-18 layout: page categories: --- Il Web API Working Group ha rilasciato il First Public Working Draft di File Upload. Le applicazioni potranno usare questo API per attivare un file selection dialog, con il quale l'utente potrà selezionare uno o più file nel file system locale. Questo permetterà una manipolazione client-side dei contenuti, per esempio per visualizzare un'immagine, o compiere l'analisi di un documento XML da disco. Il gruppo richiede commenti dagli sviluppatori di Web content e browser. Maggiori informazioni sulla pagina della Rich Web Clients Activity. (Archivio News inglese, italiano)
