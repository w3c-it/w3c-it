Last Call: API per Media Resources 1.0

11 Aprile 2013

Il Media Annotations Working Group ha pubblicato la terza Last Call Working Draft sulle API per Media Resources 1.0. Questa specifica definisce una API per accedere all'informazione dei metadati collegati alle media resources sul Web. L'obiettivo generale è di fornire agli sviluppatori un accesso conveniente all'informazione dei metadati registrati in formati differenti. L'API fornisce i mezzi per accedere all'insieme di proprietà dei metadati definiti nella specifica Ontology per Media Resources 1.0. Queste proprietà sono usate come "pivot vocabulary" in questa API. Il nucleo di questa specifica è la definizione delle interfacce API per consultare l'informazione dei metadati in modo sincrono e asincrono. Definisce inoltre le interfacce per tipi strutturati restituiti insieme alla specifica del comportamento di una implementazione API. Sono benvenuti commenti fino al 03 Giugno. Ulteriori informazioni su Video in the Web Activity.
