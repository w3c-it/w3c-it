--- title: "Pubblicate le Bozze su CSS Ruby Module Level 1, e CSS Syntax Module Level 3 " date: 2013-09-20 layout: page categories: ---

Il Gruppo di Lavoro su Cascading Style Sheets (CSS) ha pubblicato oggi due Bozze di lavoro:

  • CSS Ruby Module Level 1. “Ruby” sono brevi sequenze di testo a fianco del testo base, tipicamente utilizzati nei documenti dell'Asia orientale per indicare la pronuncia o per fornire una breve annotazione. Questo modulo descrive il modello di rendering e i comandi di formattazione relativi alla visualizzazione di annotazioni Ruby in CSS. Il CSS è un linguaggio per descrivere il rendering di documenti strutturati (come HTML e XML) sullo schermo, sulla carta, nel parlato, ecc
  • CSS Syntax Module Level 3. Questo modulo descrive, in termini generali, la struttura di base e la sintassi dei fogli di stile CSS. Definisce, in particolare, la sintassi e l'analisi dei CSS - ovvero come trasformare un flusso di byte in un foglio di stile significativo.
