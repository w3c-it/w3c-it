--- title: "Group Note: Known Issues with Canonical XML 1.0 (C14N/1.0)" date: 2006-12-20 layout: page categories: --- L'XML Core Working Group aggiorna la Working Group Note Known Issues with Canonical XML 1.0 (C14N/1.0), che indirizza alcune delle problematiche relative all'eredità degli attributi XML xml:base e xml:id e la Recommendation W3C per Canonical XML Versione 1.0. Maggiori informazioni sulla XML Core home page. (Archivio News inglese, italiano)
