--- title: "Working Draft: State Chart XML (SCXML)" date: 2007-02-21 layout: page categories: --- The Voice Browser Working Group has released an updated Working Draft of State Chart XML (SCXML): State Machine Notation for Control Abstraction. SCXML is an execution environment based on UML Harel State Tables and CCXML . SCXML is a candidate for the control language within VoiceXML 3.0, CCXML 2.0, and the authoring language under development by the Multimodal Interaction Working Group. Visit the Voice Browser home page. (Archivio News inglese, italiano)
