--- title: "XML Schema Patterns for Databinding: Working Drafts" date: 2007-10-31 categories: --- L'XML Schema Patterns for Databinding Working Group ha pubblicato un Working Draft aggiornato di Basic XML Schema Patterns for Databinding Versione 1.0 e Advanced XML Schema Patterns for Databinding Versione 1.0. Il pattern può descrivere rappresentazioni XML 1.0 di strutture dati comunemente usate, indipendenti da particolari linguaggi di programmazione, database o ambienti di modelling. W3C invita a contribuire alla test suite, alla lettura del report di interoperabilità e alla consultazione della pagina dei Web service. (Archivio News inglese, italiano)
