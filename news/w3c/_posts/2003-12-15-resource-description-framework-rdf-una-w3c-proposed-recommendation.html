--- title: "Resource Description Framework (RDF) è una W3C Proposed Recommendation" date: 2003-12-15 layout: page categories: ---

W3C è lieto di annunciare l' avanzamento di Resource Description Framework (RDF) a Proposed Recommendation. Si attendono commenti entro il 19 gennaio. RDF è usato per rappresentare informazione e scambiare conoscenza sul Web. Per ulteriori informazioni: Semantic Web Activity. Il linguaggio RDF è presentato in sei rapporti tecnici:

 

  • RDF/XML Syntax Specification (Revised) - Aggiornato per XML, namespace, Infoset, e XML Base
  • RDF Vocabulary Description Language 1.0: RDF Schema - Descrive come usare RDF per costruire vocabolari RDF. Definisce un vocabolario di base e convenzioni per l' uso da parte di applicazioni di Semantic Web
  • RDF Semantics - Teoria matematica formale per ragionare su dati RDF
  • RDF Primer - Una introduzione per tutti
  • RDF Test Cases - Casi di test machine-processable
  • Resource Description Framework (RDF): Concepts and Abstract Syntax - Sintassi, progetto, obiettivi, concetti, significato di documenti RDF, normalizzazione di caratteri e trattamento di URI reference
