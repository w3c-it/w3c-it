--- title: "Ultima Call: CSS Flexible Box Layout Module Level 1" date: 2014-03-25 layout: page categories: ---

Il  Gruppo di Lavoro su Cascading Style Sheets (CSS) ha pubblicato un'ultima Call per una Bozza di Lavoro su CSS Flexible Box Layout Module Level 1. La specifica descrive una modello di CSS ottimizzato per il design dell'interfaccia utente. Nel modello di layout flessibile, gli elementi figli di un contenitore flessibile possono essere disposti in qualsiasi direzione, e possono "flettere" le loro dimensioni, sia aumentandole per riempire lo spazio inutilizzato sia riducendole per evitare di uscire fuori dal contenitore padre. L'allineamento orizzontale e verticale degli elementi figli può essere facilmente manipolato. L'annidamento di questi contenitori (orizzontali all'interno di verticali, o verticali all'interno di orizzontali) può essere utilizzato per costruire layout in due dimensioni. Il CSS  è un linguaggio per descrivere il rendering di documenti strutturati (sia HTML che XML) nello schermo, su carta , nel parlato etc. I commenti sono benvenuti fino al 22 Aprile 2014.
