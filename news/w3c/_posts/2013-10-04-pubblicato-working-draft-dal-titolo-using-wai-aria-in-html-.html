--- title: "Pubblicato Working Draft dal titolo \"Using WAI-ARIA in HTML\"" date: 2013-10-04 categories: ---

L'HTML Working Group ha pubblicato un Working Draft aggiornato dal titolo Using WAI-ARIA in HTML. Using WAI-ARIA in HTML è una guida pratica per gli sviluppatori su come aggiungere agli elementi HTML informazioni riguardanti l'accessibilità usando la specifica Accessible Rich Internet Applications (WAI-ARIA), che definisce un modo per rendere il contenuto Web e le applicazioni Web più accessibili alle persone con disabilità. E' sviluppato dall'HTML Accessibility Task Force in coordinazione con l'HTML Working Group e il WAI Protocols and Formats Working Group (PFWG). Ulteriori informazioni sulla HTML Activity e la Web Accessibility Initiative (WAI).
