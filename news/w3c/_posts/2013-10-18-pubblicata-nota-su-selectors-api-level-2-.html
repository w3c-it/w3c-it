--- title: "Pubblicata Nota su Selectors API Level 2 " date: 2013-10-18 layout: page categories: w3c ---

Il  Gruppo di Lavoro su Web Applications ha pubblicato una Nota di Gruppo su Selectors API Level 2. I selettori,largamente usati nei CSS, sono pattern che corrispondono a elementi in una struttura ad albero. Le specifiche relative alle Selectors API definiscono metodi per recuperare gli elementi Nodi nel DOM abbinandovi un gruppo di selettori, e per testare se un dato elemento si abbina con un particolare selettore.Spesso si vuole fare un certo tipo di operazioni nel DOM su uno specifico set di elementi del documento. Queste metodiche semplificano il processo di acquisizione e test di elementi specifici, specialmente se comparati con le tecniche più ridondanti definite e usate in passato. Per saperne di più vai a Rich Web Client Activity.
