--- title: "Esempi su Mobile Accessibility: aggiornato Implementing UAAG 2.0" date: 2013-06-06 layout: page categories: ---

Un aspetto dell'accessibilità su dispositivi mobili riguarda come i browser di questi dispositivi supportano le necessità di persone con disabilità. L'accessibilità dei browser Web è trattata in "User Agent Accessibility Guidelines" (UAAG). Lo User Agent Accessibility Guidelines Working Group (UAWG) ha pubblicato oggi un Working Draft aggiornato intitolato "Implementing UAAG 2.0", che spiega come i browser, i media player e atri user agent dovrebbero supportare l'accessiblità per persone con disabilità e come dovrebbero funzionare con tecnologie assistive. L'aggiornamento fornisce una pagina dal titolo Mobile Accessibility Examples from UAAG che mostra come i browser Web che seguono le UAAG sono utili alle persone disabili che navigano da dispositivi mobili. Ulteriori informazioni sul lavoro del W3C WAI collegato alla Mobile Accessibility e sulla Web Accessibility Initiative (WAI).
