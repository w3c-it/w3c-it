--- title: "Best Practices per la Pubblicazione di Linked Data Pubblicato come Primo Working Group Note Pubblico" date: 2014-01-09 layout: page categories: ---

Il Government Linked Data Working Group ha pubblicato un Pirmo Working Group Note  Pubblico sulle Best Practices per la Pubblicazione di Linked Data.Questo documento definisce una serie di best practices progettate per facilitare lo sviluppo e la forniture di dati aperti delle amministrazioni come Linked Open Data. Linked Open Data mette il World Wide Web in un database globale, a volte indicato come “Web of Data”. Usando i Principi dei Linked Data, gli sviluppatori possono interrogare i dati associati provenienti da più fonti contemporaneamente e combinarli senza bisogno che i dati condividano un unico schema comune. Prima degli standard per lo scambio di dati internazionali per i dati sul Web, è stato lungo e difficile costruire applicazioni usando tecniche tradizionali di data management. Poiché i dati di amministrazioni più aperte sono pubblicati su Web, anche le best practices si stanno evolvendo. L'obiettivo di questo documento è quello di compilare le più rilevanti pratiche di data management per la pubblicazione e l'uso di dati di alta qualità pubblicati dalle amministrazioni di tutto il mondo come Linked Open Data.
