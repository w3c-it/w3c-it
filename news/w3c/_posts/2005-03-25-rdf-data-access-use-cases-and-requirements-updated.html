--- title: "RDF Data Access Use Cases and Requirements Updated" date: 2005-03-25 layout: page categories: --- L'RDF Data Access Working Group ha rilasciato un Working Draft aggiornato di RDF Data Access Use Cases and Requirements. La bozza suggerisce come un linguaggio di interrogazione RDF e un protocollo di accesso ai dati possano essere utilizzati nella costruzione di nuove ed usabili Semantic Web applications nei settori del Web publishing, nella gestione delle informazioni personali, nel turismo e nei trasporti. Il gruppo richiede feedback su quali features siano necessarie per una prima versione di SPARQL e su quali debbano essere invece posticipate, con l'obiettivo di facilitarne il successivo sviluppo. Maggiori informazioni sulla Semantic Web home page. (Archivio News inglese, italiano)
