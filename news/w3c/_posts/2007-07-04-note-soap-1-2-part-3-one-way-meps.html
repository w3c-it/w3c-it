--- title: "Note: SOAP 1.2 Part 3: One-Way MEPs" date: 2007-07-04 categories: ---

The XML Protocol Working Group has published SOAP 1.2 Part 3: One-Way MEP as a Working Group Note. SOAP Version 1.2 Part 2 provides a request-response Message Exchange Pattern (MEP) and a response-only MEP. SOAP Version 1.2 Part 3 provides a one-way MEP. Learn more about the Web Services Activity.
