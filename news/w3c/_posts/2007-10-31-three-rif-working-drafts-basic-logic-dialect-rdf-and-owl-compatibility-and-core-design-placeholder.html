--- title: "Three RIF Working Drafts: Basic Logic Dialect, RDF and OWL Compatibility, and Core Design (Placeholder" date: 2007-10-31 layout: page categories: w3c --- Il Rule Interchange Format Working Group ha pubblicato tre documenti: RIF Basic Logic Dialect , RIF RDF and OWL Compatibility, e RIF Core Design - Placeholder. I primi due sono First Public Working Draft. Basic Logic Dialect spacifica un formato di base che permette a regole logiche di essere scambiate tra sistemi rule-based. Le regole interscambiate usando il Rule Interchange Format RIF possono dipendere, oppure essere usate in combinazione con dati RDF e/o RDF Schema oppure modelli di dati OWL . RIF RDF e OWL Compatibility specifica la compatibilità di RIF con i Semantic Web language RDF e RDFS; in un prossimo futuro il documento gestirà anche OWL. Il documento simbolo resetta le aspettative sul nucleo di progettazione RIF. Il Working Group ha deciso che il progetto precedentemente pubblicato come RIF Core possa essere considerato più come la base proprio per le Logic Rules, piuttosto che per tutte le regole. INel futuro un nuovo Core potrà essere pubblicato, ma per ora le parti interessate preferiscono rifarsi al Basic Logic Dialect. Maggiori informazioni sulla Semantic Web Activity home page.
