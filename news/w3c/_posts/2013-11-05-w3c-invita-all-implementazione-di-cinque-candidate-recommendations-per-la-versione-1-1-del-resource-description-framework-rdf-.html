--- title: "W3C invita all'implementazione di cinque Candidate Recommendations per la versione 1.1 del Resource Description Framework (RDF)" date: 2013-11-05 categories: ---

L'RDF Working Group ha pubblicato oggi 5 Candidate Recommendations per la versione 1.1 del Resource Description Framework (RDF), stabile e diffusa tecnologia per la data interoperability:

  • RDF 1.1 Concepts and Abstract Syntax definisce le basi di tutte le sintassi e i sistemi RDF per la data interoperability in generale.
  • RDF 1.1 Semantics definisce la semantica precisa dei dati RDF, supportando l'uso di una vasta gamma di “semantic” e “knowledge” technologies.
  • RDF 1.1 N-Triples definisce una semplice sintassi line-oriented per la serializzazione dei dati RDF. N-Triples è un piccolo sottoinsieme di Turtle.
  • RDF 1.1 TriG definisce un'estensione di Turtle (allineata con SPARQL) per la gestione di grafi RDF multilpli in un singolo documento.
  • RDF 1.1 N-Quads definisce unìestensione di N-Triples per la gestione di grafi RDF multipli in un singlo documento.

Tutte queste tecnologie sono stabili e pronte per essere usate su larga scala. Ogni specifica (esclusi i Concepts) hanno una Test Suite associata  e includono un link all'Implementation Report che mostra come i vari software si comportano nei test. Chiunque mantenga del software RDF è invitato a rivedere queste specifiche, aggiornare il software se necessario, e (se rilevante) inviare i risultati dei test così come spiegato nell'Implementation Report.

RDF 1.1 è la revisione delle specifiche RDF del 2004, progettate per semplificare e migliorare RDF, garantendo però la continuità con l'esistente.
