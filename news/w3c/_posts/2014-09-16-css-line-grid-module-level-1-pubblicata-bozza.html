--- title: "CSS Line Grid Module Level 1 Pubblicata Bozza" date: 2014-09-16 layout: page categories: ---

Il Cascading Style Sheets (CSS) Working Group ha pubblicato una Bozza di Lavoro su CSS Line Grid Module Level 1. Questa specifica fornisce le funzionalità per allineare lieen e bocchi in griglie invisibili in un documento. Per saperne di più vai a Style Activity.
