--- title: "Pubblicato il First Draft Web Services SOAP Assertions (WS-SOAPAssertions)" date: 2010-06-08 categories: ---

Il Web Services Resource Access Working Group ha pubblicato il First Public Working Draft di Web Services SOAP Assertions (WS-SOAPAssertions). Questa specifica definisce due asserzioni WS-Policy che possono essere usate per pubblicizzare l'obbligo di utilizzare una determinata versione di SOAP negl scambi di messaggi. Per saperne di più si consulti Web Services Activity.
