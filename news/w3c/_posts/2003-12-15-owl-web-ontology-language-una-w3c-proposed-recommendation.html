--- title: "OWL Web Ontology Language è una W3C Proposed Recommendation" date: 2003-12-15 categories: ---

W3C è lieto di annunciare l' avanzamento di OWL Web Ontology Language (OWL) a Proposed Recommendation. Si attendono commenti entro il 19 gennaio. OWL è usato per pubblicare e condividere insiemi di termini, detti ontologie, supportando avanzate ricerche sul Web, agenti software e gestione della conoscenza. Per ulteriori informazioni: Semantic Web Activity. Il linguaggio OWL è presentato in sei parti:

  • Overview - Una semplice introduzione
  • Guide - Illustra OWL mediante un esempio esteso e fornisce un glossario
  • Reference - Una descrizione sintetica e informale delle primitive di OWL
  • Semantics and Abstract Syntax - Definizione normativa del linguaggio OWL
  • Test Cases - Casi di test che illustrano il corretto utilizzo di OWL, il significato formale dei costrutti e la risoluzione di problemi
  • Use Cases and Requirements - Scenari d' uso, scopi e requisiti di un Web ontology language
