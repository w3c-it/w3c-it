--- title: "Working Draft: Web Services Internationalization Usage Scenarios" date: 2004-05-12 layout: page categories: w3c ---

La Web Services Task Force dell' Internationalization Working Group ha pubblicato un Working Draft aggiornato di Web Services Internationalization Usage Scenarios con consigli aggiuntivi per gli implementatori di tecnologie di Web service. Il documento analizza come linguaggio, cultura e problemi ad essi collegati interagiscano con l' architettura e la tecnologia dei Web service. Si attendono commenti. Per ulteriori informazioni: Internationalization home page.
