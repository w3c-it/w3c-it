--- title: "Pubblicato Draft su Web Audio API" date: 2013-10-10 layout: page categories: ---

L'Audio Working Group ha pubblicato un Working Draft sulle Web Audio API. Questa specifica descrive una API JavaScript ad alto livello per processare e sinetizzare l'audio nelle applicazioni Web. Il paradigma principale è quello di un grafo di routing audio, in cui degli oggetti di tipo AudioNode sono connessi gli uni agli altri per definire il rendering audio complessivo. Il processing effettivo si svolge principalmente grazie all'implementazione sottostante (tipicamente codice Assembly / C / C++ ottimizzato), ma sono anche supportati il processing e la sintetizzazione diretti in JavaScript.
