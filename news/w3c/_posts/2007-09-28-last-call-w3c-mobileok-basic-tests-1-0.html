--- title: "Last Call: W3C mobileOK Basic Tests 1.0" date: 2007-09-28 layout: page categories: --- Il Mobile Web Best Practices Working Group ha rilasciato il terzo Last Call Working Draft di W3C mobileOK Basic Tests 1.0. I commenti sono benvenuti fino al 19 ottobre. Questi test forniscono le basi per reclamare nel caso in cui non ci siano compatibilità con il W3C mobileOK Basic e corrispondenze con le Mobile Web Best Practices del W3C. Maggiori informazioni sulla Mobile Web Initiative home page. (Archivio News inglese, italiano)
