--- title: "Web IDL Draft Pubblicata" date: 2008-12-22 categories: --- Il Web Applications Working Group ha pubblicato la Working Draft di Web IDL. Questo documetno definisce un linguaggio per la definizione di interfacce, Web IDL, che può essere usato per descrivere interfacce volte ad essere implementate nei browser web. Web IDL è una variante di IDL con un numero di caratteristiche che permettono di specificare più prontamente comportamenti dei script ad oggetti nelle piattaforme web. Il modo in cui le interfacce descritte con Web IDL corrispondano ai costrutti all'interno di ambienti di esecuzione Java e ECMAScript viene anche descritto. Ulteriori informaizoni su Rich Web Client Activity. (Archivio News Inglese, Italiano)
