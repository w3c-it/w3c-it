--- title: "Pubblicato CSS3 Paged Media Working Draft" date: 2003-09-09 layout: page categories: w3c ---

Il CSS Working Group ha pubblicato un Working Draft aggiornato di CSS3 module: Paged Media, parte del linguaggio Cascading Style Sheets (CSS) livello 3. Il modulo aggiunge paginazioni, margini di pagina, intestazioni e footer, note a pie' di pagina ed endnote, riferimenti incrociati con numeri di pagina. Per maggiori informazioni: CSS home page.
