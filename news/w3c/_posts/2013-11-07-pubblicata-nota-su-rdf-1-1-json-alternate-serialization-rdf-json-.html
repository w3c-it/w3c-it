--- title: "Pubblicata nota su RDF 1.1 JSON Alternate Serialization (RDF/JSON)" date: 2013-11-07 layout: page categories: ---

L'RDF Working Group ha pubblicato una Group Note su RDF 1.1 JSON Alternate Serialization (RDF/JSON). Il Resource Description Framework (RDF) è un framework per la rapperesentazione di informazioni sul Web. Questo documento definisce una sintassi testuale per l'RDF chiamata RDF/JSON che permette che un grafo RDF venga scritto interamente in una forma compatibile con JavaScript Object Notation (JSON) e alternativa a quella raccomandata in JSON-LD. La sintassi definita in questo documento non dovrebbe essere usata a meno che non ci sia una ragione specifica per farlo. E' raccomandato invece l'uso di JSON-LD.
