--- title: "Widgets 1.0: Aggiornamenti" date: 2008-10-09 layout: page categories: w3c --- Il Web Applications Working Group ha pubblicato la prima Working Draft pubblica di Widgets 1.0: Updates. Questa specifica definisce un modello per permettere ai widget user agent di individuare e sostituire una risorsa widget con una nuova o diversa versione di una risorsa widget. Il modello aggiornato è pensato per lavorare sia su HTTP sia da local storage. Per gli aggiornamenti via Web, il modello usa un semplice documento XML che gli autori mettono su un Web server per indicare, fra le altre cose, dove è possibile trovare la nuova versione di una risorsa widget. definisce anche un meccanismo che permette agli autori di avere notifica degli errori di installazione o dei successi. Questa specifica descrive anche come rinegoziare le politiche di sicurezza quando un widget viene aggiornato. Ulteriori informazioni su Rich Web Client Activity. (Archivio News Inglese, Italiano)
