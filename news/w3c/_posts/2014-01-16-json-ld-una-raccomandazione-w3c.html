--- title: "JSON-LD è una Raccomandazione W3C" date: 2014-01-16 categories: ---

L' RDF Working Group oggi ha pubblicato due Raccomandazioni:

  • JSON-LD 1.0. JSON è un utile formato di serializzazione di dati e messaggistica. Questa specifica definisce JSON-LD, un formato JSON-based per serializzare Linked Data. La sintassi è progettata per integrarsi facilmente in sistemi distribuiti che utilizzano già JSON, e fornisce un percorso di aggiornamento graduale da JSON a JSON-LD. Esso è principalmente destinato ad essere un modo per utilizzare dati collegati in ambienti di programmazione basati sul Web, per creare servizi Web interoperabili, e memorizzare Linked Data in motori di storage JSON-based.
  • JSON-LD 1.0 Algoritmi di Elaborazione e API. Questa specifica definisce un set di algoritmi per trasformazioni programmatiche di documenti JSON-LD. Ristrutturando dati secondo le trasformazioni definite spesso semplifica notevolmente il suo utilizzo. Inoltre, il presente documento propone una Application Programming Interface (API) per sviluppatori che implementano gli algoritmi specificati.
