--- title: "Clipboard Operations: Working Draft" date: 2006-11-15 layout: page categories: --- Il Web API Working Group ha rilasciato il First Public Working Draft di Clipboard Operations for the Web 1.0: Copy, Paste, Drag and Drop. Gli sviluppatori potranno usare questo API per abilitare e migliorare le comuni funzioni clipboard nelle loro applicazioni Web. Questo primo draft è basato sulla documentazione per il trasferimento di dati per Internet Explorer su Windows. Il gruppo sollecita l'invio di commenti da parte degli sviluppatori di contenuti Web e browser. Maggiori informazioni sulla pagina della Rich Web Clients Activity. (Archivio News inglese, italiano)
