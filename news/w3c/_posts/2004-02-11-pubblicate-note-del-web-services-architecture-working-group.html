--- title: "Pubblicate Note del Web Services Architecture Working Group" date: 2004-02-11 layout: page categories: w3c ---

Il Web Services Architecture Working Group ha pubblicato le Working Group Note che rappresentano il culmine del lavoro svolto: Web Services Architecture, Usage Scenarios, Glossary, e Requirements. L' architettura di riferimento identifica i componenti dei Web service, ne definisce le relazioni e le limitazioni. Per ulteriori informazioni: Web Services home page.
