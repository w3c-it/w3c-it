--- title: "Last Call: CSS Backgrounds and Borders Module Level 3" date: 2010-06-15 categories: ---

Il Cascading Style Sheets (CSS) Working Group ha pubblicato un Last Call Working Draft di CSS Backgrounds and Borders Module Level 3. Questa specifica descrive gli stili per bordi e sfondi. Include ed estende le funzionalità di CSS level 2, aggiungendo tra esse: bordi costituiti da immagini, box con sfondi multipli, box con angoli arrotondati, e box con ombre. Si accettano commenti fino al 6 luglio. Per ulteriori informazioni si consulti Style Activity.
