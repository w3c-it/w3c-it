--- title: "Last Call: Web Security Context: Linee guida User Interface" date: 2008-07-24 layout: page categories: --- Il Web Security Context Working Group ha pubblicato il Last Call Working Draft di Web Security Context: User Interface Guidelines. Queste specifiche si occupano delle decisioni di fiducia che un utente deve prendere online, e dei modi di supportarli nel prendere decisioni sicure ed informate dove possibile. Per raggiungere questo obiettivo, queste specfiche includono raccomandazioni sulla presentazione di informazioni di identità da parte dei Web user agents. Esso include anche raccomandazioni sulla gestione degli errori nei protocolli di sicurezza. Le raccomandazioni sulla gestione degli errori riducono al minimo le decisioni di fiducia lasciate all'utente, e rappresenta le pratiche maggiormente conosciute per indurre gli utenti verso comportamenti sicuri in cui debbano prendere queste decisioni. I commenti sono bene accetti fino al 15 settembre. Ulteriori informazioni su Security Activity. (Archivio News Inglese, Italiano)
