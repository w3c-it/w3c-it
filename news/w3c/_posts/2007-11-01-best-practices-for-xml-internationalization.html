--- title: "Best Practices for XML Internationalization" date: 2007-11-01 layout: page categories: --- L'Internationalization Tag Set (ITS) Working Group ha pubblicato il Working Draft di Best Practices for XML Internationalization. Questo documento fornisce un set di linee-guida per lo sviluppo di documenti XML e schemi che siano appripriatamente internazionalizzati. Seguendo le buone prassi descritte qui, sia gli sviluppatori di applicazioni XML, sia gli autori di contenuti XML possono creare materiale in linguaggi differenti. Maggiori informazioni sulla Internationalization Activity home page. (Archivio News inglese, italiano)
