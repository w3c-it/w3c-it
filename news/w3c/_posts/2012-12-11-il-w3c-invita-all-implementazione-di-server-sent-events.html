Il W3C Invita all'implementazione di Server-Sent Events

11 December 2012

Il gruppo di lavoro sulle applicazioni web invita ad implementare la Candidate Recommendation di Server-Sent Events. Questa specifica definisce delle API per aprire una connessione HTTP in grado di ricevere notifiche push da un server sotto forma di eventi DOM. Le API sono progettate in modo tale da poter essere estese per funzionare con altri schemi di notifiche push, quali Push SMS. Per saperne di più vai a Rich Web Client Activity.
