--- title: "Working Draft: WSDL 2.0 RDF Mapping" date: 2005-11-04 layout: page categories: --- Il Web Services Description Working Group ha rilasciato il Primo Public Working Draft di Web Services Description Language (WSDL) Versione 2.0: RDF Mapping. Il draft descrive i componenti WSDL 2.0 nel Resource Description Language (RDF) e nel Web Ontology Language (OWL) in modo che tutti i documenti WSDL 2 possano essere mescolati con altri dati Semantic Web.
