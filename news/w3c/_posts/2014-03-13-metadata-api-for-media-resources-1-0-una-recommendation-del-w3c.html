--- title: "Metadata API for Media Resources 1.0 è una Recommendation del W3C" date: 2014-03-13 layout: page categories: w3c ---

Il Media Annotations Working Group ha pubblicato una Recommendation di Metadata API for Media Resources 1.0. Questo documento definisce una API per accedere ai metadati relativi a risorse Web multimediali. Lo scopo generale è fornire agli sviluppatori un accesso comodo ai metadati salvati in diversi formati. L'API fornisce dei modi per accedere alle proprietà definite nella specifica Ontology for Media Resources 1.0. Per ulteriori informazioni consulta Video on the Web Activity.
