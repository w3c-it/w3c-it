--- title: "Ultima chiamata: XQuery 3.0: An XML Query Language " date: 2013-07-23 layout: page categories: ---

L' XML Query Working Group ha pubblicato una Last Call Working Draft del XQuery 3.0: An XML Query Language. XML è un linguaggio di markup versatile,in grado di etichettare il contenuto informativo di fonti di dati diverse tar cui documenti strutturati e semi-strutturati, database relazionali e repository di oggetti. Un linguaggio di query che utilizza la struttura di XML con intelligenza può esprimere le query in tutti questi tipi di dati, sia fisicamente memorizzati in formato XML che visti come XML tramite middleware. Questa specifica descrive un linguaggio di query denominato XQuery, che è stato progettato per essere ampiamente applicabile in molti tipi di fonti di dati XML. I commenti sono benvenuti fino al 13 agosto. Per saperne di più su Extensible Markup Language (XML) Activity.
