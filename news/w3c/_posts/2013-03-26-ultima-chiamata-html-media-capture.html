--- title: "Ultima chiamata: HTML Media Capture" date: 2013-03-26 categories: ---

Il gruppo di lavoro Device APIs Working Group ha pubblicato un'ultima Call per la Bozza di Lavoro HTML Media Capture. Le specifiche relative all'HTML Media Capture definiscono una estensione della form HTML che facilita l'accesso dell'utente  ai meccanismi di "media capture" disponibili sui dispositivi attuali, come fotocamere o microfoni, dall'interno di un sistema di controllo per il file in caricamento. I Commnenti sono benvenuti fino al 19 Aprile. Per saperne di più  Ubiquitous Web Applications Activity.
