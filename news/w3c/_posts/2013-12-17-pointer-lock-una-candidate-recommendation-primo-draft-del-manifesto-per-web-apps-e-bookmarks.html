--- title: "Pointer Lock è una Candidate Recommendation; Primo Draft del Manifesto per web apps e bookmarks" date: 2013-12-17 layout: page categories: ---

Il Web Applications Working Group oggi ha pubblicato tre documenti:

  • Una Candidate Recommendation di Pointer Lock, che definisce un'API che fornisce l'accesso con script a dati grezzi del movimento del mouse mentre blocca l'obiettivo dell'evento mouse ad un singolo elemento e rimuove il cursore dalla vista. Questa è una modalità di input essenziale per alcune classi di applicazioni, in particolare applicazioni di prospettiva 3D in prima persona e software di modellazione 3D.
  • Un Primo Working Draft Pubblico del Manifesto per web apps e bookmarks, che fornisce agli sviluppatori un posto centralizzato dove mettere i metadati di un'applicazione web. Questo comprende, tra le altre cose, l'abilità di specificare il nome dell'applicazione web, i link ad icone, così come l'URL preferita su cui l'applicazione web si dovrebbe aprire quando è lanciata dall'utente.
  • Uno Working Draft su Input Method Editor API, che definisce un editor di metodo di input API (IME) per applicazioni Web. Un IME è un'applicazione che consente ad una tastiera standard (come una tastiera US-101) di essere usata per scrivere caratteri e simboli che non sono direttamente rappresentati sulla tastiera stessa. In Cina, Giappone e Korea, gli IME sono usati ubiquitariamente per consentire a tastiere standard di essere utilizzate per digitare il grandissimo numero di caratteri necessari per scrivere in Cinese, Giapponese, Corean.
