--- title: "Il Web Cryptography Working Group ha pubblicato tre draft" date: 2013-01-08 layout: page categories: ---

Il Web Cryptography Working Group ha pubblicato oggi tre documenti.

  • Un Working Draft sulla Web Cryptography API. Questa specifica descrive una API JavaScript per eseguire operazioni crittografiche di base nelle applicazioni Web, come hashing, generazione e verifica di firme, criptazione e decriptazione. Descrive inoltre una API per dare la possibilità alle applicazioni di generare e gestire il materiale per la produzione delle chiavi che è necessario quando si eseguono queste operazioni. Gli usi di questa API spaziano dall'identificazione di utenti o servizi, alla firma di documenti o codice, alla segretezza e integrità delle comunicazioni.
  • Un primo Working Draft pubblico su WebCrypto Key Discovery. Questa specifica descrive una API JavaScript per l'individuazione di chiavi con nome e origin-specific pre-confezionate, per l'uso con la Web Cryptography API. Le chiavi pre-confezionate sono chiavi che sono state rese disponibili allo User Agent con sistemi diversi dalle funzioni di generazione, derivazione o importazione della Web Cryptography API. Le chiavi origin-specific sono chiavi che sono disponibili solo ad una specifica origin. Le chiavi con nome sono identificate da un nome che si intende noto alla origin in questione e che viene fornito con la chiave stessa.
  • Un primo Working Draft pubblico sui Casi d'uso della Web Cryptography API. Questo documento è un'overview informativa sui casi d'uso per una API per la crittografia sul Web. Questi casi d'uso, descritti come scenari, rappresentano alcuni degli insiemi di funzionalità che possono essere ottenute dalla Web Cryptography API, che fornisce una API per operazioni di crittografia, e dalla Key Discovery API, che descrive nello specifico la possibilità di accedere a chiavi crittografiche che sono state pre-confezionate.
