--- title: "New Editions of Core XML Standards Published" date: 2006-08-16 layout: page categories: --- The World Wide Web Consortium today published new editions of four core XML standards: the fourth edition of Extensible Markup Language (XML) 1.0 and second editions of Extensible Markup Language (XML) 1.1, Namespaces in XML 1.0 and Namespaces in XML 1.1. These new editions incorporate corrections to all known errata. Read the press release and visit the XML home page. (Archivio News inglese, italiano)
