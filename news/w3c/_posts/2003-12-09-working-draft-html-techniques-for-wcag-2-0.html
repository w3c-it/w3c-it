--- title: "Working Draft: HTML Techniques for WCAG 2.0" date: 2003-12-09 layout: page categories: ---

Web Content Accessibility Guidelines (WCAG) Working Group ha pubblicato il primo Working Draft pubblico di HTML Techniques for WCAG 2.0. Il documento fornisce informazioni agli sviluppatori di contenuti Web che intendono soddisfare i criteri di successo di Web Content Accessibility Guidelines 2.0, attualmente nello stato di Working Draft. Si attendono commenti. Per ulteriori informazioni: Web Accessibility Initiative.
