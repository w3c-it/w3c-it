--- title: "Il W3C Organizza un Workshop sulla sicurezza per le API di accesso ai dispositivi dal Web" date: 2008-09-30 layout: page categories: --- Il W3C vi invita a partecipare al Workshop on Security for Access to Device APIs from the Web organizzato dalla Vodafone di Londra (UK) per il 10-11 dicembre 2008. L'obiettivo di questo workshop é di radunare persone con un'ampia diversità d'esperienza (API designers, esperti di sicurezza, esperti di usabilità, ...) per discutere le sfide di sicurezza che rigurdano il modo per permettere alle applicazioni Web e widget di accedere alle API che consentono il controllo di queste caratteristiche (ad es., videocamere gps, rubriche, ecc.). I partecipanti daranno consigli al W3C sui passi sucessivi appropriati per superare qualsiasi ostacolo con nuove tecniche. Position papers sono pianificate per il 30 ottobre. Il W3C vi invita ad approfondire leggendo il Workshop scope. (Archivio News Inglese, Italiano)
