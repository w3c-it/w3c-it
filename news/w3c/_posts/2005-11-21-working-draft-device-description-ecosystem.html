--- title: "Working Draft: Device Description Ecosystem" date: 2005-11-21 layout: page categories: --- Il Mobile Web Initiative Device Description Working Group ha rilasciato un primo Public Working Draft di Device Description Ecosystem, futura Nota del Working Group. Servendo come input per il documento del Gruppo sui requisiti, questo draft delinea modelli business e partecipanti nel campo della descrizione dei dispositivi ed ipotizza un repository comune. Maggiori informazioni sulla pagina della Mobile Web Initiative. (Archivio News inglese, italiano)
