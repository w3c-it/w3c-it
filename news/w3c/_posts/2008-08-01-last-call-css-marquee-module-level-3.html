--- title: "Last Call: CSS Marquee Module Level 3" date: 2008-08-01 layout: page categories: --- The Cascading Style Sheets (CSS) Working Group has published a Last Call Working Draft of CSS Marquee Module Level 3. CSS describes the rendering of documents on various media. When documents (e.g., HTML) are laid out on visual media (e.g., screen or print) and the contents of some element are too large for a given area, CSS allows the designer to specify whether and how the overflow is displayed. One way, available on certain devices, is the "marquee" effect: the content is animated and moves automatically back and forth. This module defines the properties to control that effect. Comments are welcome through 01 September. Learn more about the Style Activity. (Archivio News Inglese, Italiano)
