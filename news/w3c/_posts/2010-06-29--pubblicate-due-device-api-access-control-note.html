--- title: " Pubblicate due Device API Access Control Note" date: 2010-06-29 layout: page categories: ---

Il Device APIs and Policy Working Group ha pubblicato due Group Notes:  Device API Access Control Requirements e Device API Privacy Requirements. Il primo definisce i requisiti per controllare l'accesso alle API dei dispositivi, illustrati da corrispondenti casi d'uso. Il secondo fornisce definizioni, casi d'uso e prescrizioni per rendere le API dei dispositivi più privacy-friendly. Per ulteriori informazioni si consulti Ubiquitous Web Applications Activity.
