--- title: "Working Draft: State Chart XML (SCXML)" date: 2005-07-05 layout: page categories: --- Il Voice Browser Working Group ha rilasciato il Primo Public Working Draft di State Chart XML (SCXML): State Machine Notation for Control Abstraction 1.0. SCXML è un ambiente di esecuzione basato su UML Harel State Tables e CCXML . SCXML è candidato per il controllo della lingua all'interno di VoiceXML 3.0, CCXML 2.0 e il linguaggio di authoring in fase di sviluppo da parte del Multimodal Interaction Working Group.
