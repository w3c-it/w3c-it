--- title: "Progress Events 1.0: Working Draft" date: 2007-04-20 layout: page categories: --- Il Web API Working Group ha rilasciato il First Public Working Draft di Progress Events 1.0. Cinque eventi e le loro interfacce sono definiti per essere usati nelle applicazioni Web XHR (AJAX). Quando dati aggiuntivi sono scaricati on demand, gli script possono monitorare l'avanzamento, costruire barre di caricamento e compiere azioni una volta che i dati siano stati trasferiti. Maggiori informazioni sulla pagina Rich Web Clients Activity. (Archivio News inglese, italiano)
