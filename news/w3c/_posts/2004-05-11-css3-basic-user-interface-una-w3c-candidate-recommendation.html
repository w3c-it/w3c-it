--- title: "CSS3 Basic User Interface è una W3C Candidate Recommendation" date: 2004-05-11 layout: page categories: ---

W3C è lieto di annunciare l' avanzamento di CSS3 Basic User Interface Module nello stato di Candidate Recommendation. Il linguaggio Cascading Style Sheets (CSS) è usato per restituire sullo schermo, su carta e in parlato documenti strutturati HTML e XML. Questo modulo si occupa, tra le varie cose, di caratteristiche dell'interfaccia utente, di frammenti di elementi e di form. Si attendono commenti entro l' 11 novembre 2004. Per ulteriori informazioni: CSS home page.
