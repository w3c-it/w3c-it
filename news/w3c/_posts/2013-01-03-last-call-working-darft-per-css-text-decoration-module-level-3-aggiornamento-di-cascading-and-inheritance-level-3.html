--- title: "Last Call working Darft per CSS Text Decoration Module Level 3; Aggiornamento di Cascading and Inheritance Level 3" date: 2013-01-03 layout: page categories: ---

Il gruppo di lavoro Cascading Style Sheets (CSS) ha pubblicato una Last Call Working Draft su CSS Text Decoration Module Level 3.  Questo modulo contiene le caratteristiche dei CSS in materia di text decoration, come sottolineature, ombre, testi e segni di enfasi. Si accettano commenti fino al 31 gennaio. 

Il gruppo ha anche pubblicato una bozza su CSS Cascading and Inheritance Level 3. Uno dei principi fondamentali della progettazione CSS è il cascading che consente di organizzare la presentazione di un documento usando diversi fogli si stile. E' possibile che in diversi fogli di stile, venga specificato il valore della stessa coppia elemento/proprietà e si creino dunque dei conlfitti che vanno risolti. Questa specifica contiene le regole che consentono di determinare il valore corretto di tutte le proprietà di tutti gli elementi in un documento.
