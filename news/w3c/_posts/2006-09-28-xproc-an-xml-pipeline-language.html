--- title: "XProc: An XML Pipeline Language" date: 2006-09-28 layout: page categories: --- L'XML Processing Model Working Group ha rilasciato il First Public Working Draft di XProc: An XML Pipeline Language. Utilizzato ed organizzato per controllare il flusso di documenti, il linguaggio XProc standardizza interazioni, inputs ed output per le trasformazioni di un largo gruppo di specifiche come XSLT, XML Schema, XInclude e Canonical XML che operano e producono documenti XML. Maggiori informazioni sulla XML home page. (Archivio News inglese, italiano)
