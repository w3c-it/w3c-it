--- title: "Working Draft: Authorizing Read Access to XML Content" date: 2006-05-18 categories: --- I Working Group Voice Browser, Web API e Web Application Formats (WAF) hanno rilasciato congiuntamente un Working Draft di Authorizing Read Access to XML Content Using the <?access-control?> Processing Instruction 1.0. Il draft descrive un meccanismo in uso da parte dei fornitori di voice browser, che permette ai content provider di specificare la politica di accesso ai contenuti. I programmatori potrebbero così eseguire le proprie analisi di sicurezza. Maggiori informazioni sulla pagina della Rich Web Clients Activity. (Archivio News inglese, italiano)
