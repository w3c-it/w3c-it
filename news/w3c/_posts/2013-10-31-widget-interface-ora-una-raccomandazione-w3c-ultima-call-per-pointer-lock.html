--- title: "Widget Interface è ora una Raccomandazione W3C; Ultima Call per Pointer Lock" date: 2013-10-31 layout: page categories: ---

Il Gruppo di Lavoro Web Applications ha pubblicato una Raccomandazione W3C su Widget Interface. Questa specifica definisce un API per widgets che fornisce, tra le altre cose, funzionalità per l'accesso a metadati di un widget e la  memorizzazione persistente dei dati .

Il Gruppo di Lavoro Web Applications ha pubblicato anche l'ultima Call per una Bozza di Lavoro su Pointer Lock. Questa specifica definisce un API che fornisce script di accesso ai dati grezzi di movimento del mouse mentre si tiene bloccato il target degli eventi mouse su un singolo elemento e rimuovendo il cursore dalla vista. Si tratta di una modalità di input essenziale per alcune classi di applicazioni, soprattutto  le applicazioni 3D con prospettiva in prima persona e i software di modellazione 3D. I commenti sono benvenuti fino al 28 novembre.
