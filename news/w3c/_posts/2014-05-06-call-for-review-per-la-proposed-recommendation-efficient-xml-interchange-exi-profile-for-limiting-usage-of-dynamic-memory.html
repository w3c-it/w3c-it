--- title: "Call for Review per la Proposed Recommendation: Efficient XML Interchange (EXI) Profile for limiting usage of dynamic memory" date: 2014-05-06 categories: ---

L'Efficient XML Interchange Working Group ha pubblicato una Proposed Recommendation di Efficient XML Interchange (EXI) Profile for limiting usage of dynamic memory. Questo documento descrive un insieme di opzioni che può essere usato per definire i profili di EXI 1.0, formato adatto ad applicazioni per dispositivi con vincoli di memoria dinamica. Si accettano commenti fino al 13 giugno 2014. Per ultriori informazioni visita Extensible Markup Language (XML) Activity.
