--- title: "Pubblicato Working Draft: Web Services Choreography Description Language 1.0" date: 2004-04-27 categories: ---

Il Web Services Choreography Working Group ha pubblicato il primo Working Draft pubblico di Web Services Choreography Description Language Version 1.0 (WS-CDL). Si attendono commenti sulla mailing list pubblica del gruppo. WS-CDL definisce la collaborazione peer-to-peer tra i partecipanti ad un Web service. Per ulteriori informazioni si veda il comunicato stampa in inglese e si visiti la Web services home page.
