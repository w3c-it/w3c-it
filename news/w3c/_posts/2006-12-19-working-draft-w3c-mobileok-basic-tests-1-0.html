--- title: "Working Draft: W3C mobileOK Basic Tests 1.0" date: 2006-12-19 layout: page categories: w3c --- Il Mobile Web Best Practices Working Group ha rilasciato il secondo Working Draft di W3C mobileOK Basic Tests 1.0. Questo documento definisce i test che forniscono le basi per diventare W3C mobileOK Basic compatibili, e sono basati sulle Mobile Web Best Practices del W3C. Maggiori informazioni sulla pagina del Mobile Web Best Practices Working Group. (Archivio News inglese, italiano)
