--- title: "Call for Review: pubblicata Proposed Recommendation su API for Media Resources 1.0" date: 2013-10-15 categories: ---

Il Media Annotations Working Group ha pubblicato una Proposed Recommendation su API for Media Resources 1.0. Questa specifica definisce una API per accedere ai metadati relativi a risorse multimediali sul Web. Lo scopo complessivo è fornire agli sviluppatori un accesso comodo ai metadati salvati in diversi formati. L'API fornisce diversi modi di accedere agli insiemi di proprietà definiti nella specifica Ontology for Media Resources 1.0. Tali proprietà vengono usate come vocabolario pivot in questa API. Il nucleo della specifica è la definizione di interfacce API per recuperare i metadati in modo sincrono o asincrono. Definisce anche interfacce per tipi di ritorno strutturati insieme alla specifica di comportamento di un'implementazione dell'API. Si accettano commenti fino al 25 novembre.
