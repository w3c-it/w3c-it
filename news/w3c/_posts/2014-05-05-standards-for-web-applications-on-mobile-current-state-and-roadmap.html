--- title: "Standards for Web Applications on Mobile: current state and roadmap" date: 2014-05-05 categories: ---

Il W3C ha pubblicato l'edizione di Aprile 2014 di Standards for Web Applications on Mobile che raccoglie le diverse tecnologie sviluppate dal W3C per migliorare le funzionalità delle Web applications, soprattutto nel contesto mobile. 

Nella nuova edizione del deliverable del progetto HTML5Apps include i cambiamenti e le aggiunte introdotte fino a Gennaio 2014, incluso un nuovo set di caratteristiche che evidenziano quali strumenti offrono servzi di sicurezza e privacy.
