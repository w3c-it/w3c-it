--- title: "Note pubblicate: Basic, Advanced XML Schema Patterns for Databinding Version 1.0" date: 2009-05-05 layout: page categories: --- L'XML Schema Patterns for Databinding Working Group ha pubblicato due Group Note: Basic XML Schema Patterns for Databinding Version 1.0 e Advanced XML Schema Patterns for Databinding Version 1.0. Uno schema pattern descrive i modi in cui le persone usano XML per le strutture dati comuni nei linguaggi di programmazione. Le strutture dati descritte si intendono indipendenti da qualsiasi particolare linguaggio di programmazione, database o ambiente di sviluppo. Ulteriori informazioni su Web Services Activity.
