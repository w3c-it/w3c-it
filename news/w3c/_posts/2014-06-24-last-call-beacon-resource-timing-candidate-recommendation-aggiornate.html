--- title: "Last Call: Beacon, Resource Timing Candidate Recommendation Aggiornate" date: 2014-06-24 layout: page categories: ---

Il Web Performance Working Group ha pubblicato una Ultima Call per una Bozza di lavoro su Beacon. Questa specifica definisce un mezzo interoperabile per sviluppatori di siti per trasferire piccoli dati HTTP in modalità asincrona da uno User Agent a un server web. I commenti sono benvenuti fino al 29 luglio 2014.

Il  Web Performance Working Group ha anche aggiornato la  Candidate Recommendation su Resource Timing. Questa specifica definisce un'interfaccia per applicazioni web per accedere alle informazioni di temporizzazione complete delle risorse in un documento.

Per saperne di più vai a Rich Web Client Activity.
