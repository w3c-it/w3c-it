--- title: "Aggiornate le User Agent Accessibility Guidelines (UAAG) 2.0 " date: 2010-06-17 layout: page categories: w3c ---

L' User Agent Accessibility Guidelines Working Group ha pubblicato un Working Draft aggiornato delle User Agent Accessibility Guidelines (UAAG) 2.0. Le UAAG definiscono come browser, lettori multimediali e altri "user agent" dovrebbero supportare l'accessibilità per persone con disabilità e lavorare con tecnologie assistive. Tale draft comprende aggiornamenti per focalizzare comportamento e indicazione, sottomissione di moduli, e dimensioni relative dei font. Il Working Group sollecita a fornire commenti ora in preparazione della Last Call. Il Working Group ha inoltre pubblicato un Working Draft di Implementation UAAG 2,0. Per saperne di più si legga l' invitation to review the UAAG 2.0 Working Draft e Web Accessibility Initiative (WAI).
