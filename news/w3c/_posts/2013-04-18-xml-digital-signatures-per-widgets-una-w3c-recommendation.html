XML Digital Signatures per Widgets è una W3C Recommendation

18 Aprile 2013

Il Web Applications Working Group ha pubblicato una W3C Recommendation sull' XML Digital Signatures per Widgets. Questo documento definisce un profilo per la specifica dell' XML Signature Syntax and Processing per consentire ad un widget package di essere firmato digitalmente. Autori e distributori possono firmare digitalmente un widget come meccanismo per assicurare continuità di origine e di distribuzione. Uno user agent, o altro sistema di validazione, può utilizzare una firma digitale per verificare l'integrità dei dati nei file entro un widget package e per confermare la chiave (o le chiavi) usata per la firma.
