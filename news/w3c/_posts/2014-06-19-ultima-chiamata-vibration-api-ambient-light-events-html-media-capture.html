--- title: "Ultima Chiamata: Vibration API, Ambient Light Events, HTML Media Capture" date: 2014-06-19 layout: page categories: w3c ---

Il Device APIs Working Group oggi ha pubblicato tre Last Call Working Drafts:

  • Vibration API. Questa specifica definisce un API che fornisce l'accesso al meccanismo di vibrazione del dispositivo che ospita. La vibrazione è una forma di feedback tattile.
  • Ambient Light Events. Questa specifica definisce un mezzo per ricevere gli eventi che corrispondono ad un sensore di luce per rilevare la presenza di una luce.
  • HTML Media Capture. La specifica HTML Media Capture definisce un HTML form extension che facilita l'accesso utente a un meccanismo di media capture di un dispositivo, come una fotocamera, o microfono, all'interno di un controllo di caricamento file.

I commenti su queste tre specifiche sono benvenuti fino al 24 Luglio 2014. Per saperne di più circa la Ubiquitous Web Applications Activity.
