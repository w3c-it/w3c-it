--- title: "Working Drafts: Web Services Description Language (WSDL) 2.0" date: 2005-05-11 layout: page categories: --- Il Web Services Description Working Group ha pubblicato un Working Drafts di Web Services Description Language (WSDL) Versione 2.0: la prima bozza completa di Primer, aggiornata a Part 1: Core Language e Part 2: Adjuncts e un primo Public Working Draft di SOAP 1.1 Binding. Linguaggio XML, WSDL descrive servizi di rete ed è utilizzato per documentare sistemi distribuiti e comunicazioni automatizzate tra le applicazioni. Maggiori informazioni su Web services.
