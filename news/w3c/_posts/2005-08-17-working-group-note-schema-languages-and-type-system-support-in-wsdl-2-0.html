--- title: "Working Group Note: Schema Languages and Type System Support in WSDL 2.0" date: 2005-08-17 layout: page categories: w3c --- Il Web Services Description Working Group ha pubblicato Discussion of Alternative Schema Languages and Type System Support in WSDL 2.0 come Working Group Note. La nota affronta l'estensibilità del type system WSDL 2.0, definisce l'utilizzo di XML Schema 1.0 come type system nella specifica WSDL 2.0 core, e include le estensioni base per DTDs e Relax NG.
