--- title: "Cinque documenti POWDER pubblicati; tre Last Call Draft" date: 2009-04-04 layout: page categories: --- Oggi il Protocol for Web Description Resources (POWDER) Working Group ha pubblicato cinque Working Draft. Lo scopo di Protocol for Web Description Resources (POWDER) è di fornire un significato agli individui e organizzazioni per descrivere un gruppo di risorse attraversoo la pubblicazione di metadati machine-readable. Il cambiamento principale in queste specifiche riguarda la sezione di canonizzazione di IRI del documento sul raggruppamento delle risorse (sezioni 2.1.3 - 2.1.5). Il gruppo ha pubblicato questi documenti:
  • Description Resources (Last Call); che spiega in dettaglio la creazione ed il ciclo di vita di Description Resources (DRs), che incapsula i metadati
  • Grouping of Resources (Last Call); che descrive come insiemi di IRI possano essere definiti in modo che le descrizioni o gli altri dati possano essere applicati alle risorse ottenute dereferenziando gli IRI che sono elementi dell'insieme.
  • Formal Semantics (Last Call); che descrive come il formato operazionale, relativamente semplice, di un documento POWDER possa essere trasformato per essere utilizzato dagli strumenti per il Semantic Web
  • Primer
  • Test Suite
