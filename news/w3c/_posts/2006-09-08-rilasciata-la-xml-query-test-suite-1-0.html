--- title: "Rilasciata la XML Query Test Suite 1.0" date: 2006-09-08 layout: page categories: --- L'XML Query Working Group ha rilasciato la versione 1.0 della XML Query Test Suite (XQTS). In questa release sono presenti oltre 15,000 test case. Il Working Group richiede implementatori per ottenere feedback e risultati a settembre (in forma anonima, se necessario) per aiutare a dimostrare che la specifica XML Query 1.0 può essere implementata in modo interoperabile ed è pronta ad avanzare allo stato di Proposed Recommendation. Maggiori informazioni sulla XML home page. (Archivio News inglese, italiano)
