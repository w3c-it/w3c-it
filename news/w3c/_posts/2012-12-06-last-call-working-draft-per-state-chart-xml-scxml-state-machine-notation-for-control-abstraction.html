--- title: "Last Call Working Draft per State Chart XML (SCXML): State Machine Notation for Control Abstraction" date: 2012-12-06 categories: ---

Il Voice Browser Working Group ha pubblicato un Last Call Working Draft per State Chart XML (SCXML): State Machine Notation for Control Abstraction. Questo documento descrive SCXML, "State Chart extensible Markup Language". SCXML fornisce un generico ambiente di esecuzione basato su macchine a stati, sfruttando CCXML e le Harel State Tables. Si accettano commenti fino all'11 di Gennaio. Ulteriori informazioni sulla Voice Browser Activity.
