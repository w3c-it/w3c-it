--- title: "Efficient XML Interchange (EXI) Formato 1.0 (Seconda Edizione) è una Raccomandazione W3C" date: 2014-02-11 layout: page categories: ---

L' Efficient XML Interchange Working Group ha pubblicato una Raccomandazione W3C su Efficient XML Interchange (EXI) Format 1.0 (Second Edition).Questo documento è la specifica del formato Efficient XML Interchange (EXI). EXI è una rappresentazione molto compatta per il Extensible Markup Language (XML) Information Set che è destinata ad ottimizzare contemporaneamente le prestazioni e l'utilizzo di risorse computazionali. Il formato EXI usa un approccio ibrido tratto dalle teorie dell'informazione e del linguaggio formale, oltre a tecniche pratiche verificate da misurazioni, per informazioni XML sulla codicica dell'entropia. Usando un algoritmo relativamente semplice, che è suscettibile di implementazione veloce e compatta, e un piccolo set di rappresentazioni di tipi di dati, produce in modo affidabile codifiche efficienti di flussi di eventi XML. Sono presentati il sistema di produzione grammatica e la definizione del formato di EXI. Per saperne di più circa la Extensible Markup Language (XML) Activity.
