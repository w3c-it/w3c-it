--- title: "Pubblicato il Working Draft CSS Template Layout Module" date: 2010-04-29 layout: page categories: w3c ---

Il Cascading Style Sheets (CSS) Working Group ha pubblicato un Working Draft del CSS Template Layout Module. Questa specifica fa parte del livello 3 di CSS e contiene funzionalità per descrivere i layout ad alto livello, destinati a compiti di posizionamento e allineamento di "widget" in un'interfaccia utente grafica o la griglia di layout per una pagina o una finestra, in particolare quando l'ordine visivo desiderato è differente dall'ordine degli elementi nel documento sorgente. Per saperne di più si consulti Style Activity.
