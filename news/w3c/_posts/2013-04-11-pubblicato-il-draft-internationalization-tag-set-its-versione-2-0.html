Pubblicato il Draft Internationalization Tag Set (ITS) Versione 2.0

11 Aprile 2013

Il MultilingualWeb-LT Working Group ha pubblicato un working draft aggiornato sull'Internationalization Tag Set (ITS) Versione 2.0. L'Internationalization Tag Set (ITS) 2.0 migliora le fondamenta che integrano l'elaborazione automatizzata del linguaggio umano nelle technologie Web. ITS 2.0 fornisce i metadati per favorire l'adozione del multilingual Web. Il draft riflette i cambiamenti fino alla ultima publication call il 6 Dicembre 2012 e richiede un feedback sui problemi aperti rimanenti. Per saperne di più vai al link Internationalization Activity.
