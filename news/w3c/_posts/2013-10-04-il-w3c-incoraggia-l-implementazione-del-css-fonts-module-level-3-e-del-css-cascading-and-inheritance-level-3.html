--- title: "Il W3C incoraggia l'implementazione del CSS Fonts Module Level 3, e del CSS Cascading and Inheritance Level 3" date: 2013-10-04 layout: page categories: w3c ---

Il Cascading Style Sheets (CSS) Working Group incoraggia l'implementazione di due Candidate Recommendation:

  • CSS Cascading and Inheritance Level 3. Questo modulo CSS descrive come raccogliere regole di stile e assegnare valori a tutte le proprietà di tutti gli elementi. Per mezzo della sovrapposizione e dell'ereditarietà i valori si propagano a tutte le proprietà di ogni elemento. CSS è un linguaggio per descrivere il rendering di documenti strutturati (come HTML e XML) su schermo, su carta, in forma di discorso, ecc.
  • CSS Fonts Module Level 3. Questo modulo CSS descrive come vengono specificate le proprietà dei font e come i font vengono caricati dinamicamente. I contenuti di questa specifica sono il consolidamento di ciò che precedentemente era diviso nei moduli CSS3 Fonts e CSS3 Web Fonts. La descrizione degli eventi di caricamento dei font è stata spostata nel modulo CSS3 Font Load Events.
