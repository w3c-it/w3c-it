--- title: "Pubblicata draft su CSS Grid Layout Module Level 1 " date: 2013-09-10 categories: ---

Il Cascading Style Sheets (CSS) Working Group ha pubblicato una Working Draft su CSS Grid Layout Module Level 1. Questo modulo CSS definisce un sistema di layout bidimensionale, grid-based, ottimizzato per il design di interfacce utente. Nel modello grid layout, il figlio di un grid container può essere posizionati in slot arbitrari all'interno di una griglia di layout flessibile o fissa. CSS è il linguaggio per descrivere il rendering dei documenti strutturato (Come HTML o XML) sulo schermo, su carta, nel parlato, etc.
