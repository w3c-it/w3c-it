--- title: "Pubblicata draft su Display Module Level 3 " date: 2014-09-11 categories: ---

Il Cascading Style Sheets (CSS) Working Group ha pubblicato una Working Draft su CSS Display Module Level 3. Questo modulo descrive come generare il CSS formatting box tree a partire dal document element tree e definisce le proprietà del display e delle box-suppress per controllarlo. CSS è un linguaggio usato per descrivere il rendering di docuemnti strutturati (come HTML o XML) su schermo, carta, ecc. Maggiori informazioni su Style Activity.
