--- title: "XML Query Test Suite Released" date: 2005-08-11 layout: page categories: --- L'XML Query Working Group e l'XSL Working Group sono lieti di annunciare il rilascio di XML Query Test Suite (XQTS). I gruppi invitano i Membri W3C ed il pubblico ad utilizzare questa suite che contiene circa 7,000 test-case con alcune o anche tutte le oltre 20 implementazioni della specifica della bozza di XML Query. I feedback aiuteranno i Working Group a giudicare l'implementabilità del linguaggio XML Query, a testarne l'interoperabilità e ne favoriranno l'avanzamento allo stato di W3C Recommendation. Sono benvenuti anche contributi di test-case aggiuntivi. Maggiori informazioni sulla XML home page. (Archivio News inglese, italiano)
