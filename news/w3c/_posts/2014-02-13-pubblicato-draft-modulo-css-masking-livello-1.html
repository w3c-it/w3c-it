--- title: "Pubblicato Draft Modulo CSS Masking Livello 1" date: 2014-02-13 layout: page categories: ---

Il CSS Working Group e il SVG Working Group hanno pubblicato un Working Draft su CSS Masking Module Level 1. CSS Masking fornisce due mezzi per nascondere parzialmente o completamente porzioni di elementi visivi: mascheramento e clipping. Il mascheramento descrive come usare un altro elemento grafico o immagine come maschera di luminance o alpha. In genere, il rendering di un elemento tramite CSS o SVG può concettualmente essere descritto come se l'elemento, compresi i suoi figli, sono disegnati in un buffer e quindi quel buffer viene composto nel genitore dell'elemento. Le maschere luminance e alpha influenzano la trasparenza di questo buffer prima della fase di composing. Il clipping descrive la regione visibile degli elementi visivi. La regione può essere descritta usando determinati elementi grafici o forme base SVG. Qualsiasi cosa al di fuori della regione non viene renderizzato. Il CSS è un linguaggio per descrivere il rendering di documenti strutturati (come HTML e XML) su schermo, carta, discorsi, etc. Per saperne di più circa la Style Activity e la Graphics Activity.
