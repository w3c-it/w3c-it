--- title: "Website Accessibility Conformance Evaluation Methodology (WCAG-EM) 1.0 Note Published" date: 2014-07-10 layout: page categories: ---

Website Accessibility Conformance Evaluation Methodology (WCAG-EM) 1.0 è stato pubblicato oggi come Nota W3C dal Web Content Accessibility Guidelines Working Group (WCAG WG) e dall'Evaluation and Repair Tools Working Group (ERT WG), tramite la WCAG 2.0 Evaluation Methodology Task Force (Eval TF). WCAG-EM descrive un approccio per valutare quanto i siti Web, incluse le applicazioni Web e i siti per dispositivi mobili, sono conformi a WCAG 2.0. WCAG-EM fornisce una guida per individui e organizzazioni che valutano l'accessibilità, ed è utile anche per coloro che affidano a terzi la velutazione dei propri siti. Per ulteriori informazioni consulta WCAG-EM Overview e Web Accessibility Initiative (WAI).
