--- title: "Pubblicati Last Call per User Agent Accessibility Guidelines (UAAG) 2.0 e Working Draft su Implementing UAAG 2.0" date: 2013-11-07 layout: page categories: w3c ---

Il User Agent Accessibility Guidelines Working Group (UAWG) ha pubblicato oggi il Last Call Working Draft su User Agent Accessibility Guidelines (UAAG) 2.0 e un Working Draft su Implementing UAAG 2.0. UAAG definisce il modo in cui browser, media player e altri "user agent" dovrebbero supportare l'accessibilità per persone con disabilità e lavorare con tecnologie assistive. UAWG richiede in particolare che gli sviluppatori di browres, media player e applicazioni Web rivedano e commentino queste linee guida entro il 16 dibembre 2013. Ulteriori informazioni sulla Web Accessibility Initiative (WAI).
