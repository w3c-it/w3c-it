--- title: "Pubblicate le Bozze su CSS Exclusion e CSS Regions" date: 2013-05-28 layout: page categories: w3c ---

Il Gruppo di Lavoro su Cascading Style Sheets (CSS) ha pubblicato oggi 3 nuove Bozze di Lavoro:

  • CSS Exclusions Module Level 1. La bozza su CSS Exclusions definisce aree arbitrarie attorno alle quali i contenuti in linea (CSS21) possono fluire. CSS Exclusions può essere definito in ogni elemento parte di un CSS block-level. CSS Exclusions estende il concetto di contenuto che avvolge un elemento prima limitato ai contenuti flottanti. 
  • CSS Regions Module Level 1. Il  modulo CSS regions permette al contenuto di fluire attraverso aree multiple chiamate regioni. Le regioni non sono necessariamente contigue rispetto all'ordine stabilito nel documento. Il modulo CSS regions fornisce un meccanismo avanzato di flusso di contenuti, che può essere combinato con schemi di posizionamento come definito su altri moduli CSS tipo il Multi-Column Module o il Grid Layout Module che permettono di posizionare le regioni all'interno delle quali il contenuto fluisce.
