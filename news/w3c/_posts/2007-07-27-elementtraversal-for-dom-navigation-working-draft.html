--- title: "ElementTraversal for DOM Navigation: Working Draft" date: 2007-07-27 categories: ---

Il Web API Working Group ha rilasciato il First Public Working Draft di ElementTraversal Specification. L'interfaccia ElementTraversal definisce proprietà che gli script possono usare per maneggiare il DOM Elements e fornisce anche la proprietà childElementCount per il preprocessing. La specifica faceva originariamente parte di SVG Tiny 1.2. Maggiori informazioni sulla rich Web clients home page. (Archivio News inglese, italiano)
