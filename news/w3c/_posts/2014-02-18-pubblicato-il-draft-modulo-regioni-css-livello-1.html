--- title: "Pubblicato il Draft Modulo Regioni CSS Livello 1" date: 2014-02-18 categories: ---

Il Cascading Style Sheets (CSS) Working Group ha pubblicato un Working Draft su Modulo Regioni CSS Livello 1. Il modulo Regioni CSS consente al contenuto di uno o più elementi di fluire attraverso una o più caselle denominate Regioni CSS, frammentate come definito nel CSS3-BREAK. Questo modulo definisce anche CSSOM per esporre sia gli input che gli output di questa frammentazione. Per saperne di più circa la Style Activity.
