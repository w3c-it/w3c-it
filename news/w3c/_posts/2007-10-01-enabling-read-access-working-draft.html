--- title: "Enabling Read Access: Working Draft" date: 2007-10-01 categories: --- Il Web Application Formats Working Group ha rilasciato un Working Draft aggiornato di Enabling Read Access for Web Resources. Con questo meccanismo le restrizioni sandbox sull'accesso cross-site ai browser possono essere rilasciate selettivamente. Un header HTTP o XML o entrambi, che processino le istruzioni, possono indicare che l'accesso in lettura è consentito. Maggiori informazioni sulla Rich Web Clients Activity home page.
