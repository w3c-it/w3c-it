--- title: "XML Base Proposed Edited Recommendation: Call for Review" date: 2006-12-20 layout: page categories: --- L'XML Core Working Group ha pubblicato una Proposed Edited Recommendation per XML Base (Second Edition). XML Base descrive il vantaggio, simile a quello di HTML BASE, di definire URIs di base per parti di documenti XML. I commenti sono benvenuti fino al 31 gennaio 2007. Maggiori informazioni sulla XML Core home page. (Archivio News inglese, italiano)
