--- title: "RDF Core Working Group Pubblica Sei Working Draft" date: 2003-09-05 layout: page categories: w3c ---

RDF Core Working Group ha pubblicato sei Working Draft in risposta ai commenti del Last Call. Resource Description Framework (RDF) supporta lo scambio di conoscenza sul Web. Pubblicato come Working Group Note c' è anche LBase, un framework per specificare il significato di linguaggi del Semantic Web. Per maggiori informazioni: Semantic Web Activity

.

 

I documenti pubblicati sono:

  • RDF Primer
  • RDF Concepts and Abstract Syntax
  • RDF Semantics
  • RDF/XML Syntax Specification (Revised)
  • RDF Vocabulary Description Language 1.0: RDF Schema
  • RDF Test Cases
