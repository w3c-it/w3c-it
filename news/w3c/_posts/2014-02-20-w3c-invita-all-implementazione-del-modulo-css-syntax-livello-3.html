--- title: "W3C Invita all'Implementazione del Modulo CSS Syntax Livello 3" date: 2014-02-20 layout: page categories: w3c ---

Il Cascading Style Sheets (CSS) Working Group invita all'implementazione della Raccomandazione Candidata sul Modulo CSS Syntax Livello 3. Questo modulo descrive, in termini generali, la struttura di base e la sintassi dei fogli di stile CSS. Definisce, nel dettaglio, la sintassi e il parsing del CSS - come trasformare un flusso di bytes in un foglio di stile significativo. Il CSS è un linguaggio per la descrizione del rendering di documenti strutturati (come HTML e XML) su schermo, carta, discorsi, etc. Per saperne di più circa la Style Activity.
