--- title: "WebRTC 1.0: pubblicata draft su Real-time Communication tra Browsers " date: 2013-09-10 layout: page categories: ---

Il Web Real-Time Communications Working Group ha pubblicato una Working Draft su WebRTC 1.0: Real-time Communication tra Browsers. Questo documento definisce un set di ECMAScript APIs in WebIDL che consente di inviare e ricevere media da e verso un altro browser o dispositivo che implementi un appropriato set di protocolli real time. Queste specifiche sono state sviluppate unitamente al protocollo per le specifiche sviluppato dal IETF RTCWEB group e ad una API specification che consente l'accesso ai media del dispositivo sviluppata dalla Media Capture Task Force. Maggiori informazioni su Ubiquitous Web Applications Activity.
