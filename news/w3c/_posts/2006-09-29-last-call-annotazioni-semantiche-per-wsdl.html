--- title: "Last Call: Annotazioni semantiche per WSDL" date: 2006-09-29 layout: page categories: --- Il Semantic Annotations for Web Services Description Language (SAWSDL) Working Group ha rilasciato il Last Call Working Draft di Semantic Annotations for WSDL. Con questi attributi, le annotazioni semantiche possono essere aggiunte ai componenti Web Service Description Language (WSDL) per l'utilizzo in attività di classificazione, scoperta, matching, composizione e invocazione dei Web service. I commenti sono benvenuti fino all'1 novembre. Il gruppo ha rilasciato anche il First Public Working Draft del compagno Usage Guide. Maggiori informazioni sulla pagina dei Web service.
