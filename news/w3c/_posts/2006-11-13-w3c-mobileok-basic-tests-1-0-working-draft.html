--- title: "W3C mobileOK Basic Tests 1.0 Working Draft" date: 2006-11-13 categories: --- Il Mobile Web Best Practices Working Group ha rilasciato il First Public Working Draft di W3C mobileOK Basic Tests 1.0. Il documento definisce i test che forniscono le basi per rendere una richiesta compatibile con W3C mobileOK Basic e che sono fondate sulle Mobile Web Best Practices del W3C. Maggiori informazioni sulla pagina del Mobile Web Best Practices Working Group. (Archivio News inglese, italiano)
