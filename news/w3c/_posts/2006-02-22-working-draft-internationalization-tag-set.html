--- title: "Working Draft: Internationalization Tag Set" date: 2006-02-22 layout: page categories: w3c --- L'Internationalization Tag Set Working Group ha pubblicato un Working Draft aggiornato di Internationalization Tag Set (ITS). Organizzato per categorie di dati, questo set di elementi ed attributi supporta l'internazionalizzazione e la localizzazione di schemi e documenti. Le implementazioni sono fornite per DTDs, XML Schema e Relax NG, e per i vocabolari esistenti quali XHTML, DocBook ed OpenDocument. Maggiori informazioni sulla Internationalization home page.
