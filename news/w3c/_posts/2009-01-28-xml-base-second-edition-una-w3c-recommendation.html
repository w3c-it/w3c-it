--- title: "XML Base (Second Edition) è una W3C Recommendation" date: 2009-01-28 layout: page categories: --- L'XML Core Working Group ha pubblicato la W3C Recommendation di XML Base (Second Edition). Questo documento descrive un meccanismo per fornire servizi base URI ad XLink, ma come una specifica modulare in modo che altre applicazioni XML possano trarre beneficio da controlli addizionali sulle URI relative ma non costruite su XLink. La sintassi consiste di un unico attributo XML chiamato xml:base. Le funzionalità sono simili all'elemento HTML base. Questo documento è parte del lavoro del W3C per mantenere le tecnologie core XML. Ulteriori informazioni su Extensible Markup Language (XML) Activity. (Archivio News Inglese, Italiano)
