--- title: "Pubblicato il draft di HTML Image Description Extension" date: 2013-03-12 layout: page categories: w3c ---

12 marzo 2013

L'HTML Working Group ha reso pubblico oggi la HTML Image Description Extension come primo Working Draft pubblico. Questa specifica ("HTML-longdesc") consente agli autori di contenuti web di fornire descrizioni testuali più lunghe per immagini complesse. Viene definito l'attributo "longdesc" per collegare descrizioni e immagini in contesto HTML5. HTML-longdesc é una estensione di specifica che fa parte della famiglia di specifiche HTML5, la quale consente una evoluzione indipendente e una finalizzazione più rapida. É sviluppata dalla HTML Accessibility Task Force in coordinamento con l'HTML WG e il WAI Protocols and Formats Working Group (PFWG). HTML-longdesc é parte del lavoro del W3C per assicurare che la Open Web Platform sia accessibile a persone disabili. Importanti informazioni aggiuntive si possono trovare nella e-mail di call for review . Seguite i link per saperne di più sulla HTML Activity e la Web Accessibility Initiative (WAI).
