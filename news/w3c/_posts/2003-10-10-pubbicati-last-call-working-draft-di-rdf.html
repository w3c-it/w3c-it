--- title: "Pubbicati Last Call Working Draft di RDF" date: 2003-10-10 layout: page categories: ---

RDF Core Working Group ha pubblicato il secondo Last Call di sei Working Draft. Resource Description Framework (RDF) supporta lo scambio di conoscenza sul Web. Si attendono commenti entro il 7 novembre. È stato inoltre aggionato il Working Group Note LBase, un framework per specificare il significato di linguaggi per il Semantic Web. Per maggiori informazioni: Semantic Web Activity.
I documenti sono:

 

  • RDF Primer
  • RDF Concepts and Abstract Syntax
  • RDF Semantics
  • RDF/XML Syntax Specification (Revised)
  • RDF Vocabulary Description Language 1.0: RDF Schema
  • RDF Test Cases
