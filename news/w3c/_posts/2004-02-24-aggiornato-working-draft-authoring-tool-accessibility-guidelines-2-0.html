--- title: "Aggiornato Working Draft: Authoring Tool Accessibility Guidelines 2.0" date: 2004-02-24 layout: page categories: ---

L' Authoring Tool Accessibility Guidelines Working Group ha pubblicato il secondo Working Draft di Authoring Tool Accessibility Guidelines 2.0. Le linee guida sono state scritte per aiutare gli sviluppatori a creare interfacce di authoring che producano contenuti Web accessibili. Il contenuto risultante può essere fruito da un ampio spettro di lettori. Per ulteriori informazioni: Web Accessibility Initiative home page.
