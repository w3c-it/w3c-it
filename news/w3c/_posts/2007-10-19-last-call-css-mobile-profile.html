--- title: "Last Call: CSS Mobile Profile" date: 2007-10-19 layout: page categories: --- Il CSS Working Group ha rilasciato un Working Draft CSS Mobile Profile 2.0. I commenti sono benvenuti fino al 15 novembre. Questo sottoinsieme di Cascading Style Sheets (CSS) 2.1 è una base per le implementazioni di CSS su dispositivi come telefoni mobili, scritti con WICD Mobile 1.0 per assicurare l'interoperabilità e per l'allineamento con la specifica OMA Wireless CSS 1.1. Maggiori informazioni sulla CSS home page. (Archivio News inglese, italiano)
