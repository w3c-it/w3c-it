--- title: "CSS Backgrounds and Borders Module Level 3 Draft Pubblicata" date: 2008-09-10 layout: page categories: w3c --- Il Cascading Style Sheets (CSS) Working Group ha pubblicato una Working Draft di CSS Backgrounds and Borders Module Level 3. CSS è un linguaggio che descrive il rendering di documenti strutturati (quali HTML e XML). Questa draft contiene le caratteristiche di CSS level 3 per quanto riguarda bordi e sfondi. Include ed estende le funzionalità di CSS level 2. la principale estensione rispetto a level 2 sono i bordi composti da immagini, box con sfondi multipli, box con angoli smussati e con ombre. Ulteriori informazioni su Style Activity. (Archivio News Inglese, Italiano)
