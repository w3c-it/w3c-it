--- title: "Mobile Web Application Best Practices Draft Pubblicata" date: 2008-12-22 layout: page categories: w3c --- Il Mobile Web Best Practices Working Group ha pubblicato una Working Draft di Mobile Web Application Best Practices. Questo documento specifica le Best Practices per lo sviluppo e il delivery di applicazioni Web su dispositivi mobili. La recommendation si espande per quanto riguarda gli statement creati in Mobile Web Best Practices 1.0 (BP1), specialmente per gli statement che sono collegati allo sfruttamento delle potenzialità dello strumento e alla consapevolezza del contesto di delivery. Inoltre, da quando fu scritta la BP1, il network e gli strumenti hanno continuato ad evolvere, con il risultato che un numero di Best Practices che erano state omesse dalla BP1 possono adesso essere inserite. Ulteriori informazioni su Mobile Web Initiative Activity. (Archivio News Inglese, Italiano)
