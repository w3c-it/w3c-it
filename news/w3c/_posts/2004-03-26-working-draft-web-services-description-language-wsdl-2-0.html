--- title: "Working Draft: Web Services Description Language (WSDL) 2.0" date: 2004-03-26 categories: ---

Il Web Services Description Working Group ha aggiornato due Working Draft del Web Services Description Language (WSDL) Version 2.0: Part 1: Core Language e Part 2: Message Patterns. WSDL è un modello e un formato XML per descrivere servizi di rete. I draft permettono funzionalità astratte e dettagli concreti, definiscono sequenza, cardinalità e criteri per processori conformi alle specifiche. Per ulteriori informazioni: Web Services.
