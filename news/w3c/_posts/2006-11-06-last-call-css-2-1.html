--- title: "Last Call: CSS 2.1" date: 2006-11-06 categories: --- Il CSS Working Group ha pubblicato un Last Call Working Draft di Cascading Style Sheets, livello 2, revisione 1 (CSS 2.1). I commenti sono benvenuti fino al 7 dicembre. CSS 2.1 è derivato e costruito per sostituire CSS2. Istantanea dell'uso del linguaggio CSS, la specifica aggiunge alcune feature richieste, fissa gli errata, e porta CSS2 in linea con le implementazioni. Maggiori informazioni sulla CSS home page.
