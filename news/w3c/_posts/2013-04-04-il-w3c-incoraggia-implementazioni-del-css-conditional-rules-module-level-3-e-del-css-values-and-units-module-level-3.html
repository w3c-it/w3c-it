--- title: "Il W3C incoraggia implementazioni del CSS Conditional Rules Module Level 3 e del CSS Values and Units Module Level 3" date: 2013-04-04 layout: page categories: w3c ---

Il Cascading Style Sheets (CSS) Working Group incoraggia implementazioni di due Candidate Recommendations:

  • CSS Conditional Rules Module Level 3. Questo modulo contiene le componenti del CSS per l'elaborazione condizionale di parti di fogli di stile, dipendentemente dalle capacità del processore o in base al documento al quale il foglio di stile viene applicato. Include ed estende le funzionalità del CSS level 2, che a sua volta si basa sul CSS level 1. Rispetto al level 2, le principali estensioni riguardano l'annidamento di certe "at-rules" all'interno di ‘@media’ e l'aggiunta della regola ‘@supports’ per l'elaborazione condizionale.
  • CSS Values and Units Module Level 3. Questo modulo CSS3 descrive i valori comuni e le unità che le proprietà CSS accettano, e la sintassi usata per descriverli nella definizione delle proprietà CSS. Questo aggiornamento chiarifica che la dimensione della viewport usata dalle unità 'vw' e 'vh' include la dimensione delle scrollbar solo quando le scrollbar sono "automatiche" (regola CSS 'overflow: auto'), e che la definizione corrispondente della dimensione dellla viewport per media paginati *non* è parte di questo modulo.

Ulteriori informazioni riguardo la Style Activity.
