--- title: "Device Description Repository Simple API è una W3C Recommendation" date: 2008-12-08 categories: --- Il Mobile Web Initiative Device Description Working Group ha pubblicato la W3C Recommendation di Device Description Repository Simple API. Il contenuto web mandato ai dispositivi mobili, spesso trae benefici dall'essere ritagliato per considerare una gamma di fattori come la grandezza dello schermo, supporto a linguaggi di markup e formati di immagine. Questo informazioni vengono conservate su "Device Description Repositories" (DDRs). Questo documento descrive una semplice API per accedere a DDRs, per facilitare e promuovere lo sviluppo di contenuti Web che si adattano al loro Delivery Context. Il Working Group pubblicata anche una Group Note di Device Description Structures, un'espressione del linguaggio per la categorizzazione strutturata di dispositivi nell'adattabilità dei contenuti. Ulteriori informzioni su Mobile Web Initiative Activity. (Archivio News Inglese, Italiano)
