--- title: "Pubblicata Draft su MediaStream Recording" date: 2013-02-05 categories: ---

Il Web Real-Time Communication Working Group, insieme al Device APIs Working Group, ha pubblicato la prima Working Draft pubblica su MediaStream Recording. Il documento definisce la recording API da usare con MediaStream, così come specificato in Media Capture and Stream. Con questa API si vuole rendere semplice la registrazione base, consentendo comunque la realizzazione di use case più complessi. Il contenuto della registrazione sarà reso disponibile, nel formato usato di default dalla piattaforma, attraverso il datavailable event. Sono disponibili funzioni per interrogare i set di encoding messi a disposizione dalla piattaforma e per selezionare quelli preferiti dall'autore. L'applicazione può scegliere anche quanti dati ricevere ogni volta.
