--- title: "Il CSS Working Group pubblica gli aggiornamenti di quattro draft" date: 2012-08-23 layout: page categories: ---

Il Cascading Style Sheets (CSS) Working Group ha pubblicato oggi quattro Working Draft:

  • CSS Fonts Module Level 3. Questo modulo CSS3 descrive come si specificano le proprietà dei font e come avviene il caricamento dinamico dele risorse relative. Il contenuto di questa specifica è un consolidamento dei contenuti precedentemente suddivisi tra i moduli CSS3 Fonts e CSS3 Web Fonts.
  • CSS Regions Module Level 3. Il modulo CSS regions consente di far fluire i contenuti tra più aree, denominate region. Le region non sono necessariamente contigue nell'ordine in cui sono specificate nel documento. Il modulo CSS regions consente di gestire il flusso di contenuti con un meccanismo sofisticato, che può essere combinato con gli schemi di posizionamento definiti in altri moduli CSS quali il Multi-Column Module o il Grid Layout Module per posizionare le regioni in cui fluisce il contenuto.
  • Selectors Level 4. I selettori sono dei modelli che devono corrispondere a alementi in un albero, e costituiscono una delle tecnologie che è possibile utilizzare per per selezionare dei nodi in un documento XML. I selettori sono stati ottimizzati per essere utilizzati con HTML e XML, e sono progettati per essere utilizzati in software con requisiti critici per quanto concerne la performance. Sono un componente fondamentale del CSS (Cascading Style Sheets), che utilizza i selettori per legare le proprietà di stile agli elementi dei documenti.
  • CSS Fragmentation Module Level 3. Questo modulo descrive il modello di frammentazione che suddivide il flusso tra varie pagine. Basato sul modulo Page model,  introduce e definisce il modello di frammentazione. Aggiiunge funzionalità per la paginazione, la suddivisione in frammenti di dimensione e orientamento variabili, e la gestione delle righe isolate, come titoli senza contenuto nella stessa pagina (widow) e inizio di paragrafi senza il relativo titolo (orphan).

Per approfondimenti si consulti la pagina Style Activity.
