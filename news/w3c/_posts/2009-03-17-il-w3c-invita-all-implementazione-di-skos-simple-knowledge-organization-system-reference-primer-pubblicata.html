--- title: "Il W3C invita all'implementazione di SKOS (Simple Knowledge Organization System) Reference; Primer pubblicata" date: 2009-03-17 layout: page categories: --- Il Semantic Web Deployment Working Group invita all'implementazione della Candidate Recommendation di SKOS Simple Knowledge Organization System Reference. Questo documento definisce Simple Knowledge Organization System (SKOS), un modello dei dati comune per sistemi di organizzazione per la condivisione ed il collegamento di conoscenza via Web. SKOS?Simple Knowledge Organization System?fornisce un modello per esprimere la struttura base ed il contenuto di schemi concettuali come thesauri, schemi di classificazione, liste subject heading, tassonomie, folksonomie, ed altri tipi di vocabolari controllati simili. Come applicazione di Resource Description Framework (RDF), SKOS permette di comporre concetti e pubblicarli sul World Wide Web, collegarli con dati sul Web e integrarli in altri schemi concettuali. Oggi il Working Group ha anche pubblicato una Working Draft di SKOS Simple Knowledge Organization System Primer. Ulteriori informazioni su Semantic Web Activity. (Archivio NewsInglese, Italiano)
