--- title: "Last Call: Web Services Addressing Metadata" date: 2007-05-16 categories: --- Il Web Services Addressing Working Group ha rilasciato un terzo Last Call Working Draft di Web Services Addressing 1.0 - Metadata per preparare la specifica che permetta l'avanzamento allo stato di Candidate Recommendation. Usata per descrivere il supporto per il meccanismo Web Services Addressing 1.0 utilizzando il Web Services Policy 1.5 framework e per definire come esprimere le proprietà WS-Addressing nel Web Services Description Language (WSDL), questa nuova versione rifinisce le asserzioni Web Services Policy. I commenti sono benvenuti fino all'8 giugno. Maggiori informazioni sulla pagina dei Web service. (Archivio News inglese, italiano)
