--- title: "Call for Review: pubblicata Web Storage Proposed Recommendation " date: 2013-04-09 categories: ---

Il Web Applications Working Group ha pubblicato una Proposed Recommendation relativa al Web Storage. Questa specifica definisce le API per il data storage permanente di coppie chiave-valore nei Web client. Si accettano commenti fino al 7 Maggio 2013. Maggiori informazioni su Rich Web Client Activity.
