--- title: "User Agent Accessibility Guidelines (UAAG) 2.0 Updated; Pubblicata la Nuova Guida d'Implementazione" date: 2010-03-11 categories: ---

Il User Agent Accessibility Guidelines Working Group ha pubblicato un Working Draft Aggiornato del User Agent Accessibility Guidelines (UAAG) 2.0. UAAG definisce come i browsers, media players, e altri "user agents" dovrebbero supportare l'accessibilità per le persone con disabilità e lavorare con le tecnologie assistive. Questo bozza aggiunge requisiti in sette nuove aree, incluso il supporto per l'input vocale, controlli video playback e una nuova sezione sulla conformità. Introduce un nuovo documento di supporto, Implementing UAAG 2.0 come un First Public Working Draft. Si legga invitation to review the UAAG 2.0 Working Draft e Web Accessibility Initiative (WAI).
