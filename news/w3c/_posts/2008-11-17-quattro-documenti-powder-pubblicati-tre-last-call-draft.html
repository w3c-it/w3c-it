--- title: "Quattro documenti POWDER pubblicati; tre Last Call Draft" date: 2008-11-17 layout: page categories: --- Il Protocol for Web Description Resources (POWDER) Working Group oggi ha pubblicato quattro Working Drafts. Lo scopo di Protocol for Web Description Resources (POWDER) è di fornire un significato per individui o organizzazioni per descrivere un gruppo di risorse attraverso la pubblicazione di metadati machine-readable.
  • Description Resources (Last Call); che mette a fuoco la crezione ed il ciclo di vita di Description Resources (DRs), che encapsula i metadati
  • Grouping of Resources (Last Call); che descrive come insiemi di IRIs possano essere definiti come descrizioni o come altri dati possano essere applicati alle risorse ottenute dereferenziando IRIs che sono elementi dell'insieme.
  • Formal Semantics (Last Call); che descrive come il formato operazionale relativamente semplice di un documento POWDER possa essere trasformato dagli strumenti per il Semantic Web
  • Primer (First Public Draft)

I commenti alla Last Call sono bene accetti fino al 5 dicembre. Ulteriori informazioni su Semantic Web Activity. (Archivio News Inglese, Italiano)
