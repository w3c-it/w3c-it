--- title: "Pubblicata draft del modulo CSS Ruby Layout Module Level 1 " date: 2014-08-05 layout: page categories: w3c ---

Il Cascading Style Sheets (CSS) Working Group ha pubblicato una Working Draft su CSS Ruby Layout Module Level 1. Le “Ruby”, forma di annotazione interlineare, sono brevi serie di testi affiancate al testo base. Sono usate, tipicamente, per indicare la pronuncia in documenti per l'Est Asia, o per fornire brevi annotazioni. Questo modulo descrive il rendering model e i controlli di formattazione relative al displaying delle ruby annotation in CSS. CSS è un linguaggio usato per descrivere il rendering di documenti strutturati (come HTML e XML) su schermi, carta, ecc. Maggiori informazioni su Style Activity.
