--- title: "Pubblicato il draft Runtime e Modelli di Sicurezza per Applicazioni Web" date: 2013-03-21 layout: page categories: w3c ---

Il System Applications Working Group ha pubblicato una bozza di lavoro sul Runtime e Security Model per Web Applications. Questo documento specifica un modello di runtime e uno di sicurezza per le applicazioni Web. Descrive come definire un'applicazione tramite un manifesto dell'applicazione, e come questa può essere installata, aggiornata e inpacchettata. Specifica anche come tale applicazione può essere messa in secondo piano (background), in primo piano o riattivata. Infine, il documento descrive il modello di sicurezza per questo tipo di applicazione. Questo include il modello di permesso e le differenti regole di sicurezza da applicare. Per saperne di più Ubiquitous Web Applications Activity.
