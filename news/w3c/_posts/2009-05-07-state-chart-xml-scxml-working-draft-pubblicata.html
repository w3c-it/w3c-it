--- title: "State Chart XML (SCXML) Working Draft pubblicata" date: 2009-05-07 layout: page categories: w3c --- Il Voice Browser Working Group ha pubblicato una Working Draft aggiornata di State Chart XML (SCXML): State Machine Notation for Control Abstraction. SCXML è un linguaggio per macchine a stati general-purpose basato sugli eventi che può essere usato in molti modi, incluso come linguaggio per dialoghi ad alto livello per controllare i moduli per la voce di VoiceXML 3.0, o come linguaggio per il controlo multimodale nei framework di interazione multimodale. Le principali differenze dalle draft precedenti sonon (1) una revisione degli elementi send e invoke e (2) l'introduzione di processori Event I/O processors per supportarli.
