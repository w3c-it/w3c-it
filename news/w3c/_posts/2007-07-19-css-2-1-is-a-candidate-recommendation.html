--- title: "CSS 2.1 Is a Candidate Recommendation" date: 2007-07-19 layout: page categories: ---

W3C è lieto di annunciare l'avanzamento di Cascading Style Sheets (CSS) 2.1 allo stato di Candidate Recommendation. Feedback sull'implementazione sono benvenuti fino al 20 dicembre. CSS è uno dei linguaggi Web più ampiamente implementati. Con la separazione della presentazione dello stile dal contenuto dei documenti, CSS semplifica il Web authoring ed il mantenimento dei siti. CSS 2.1 è derivato ed è inteso come sostituto di CSS Livello 2. Istantanea di utilizzo, la specifica porta il linguaggio in linea con le implementazioni, fissa gli errata ed aggiunge poche feature molto richieste come inline-block valori per display property, colore orange e i valori pre-wrap e pre-line per la proprietà white-space.
