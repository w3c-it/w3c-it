--- title: "Representing Specified Values in OWL" date: 2005-05-19 layout: page categories: --- Il Semantic Web Best Practices and Deployment (SWBPD) Working Group ha rilasciato Representing Specified Values in OWL: "value partitions" e "value sets" come nota del Working Group. Prodotta dalla Ontology Engineering and Patterns Task Force, la nota descrive due metodi per rappresentare le caratteristiche descrittive nel OWL Web Ontology Language: partizioni di classi ed enumerazione di entità singole. Maggiori informazioni sulla Semantic Web home page. (Archivio News inglese, italiano)
