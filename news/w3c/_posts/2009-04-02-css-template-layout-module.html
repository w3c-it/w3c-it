--- title: "CSS Template Layout Module" date: 2009-04-02 layout: page categories: --- Il Cascading Style Sheets (CSS) Working Group ha pubblicato la Working Draft di CSS Template Layout Module. Questa specifica è parte del level 3 di CSS (?CSS3?) e contiene alcune caratteristiche per descrivere layout ad alto livello, da usare per il posizionamento e l'allineamento di ?widgets? nelle interfacce grafiche o nella griglia di layout per la pagina o la finestra, in particolare quando l'ordine di visualizzazione è diverso dall'ordine degli elementi nel documento sorgente. Ulteriori informazioni su Style Activity. (Archivio News Inglese, Italiano)
