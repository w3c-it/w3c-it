--- title: "Il W3C invita a Commenti sul First Draft di XML Processor Profiles " date: 2010-05-18 categories: ---

L' XML Processing Model Working Group ha pubblicato il First Public Working Draft di XML processor profiles. Questa specifica definisce diversi profili di XML processor, ciascuno dei quali determina del tutto un modello di dati per ogni documento XML dato. Esso è inteso come una risorsa per altre specifiche, che, da un unico riferimento normativo,  possono stabilire con precisione di quale procedimento di input necessitano. Ulteriori informazioni su Extensible Markup Language (XML) Activity.
