--- title: "CSS3 Media" date: 2007-06-06 categories: ---

Il CSS Working Group ha rilasciato una Candidate Recommendation aggiornata per modifiche editoriali a Media Queries, modulo di Cascading Style Sheets Level 3 (CSS3). Costruito sul meccanismo delineato in HTML, propone un registro di media type per descrivere come si applica lo style sheet, ed espressioni per definirne lo scopo. Le presentazioni possono essere adattate ad uno specifico range di dispositivi di output senza modificare i contenuti. Maggiori informazioni sulla CSS home page.(Archivio News inglese, italiano)
