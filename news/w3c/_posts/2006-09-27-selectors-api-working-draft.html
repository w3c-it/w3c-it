--- title: "Selectors API: Working Draft" date: 2006-09-27 layout: page categories: --- Il Web APIs Working Group ha rilasciato un Working Draft aggiornato di Selectors API. Sono definiti metodi per identificare elementi in un documento, con lo scopo di eseguire script o operazioni Document Object Model (DOM) su di esso. Le specifiche dei selettori definiti in CSS3 Selectors, sono utilizzate per identificare gli elementi. Maggiori informazioni sulla Web APIs Working Group home page. (Archivio News inglese, italiano)
