--- title: "Last Call: Web Services SOAP Assertions (WS-SOAPAssertions) " date: 2010-07-13 layout: page categories: w3c ---

Il Web Services Resource Access Working Group ha pubblicato un Last Call Working Draft di Web Services SOAP Assertions (WS-SOAPAssertions). Questa specifica definisce due asserzioni WS-Policy che possono essere usate per pubblicizzare l'obbligo di utilizzare una determinata versione di SOAP negli scambi di messaggi. Commenti sono benvenuti fino al 27 agosto. Il gruppo ha anche pubblicato oggi un Web Services Resource Transfer (WS-RT) come Note. Questa specifica definisce le estensioni di WS-Transfer principalmente per fornire l'accesso fragment-based alle risorse. Per saperne di più si veda Web Services Activity.
