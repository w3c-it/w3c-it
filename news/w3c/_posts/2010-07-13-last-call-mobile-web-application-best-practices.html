--- title: "Last Call: Mobile Web Application Best Practices" date: 2010-07-13 layout: page categories: w3c ---

Il Mobile Web Best Practices Working Group ha pubblicato un Last Call Working Draft di Mobile Web Application Best Practices. L'obiettivo di questo documento è sostenere lo sviluppo di applicazioni “web mobile” ricche e dinamiche. Esso raccoglie le pratiche ingegneristiche più rilevanti, promuovendo quelle che permettono una migliore esperienza per l'utente e diffidando di quelle che sono considerate dannose. Il Mobile Web Best Practices Working Group non si aspetta più cambiamenti sostanziali a questo punto, ma si aspetta di essere in grado di passare direttamente alla Proposed Recommendation, alla fine del periodo di revisione della Last Call. Si accettano commenti sulla Last Call commenti fino al 6 Agosto 2010 . Per ulteriori informazioni si consulti Mobile Web Initiative Activity.
