--- title: "Publicati il draft di CSS Paged Media Module, e il CSS Print Profile Note " date: 2013-03-14 layout: page categories: ---

14 marzo 2013

Oggi il gruppo di lavoro sul Cascading Style Sheets (CSS) ha publicato due documenti:

  • Working draft del CSS Paged Media Module Level 3. Questo modulo descrive il modello di pagina che partiziona un flusso continuo in pagine. É costruito sul modello del modulo Box e introduce e definisce il modello pagina e i media paginati. Aggiunge funzionalità per paginazione, margini di pagina, dimensioni e orientamento di pagina, header e footer, widow e orphan, e orientamento di immagini. Infine, estende il contenuto generato per consentire la numerazione delle pagine e gli header / footer.
  • Group Note del CSS Print Profile. Questa specifica definisce un sottoinsieme del Cascading Style Sheets Level 2, revision 1 e CSS Paged Media Level 3 per la stampa da molteplici dispositivi e diversi tipi di media. É progettata per la stampa in situazioni dove non é fattibile o desiderabile installare un driver di stampante specifico, e per situazioni in cui é accettabile una qualche variabilità nell'output. Questo profilo é progettato per funzionare in concomitanza con XHTML-Print e definisce un livello minimo di conformità insieme a un set di estensioni che consente un maggiore controllo del layout per la stampa di testo misto a immagini, tabelle e collezioni di immagini.

Per saperne di più vai a Style Activity.
