--- title: "Prima Bozza Pubblica su IndieUI: User Context per le preferenze sulle Interfacce Web" date: 2014-06-26 layout: page categories: ---

L' IndieUI Working Group ha pubblicato oggi una prima Bozza di Lavoro su IndieUI: User Context 1.0 – Contextual Information for User Interface Independence. Essa definisce una serie di preferenze che gli utenti possono scegliere di esporre alle applicazioni web, e una API per gli user agents per accedere alle preferenze e recepire le modifiche. Gli utenti possono impostare le preferenze per caratteristiche quali la dimensione dello schermo, dimensione del carattere e il colore. Gli utenti con disabilità possono fornire informazioni sulle tecnologie assistive in uso, indicare che il display è in modalità di accessibilità, e indicare di che tipo di sottotitoli e descrizioni audio hanno bisogno. Le applicazioni Web possono utilizzare queste informazioni per ottimizzare la presentazione senza l'obbligo di soddisfare uno specifico dispositivo, un sistema operativo o locale. IndieUI: User Context complements IndieUI: Events 1.0 per fornire agli sviluppatori di applicazioni web un modo uniforme per progettare applicazioni web che lavorano in una vasta gamma di contesti - dispositivi differenti (come i telefoni cellulari e tablet), diverse tecnologie assistive (AT) e diverse esigenze degli utenti. I commenti su questo progetto sono benvenuti fino al 1 Agosto 2014. Per saperne di più su IndieUI Overview; leggi riguardo a  Web Accessibility Initiative (WAI).
