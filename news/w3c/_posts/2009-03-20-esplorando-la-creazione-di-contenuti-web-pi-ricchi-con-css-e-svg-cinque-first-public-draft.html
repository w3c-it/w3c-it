--- title: "Esplorando la creazione di contenuti Web più ricchi con CSS e SVG: cinque First Public Draft" date: 2009-03-20 categories: --- Il CSS e l' SVG Working Groups oggi hanno rilasciato cinque nuove specifiche per la review pubblica, mirate alla possibilità di creare contenuti più ispirati alle tecnologie Web open. Le cinque draft sono: SVG Transforms 1.0, Part 2: Language, CSS 2D Transforms Module Level 3, CSS 3D Transforms Module Level 3, CSS Animations Module Level 3, e CSS Transitions Module Level 3. SVG Transforms permette di trasformare oggetti 2D usando trasformazioni tridimensionali. CSS 2D Transforms agli elementi renderizzati da CSS di essere trasformati in uno spazio 2D. CSS 3D Transforms estende CSS Transforms per permettere agli elementi renderizzati tramite CSS di essere trasformati in uno spazio 3D. CSS Animations permette di modificare proprietà CSS nel tempo. CSS Transitions permette di cambiare le proprietà nei valori di CSS specificando una durata. I gruppi stanno lavorando insieme per rendere l'implementazione e la creazione facile e coerente per i linguaggi Web. Ulteriori informazioni su Style Activity e su Graphics Activity.
