--- title: "Last Call: Ontology for Media Resource 1.0, API for Media Resource 1.0" date: 2010-06-08 layout: page categories: w3c ---

Il Media Annotations Working Group ha pubblicato un Last Call Working Draft di Ontology for Media Resource 1.0 e API for Media Resource 1.0. Il primo documento definisce l'Ontologia per Media Resource 1.0. Questa ontologia, o “vocabolario di base”, è destinata a colmare le differenze tra le varie descrizioni di risorse multimediali sul Web, in opposizione alle risorse multimediali in archivi locali o musei. Essa è definita sulla base di un insieme di proprietà che copre i metadati di base per descrivere le risorse multimediali. L'ontologia è stata progettata per favorire l'interoperabilità tra i vari tipi di formati di metadati attualmente utilizzati per descrivere le risorse multimediali sul Web. La seconda specifica definisce un' API lato client per accedere alle informazioni di metadati relative alle risorse multimediali sul web. Sono benvenuti commenti fino all'11 luglio.
