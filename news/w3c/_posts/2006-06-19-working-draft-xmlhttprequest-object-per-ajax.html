--- title: "Working Draft: XMLHttpRequest Object per AJAX" date: 2006-06-19 layout: page categories: ---

Il Web API Working Group ha rilasciato un Working Draft aggiornato di The XMLHttpRequest Object. Il draft documenta le caratteristiche dell'oggetto XMLHttpRequest, il componente base di AJAX. L'interfaccia permette agli script di migliorare le funzioni del client HTTP, come ad esempio presentare form data o caricare dati da un sito Web remoto. Maggiori informazioni sulla pagina della Rich Web Clients Activity. (Archivio News inglese, italiano)
