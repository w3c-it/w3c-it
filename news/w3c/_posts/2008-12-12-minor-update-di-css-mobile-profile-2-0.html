--- title: "Minor Update di CSS Mobile Profile 2.0" date: 2008-12-12 layout: page categories: w3c --- Il Cascading Style Sheets (CSS) Working Group ha aggiornato la Candidate Recommendatino di CSS Mobile Profile 2.0. La specifica definisce in generale un sottoinsieme di CSS 2.1 [CSS21] che deve essere considerato una baseline per l'interoperabilità fra le implementazioni di CSS su dispositivi limitati (ad esempio i telefoni mobili). Il suo scopo non è di produrre un profilo di CSS incompatibile con la specifica completa, piuttosto di assicurare che le implementazioni che a causa delle limitazioni della piattaforma non possono supportare l'intera specifica implementino un sottoinsieme comune che è interoperabile non solo fra le implementazioni limitate ma anche con quelle complete. Questo documento è lo stesso dell'ultima working draft, ad eccezione per i cambiamenti editoriali. Ulteriori informazioni su Style Activity. (Arhivio News Inglese, Italiano)
