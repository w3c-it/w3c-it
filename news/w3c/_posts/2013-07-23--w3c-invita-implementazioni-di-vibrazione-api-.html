--- title: " W3C invita Implementazioni di Vibrazione API " date: 2013-07-23 categories: ---

Il Device APIs Working Group invita implementazioni del Candidate Recommendation del Vibration API. Questa specifica definisce un API che fornisce accesso al meccanismo di vibrazione del dispositivo che ospita. La vibrazione è una forma di feedback tattile. Per saperne di più su Ubiquitous Web Applications Activity.
