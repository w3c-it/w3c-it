--- title: "XBL 2.0 Primer: An Introduction for Developers" date: 2007-07-20 layout: page categories: ---

Il Web Application Formats Working Group ha pubblicato il First Public Working Draft di XBL 2.0 Primer: An Introduction for Developers. Questa guida pratica per utilizzare l'XML Binding Language introduce concetti base ed avanzati e descrive le buone prassi. XBL estende l'aspetto e il comportamento degli elementi in formati Web come HTML. Maggiori informazioni sulla Rich Web Client Activity home page. (Archivio News inglese, italiano)
