--- title: "Aggiornato lo Working Draft CSS Grid Layout Module Livello 1" date: 2014-01-23 layout: page categories: w3c ---

Il CSS Working Group ha pubblicato uno Working Draft aggiornato su CSS Grid Layout Module Level 1. Questo modulo CSS definisce un sistema di layout bi-dimensionale basato su griglia., ottimizzato per design dell'interfaccia utente. Nel modello di layout a griglia, i figli di un contenitore griglia possono essere posizionati in slot arbitrari in una predefinita griglia di layout flessibile o fissa. CSS è un linguaggio per descrivere il rendering di documenti strutturati (come HTML e XML) su schermo, carta, discorsi, etc. Per saperne di più circa la Style Activity.
