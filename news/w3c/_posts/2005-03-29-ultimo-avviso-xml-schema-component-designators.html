--- title: "Ultimo avviso: XML Schema Component Designators" date: 2005-03-29 categories: --- L'XML Schema Working Group ha rilasciato il Last Call Working Draft di XML Schema: Component Designators. I commenti sono benvenuti fino al 26 aprile. Il documento definisce uno schema per identificare i componenti di XML Schema, specificati nelle Raccomandazioni XML Schema Parte 1 e Parte 2.
