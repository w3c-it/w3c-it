--- title: "Pubblicati Working Draft di Web Services Description Language (WSDL) 2.0" date: 2003-11-10 layout: page categories: w3c ---

Il Web Services Description Working Group ha pubblicato due Working Draft di Web Services Description Language (WSDL) Version 2.0: Part 1: Core Language e Part 2: Message Patterns. WSDL è un modello e un formato XML per descrivere reti di servizi. Il linguaggio consente di avere stadi fondamentali separati per funzioni astratte e dettagli concreti. Per ulteriori informazioni: Web Services.
