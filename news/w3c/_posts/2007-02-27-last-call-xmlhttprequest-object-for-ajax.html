--- title: "Last Call: XMLHttpRequest Object for AJAX" date: 2007-02-27 categories: ---

The Web API Working Group has released a Last Call Working Draft of The XMLHttpRequest Object. Comments are welcome through 2 April. The core component of AJAX, the XMLHttpRequest object is an interface that allows scripts to perform HTTP client functions, such as submitting form data or loading data from a remote Web site.
