--- title: "Last Call: W3C mobileOK Basic Tests 1.0" date: 2007-01-30 layout: page categories: --- Il Mobile Web Best Practices Working Group ha rilasciato un Last Call Working Draft di W3C mobileOK Basic Tests 1.0. I commenti sono benvenuti fino al 6 marzo. Questi test forniscono le basi per la compatibilità W3C mobileOK Basic e sono basati sulle Mobile Web Best Practice del W3C. Maggiori informazioni sulla pagina del Mobile Web Best Practices Working Group. (Archivio News inglese, italiano)
