--- title: "Versioning XML Languages Using XML Schema 1.1" date: 2007-07-20 categories: --- L'XML Schema Working Group ha rilasciato un Working Draft aggiornato di Guide to Versioning XML Languages using XML Schema 1.1. XML Schema 1.1 introduce nuove feature che rendono più semplice definire linguaggi XML più flessibili, adatti a sopportare revisioni successive, in un modo compatibile con le nuove versioni. Scritta per sviluppatori di applicazioni e schemi, la guida mostra i nuovi meccanismi ed illustra numerose tecniche. Maggiori informazioni sulla XML home page. (Archivio News inglese, italiano)
