--- title: "API per la sicurezza nell'accesso ai dispositivi dal Web: Workshop report pubblicato" date: 2009-02-05 layout: page categories: w3c --- Oggi il W3C h pubblicato un report dal W3C Workshop on Security for Access to Device APIs from the Web. I partecipanti al Workshop hanno identificato diverse sfide come oggetto di lavoro ad alta priorità, fra le quali:
  • Dichiarazione delle API usate da applicazioni web e widgets
  • Descrizione delle politiche
  • Pattern dell'API e API concreta

Il W3C vi invita a seguire la discussione sulla mailing list pubblica public-device-apis@w3.org (archivio pubblico). Ulteriori informazioni su W3C Mobile Web Initiative. (Archivio News Inglese, Italiano)
