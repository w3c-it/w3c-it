--- title: "Pubblicata la Bozza su User Interface Security Directives per Politiche di Content Security " date: 2013-05-23 categories: ---

Il Gruppo di Lavoro su Web Application Security ha pubblicato una Bozza di Lavoro su User Interface Security Directives per politiche di Content Security. Questo documento definisce le direttive con cui il meccanismo di Content Security Policy dichiara un set di protezioni in ingresso per l'interfaccia utente di una risorsa web, definisce un set di euristiche di tipo non-normativo per gli User Agents del Web per implementare queste protezioni in ingresso e un meccanismo di report che informi quando queste protezioni risultano attivate. Per saperne di più vai a  Security Activity.
