--- title: "Ultima chiamata: State Chart XML (SCXML): State Machine Notation for Control Abstraction" date: 2013-08-01 categories: ---

Il Voice Browser Working Group ha pubblicato un Last Call Working Draft dal titolo State Chart XML (SCXML): State Machine Notation for Control Abstraction. Questo documento descrive l'SCXML, "State Chart extensible Markup Language". L'SCXML fornisce un ambiente di esecuzione a macchine a stati basato su CCXML e Harel State Tables. Si accettano commenti fino al 27 settembre. Ulteriori informazioni sulla Voice Browser Activity.
