--- title: "Pubblicato Scope per Mobile Web Best Practices" date: 2005-09-02 layout: page categories: --- Il Mobile Web Best Practices Working Group ha rilasciato il First Public Working Draft di Scope of Mobile Web Best Practices. Questo documento definisce le linee-guida consigliate per il trasferimento e la visualizzazione dei contenuti su dispositivi mobili o dotati di small-screen, precisa i requisiti per il "mobileOK" trustmark, ed identifica l'obiettivo di un solo Web. Maggiori informazioni riguardo la W3C Mobile Web Initiative, sforzo congiunto di fornitori di authoring tool, content provider, produttori di microtelefoni , fornitori di browser e operatori del mobile.
