--- title: "Efficient XML Interchange (EXI) Profile for limiting usage of dynamic memory è una W3C Recommendation" date: 2014-09-09 layout: page categories: ---

Il EXI Working Group ha pubblicato una nuova W3C Recommendation: Efficient XML Interchange (EXI) Profile for limiting usage of dynamic memory. EXI 1.0 è un efficiente formato di rappresentazione per l'XML Information Set, adattabile a diversi use case, dalle B2B applications agli embedded-systems. Soddisfa i requisiti di compattezza e processing efficiency, pur preservando tutte le inforazioni contenute nell'XML InfoSet. Essendo una rappresentazione di XML, è estendibile by design.

EXI è in grado di gestire scostamenti o estensioni dallo Schema in uso grazie al self-augmenting grammar system. Questo sistema può talvolta porre dei problemi se usato per low-memory systems, come sensori o microcontrollori embedded in power grids, veicoli o altri piccoli dispositivi.

Questo EXI Profile affronta queste problematiche consentendo alle applicazioni di definire profili interoperabili e adatti ai vincoli di memoria, pur mantenendo tutta la sue estensibilità.
