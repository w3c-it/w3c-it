--- title: "Ultima Call: Web Cryptography API" date: 2014-03-25 categories: ---

Il  Gruppo di Lavoro Web Cryptography ha pubblicato un'ultima Call per una Bozza di Lavoro su Web Cryptography API. Questa API JavaScript esegue operazioni crittografiche di base nelle applicazioni web,quali hashing, generazione e verifica della firma, crittografia e decrittografia. Inoltre, descrive come le applicazioni possono generare e /o gestire le chiavi necessarie per eseguire queste operazioni. I commenti sono benvenuti fino al 20 maggio 2014. Per saperne di più vai a Security Activity.
