--- title: "Candidate Recommendation: Web Services Description Language (WSDL) 2.0" date: 2006-01-06 categories: --- W3C è lieto di annunciare l'avanzamento delle specifiche Web Services Description Language (WSDL) Versione 2.0 allo stato di Candidate Recommendation: Part 0: Primer, Part 1: Core Language e Part 2: Adjuncts. I commenti sono benvenuti fino al 15 marzo. SOAP 1.1 Binding è un Working Draft aggiornato. WSDL 2.0 modella e descrive i servizi del modular Web ed è utilizzato per documentare sistemi distribuiti ed automatizzare le comunicazioni tra le applicazioni. Maggiori informazioni sulla pagina dei Web service. (Archivio News inglese, italiano)
