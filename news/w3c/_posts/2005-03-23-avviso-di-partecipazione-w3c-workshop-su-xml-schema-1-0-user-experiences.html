--- title: "Avviso di partecipazione: W3C Workshop su XML Schema 1.0 User Experiences" date: 2005-03-23 layout: page categories: --- Scadono il 20 maggio le position papers per il W3C Workshop su XML Schema 1.0 User Experiences che si terrà il 21-22 giugno a Redwood Shores, California, USA. Si riuniranno per discutere sulle conoscenza acquisite con l'utilizzo di XML Schema 1.0 autori ed utenti dello Schema, sviluppatori e rivenditori di schema-aware, generatori di codice, middleware e sistemi di validazione, oltre al W3C XML Schema Working Group. Obiettivo del workshop è un piano di lavoro relativamente ad interoperabilità, errori e chiarimenti su XML Schema 1.0. Visitate i W3C workshops e la XML home page. (Archivio News inglese, italiano)
