--- title: "POWDER Pubblicate le Draft: Grouping of Resources; Description Resources" date: 2008-06-30 categories: --- Il Protocol for Web Description Resources (POWDER) Working Group ha pubblicato due working draft di Protocol for Web Description Resources (POWDER): Grouping of Resources e Description Resources. Il primo documento descrive come pubblicare le descrizioni di risorse multiple quali tutte quelle disponibili su un sito Web. Queste descrizioni sono sempre attribuite ad una persona, organizzazione o ente designato che può o meno essere il creatore delle risorse descritte. La seconda pubblicazione fornisce un significato per persone o organizzazioni per creare descrizioni machine-readable. Ulteriori informazioni su Semantic Web Activity.
