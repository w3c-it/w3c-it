--- title: "Last Call per i Draft Relativi a XML Security: Encryption, Signature, Generic Hybrid Ciphers" date: 2010-05-17 layout: page categories: w3c ---

L' XML Security Working Group ha pubblicato tre Last Call Working Draft: XML Encryption Syntax and Processing Version 1.1, XML Signature Syntax and Processing Version 1.1 e XML Security Generic Hybrid Ciphers. Il gruppo ha anche pubblicato un Working Draft di XML Security Algorithm Cross-Reference. XML Encryption specifica un processo per cifrare i dati e rappresentare il risultato in XML. XML Signatures garantisce integrità, autenticazione dei messaggi e/o servizi di autenticazione di firme per dati di qualsiasi tipo, sia se localizzati dentro XML che include la firma sia altrove. Il terzo documento accresce la potenza di XML Encryption definendo gli algoritmi,  tipi XML e gli elementi necessari per consentire l'uso di algoritmi generici ibridi nelle applicazioni XML Security. Il documento finale sintetizza gli identificatori URI dell'algoritmo XML Security e le specifiche ad essi associati. Si accettano commenti al Last Call  fino al 10 giugno. Per ulteriori informazioni si consulti Security Activity.
