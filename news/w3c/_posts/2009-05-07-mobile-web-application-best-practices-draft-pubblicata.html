--- title: "Mobile Web Application Best Practices Draft pubblicata" date: 2009-05-07 layout: page categories: w3c --- Il Mobile Web Best Practices Working Group ha pubblicato una Working Draft di Mobile Web Application Best Practices. Questo documento specifica le Best Practices per lo sviluppo ed il delivery di applicazioni Web su dispositivi mobili. Le raccomandazioni espandono i contenuti di the Mobile Web Best Practices 1.0 (BP1), specialmente ciò che si riferisce allo sfruttamento delle capacità del dispositivo e alla conoscenza del contesto di dellivery. Ulteriori informazioni su Mobile Web Initiative Activity. (Archivio News Inglese, Italiano)
