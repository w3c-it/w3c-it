--- title: "Report: Current State and Roadmap of Standards for Web Applications on Mobile" date: 2013-10-01 layout: page categories: w3c ---

Il W3C ha pubblicato l'edizione di Settembre 2013 di Standards for Web Applications on Mobile, una overview sulle varie tecnologie sviluppate dal W3C per incrementare le potenzialità delle Web applications, soprattutto nel contesto mobile.

Questa edizione del documento, come deliverable del progetto webinos, include modifiche e aggiunte fino a Marzo 2013 quali, ad esempio, il recente Web and Mobile IG (compreso nella prossima versione del documento), migliorie delle performance attraverso una migliore integrazione dello scrolling e priorità nelle richieste della rete, nuovi approcci per una gestione più responsive di immagini e filesystem. Il report mostra inolotre graficamente le recenti attività editoriale per le specifiche trattate.
