--- title: "Il Web Applications Working Group pubblica quattro Draft" date: 2013-05-14 layout: page categories: ---

Il Web Applications Working Group ha pubblicato oggi quattro Working Drafts:

  • Una Working Draft su Shadow DOM. Questa specifica descive un metodo per stabilire e mantenere confini funzionali tra gli alberi DOM e come questi alberi interagiscono con i documenti al fine di rendere più funzionale l'incapsulazione con DOM.
  • Una Working Draft su Custom Elements. Questa specifica descrive un metodo per Questa specifica descrive il metodo per permettere all'autore di definire e utilizzare nuovi tipi di elementi DOM in un documento.
  • Una Working Draft su HTML Imports. Gli HTML Imports sono un modo per includere e riutilizzare documenti HTML in altri documenti HTML.
  • Una Group Note su Java language binding for Web IDL. Questo documento definisce il binding del linguaggio Java per il Web IDL, il linguaggio di definizione dell'interfaccia per la piattaforma web.
