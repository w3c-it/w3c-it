--- title: "Primo Working Draft Pubblico: Data on the Web Best Practices Use Cases and Requirements" date: 2014-06-05 layout: page categories: ---

Il Data on the Web Best Practices Working Group ha pubblicato un primo Working Draft pubblico di Data on the Web Best Practices Use Cases & Requirements. Questo documento contiene un elenco di alcuni casi d'uso, compilato dal Data on the Web Best Practices Working Group, che rappresentano degli scenari su come i dati vengono comunemente pubblicati sul Web e come sono usati. Questo documento fornisce anche un insieme di requisiti derivati da questi casi d'uso, che sono stati usati come guida per lo sviluppo delle best practice e per lo sviluppo di due vocabolari: Quality and Granularity Description Vocabulary e Data Usage Description Vocabulary. Per ulteriori informazioni consulta Data Activity.
