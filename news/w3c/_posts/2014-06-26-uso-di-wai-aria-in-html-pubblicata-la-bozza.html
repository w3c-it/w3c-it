--- title: "Uso di WAI-ARIA in HTML Pubblicata la Bozza" date: 2014-06-26 layout: page categories: ---

L' HTML Working Group ha pubblicato un aggiornamento su Using WAI-ARIA in HTML. L'uso di WAI-ARIA con HTML è una guida pratica per gli sviluppatori che possono così aggiungere informazioni di accessibilità agli elementi HTML usando le specifiche Accessible Rich Internet Applications (WAI-ARIA), le quali definiscono un modo per rendere i contenuti Web e le applicazioni Web più accessibili per persone con disabilità. WAI-ARIA è introdotta nel WAI-ARIA Overview. L'uso di  WAI-ARIA nell' HTML è sviluppato dal HTML Accessibility Task Force in coordinamento con l' HTML Working Group e WAI Protocols and Formats Working Group (PFWG). Per saperne di più vai a HTML Activity e a Web Accessibility Initiative (WAI).
