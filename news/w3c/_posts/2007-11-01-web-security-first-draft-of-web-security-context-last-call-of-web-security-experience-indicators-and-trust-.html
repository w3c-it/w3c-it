--- title: "Web Security: First Draft of \"Web Security Context\"; Last Call of \"Web Security Experience, Indicators and Trust\"" date: 2007-11-01 categories: --- Il Web Security Context Working Group ha pubblicato due documenti: il First Public Working Draft di Web Security Context: Experience, Indicators, and Trust, che definisce linee guida ed i requisiti per la presentazione e la comunicazione delle informazioni sui contenuti di sicurezza Web agli utenti finali, pratiche per un data entry sicuro e buone prassi per autori di siti Web. Il secondo documento è un Last Call Working Draft di Web Security Experience, Indicators and Trust: Scope and Use Cases, che aiuta a chiarire l'obiettivo del gruppo, come le tecnologie possono essere usate e come valutare le proposte tecniche. I commenti al Last Call sono benvenuti fino al 30 novembre. Per maggiori informazioni consulta anche il compagno del Last Call draft, Web User Interaction: Threat Trees, W3C Group Note. Maggiori informazioni sulla pagina della Security Activity. (Archivio News inglese, italiano)
