--- title: "EARL 1.0 Schema: Working Draft" date: 2006-09-27 layout: page categories: w3c --- L'Evaluation and Repair Tools Working Group ha rilasciato un Working Draft aggiornato di Evaluation and Report Language (EARL) 1.0 Schema. EARL è un formato flessibile utilizzato per scambiare, combinare e comparare risultati di test che includano anche bug report, test suite evaluation e dichiarazioni di conformità. I soggetti del test possono essere siti Web, authoring tool, user agent o altro. Per maggiori informazioni vedere EARL Overview. Il gruppo richiede feedback dagli sviluppatori Web e dai ricercatori. Maggiori informazioni sulla pagina della Web Accessibility Initiative. (Archivio News inglese, italiano)
