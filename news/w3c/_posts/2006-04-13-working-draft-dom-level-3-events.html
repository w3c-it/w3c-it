--- title: "Working Draft: DOM Level 3 Events" date: 2006-04-13 categories: --- Il Web API Working Group ha rilasciato un Working Draft di Document Object Model (DOM) Level 3 Events. Linguaggio e piattaforma neutrale, il sistema permette la registrazione di responsabili di eventi, descrive un flusso di eventi attraverso una struttura ad albero e fornisce contesti per ogni tipo di evento. La precedente versione di questo documento era una Working Group Note dal Document Object Model (DOM) Working Group. Maggiori informazioni sulla pagina della Rich Web Clients Activity.
