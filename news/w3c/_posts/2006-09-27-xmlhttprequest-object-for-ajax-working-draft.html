--- title: "XMLHttpRequest Object for AJAX: Working Draft" date: 2006-09-27 categories: ---

Il Web API Working Group ha rilasciato un Working Draft aggiornato di The XMLHttpRequest Object. Il draft illustra le caratteristiche dell'oggetto XMLHttpRequest, componente base di AJAX. L'interfaccia consente script per eseguire le funzioni HTTP client, come ad esempio presentare dati form o caricare dati da un sito web remoto. Maggiori informazioni sulla pagina della Rich Web Clients Activity. (Archivio News inglese, italiano)
