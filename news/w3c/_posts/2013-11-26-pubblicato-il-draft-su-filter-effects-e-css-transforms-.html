--- title: "Pubblicato il Draft su Filter Effects e CSS Transforms " date: 2013-11-26 layout: page categories: w3c ---

Il CSS Working Group e il SVG Working Group oggi hanno pubblicato due Working Drafts.

  • Uno Working Draft su Filter Effects Module Level 1. I Filiter Effects sono un modo di elaborare il rendering di un elemento prima che venga visualizzato nel documento. Tipicamente, rendere un elemento tramite CSS o SVG può essere concettualmente descritto come se l'elemento, compresi i suoi figli, sono disegnati in un buffer (come un'immagine raster) e poi quel buffer viene composto nel genitore degli elementi. I filtri applicano un effetto prima della fase di composing. Esempi di tali effetti sono la sfocatura, il cambio di intensità di colore e la deformazione dell'immagine. Sebbene originariamente progettati per l'uso in SVG, i filtri di effetti sono un set di operazioni da applicare su un buffer di immagine e quindi può essere applicato a quasi tutti gli ambienti di presentazione, incluso CSS. Essi sono attivati da un'istruzione di stile (la proprietà del filtro). Questa specifica descrive i filtri in un modo che permette loro di essere usati in contenuti a cui è applicato uno stile CSS, come HTML o SVG. Inoltre definisce una funzione di valore di proprietà CSS che produce un valore <image> CSS. CSS è un linguaggio per descrivere il rendering di documenti strutturati (come HTML e XML) su schermo, carta discorsi, etc...
  • Uno Working Draft su CSS Transforms Module Level 1. Le trasformazioni CSS permettono agli elementi di stile CSS di essere trasformati in uno spazio bi o tri-dimensionale. Questa specifica è la convergenza delle specifiche delle trasformazioni CSS 2D, CSS 3D e SVG. CSS è un linguaggio per descrivere il rendering di documenti strutturati (come HTML e XML) su schermo, carta discorsi, etc...
