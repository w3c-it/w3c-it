Il POWDER Working Group ha pubblicato la prima bozza su Protocol for Web Description Resources (POWDER): Description Resources e Protocol for Web Description Resources (POWDER): Web Description Resources (WDR) Vocabulary. POWDER è un sistema per allegare piccole annotazioni personalizzate al contenuto Web. Le risorse Web possono quindi essere recuperate, personalizzate e distribuite in vari contesti di distribuzione per soddisfare utilizzi personali e commerciali per l?adattamento del contenuto. Maggiori informazioni sulla Semantic Web home page. (Archivio News inglese, italiano)
