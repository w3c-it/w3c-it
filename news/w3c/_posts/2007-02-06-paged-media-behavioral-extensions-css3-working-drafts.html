--- title: "Paged Media, Behavioral Extensions: CSS3 Working Drafts" date: 2007-02-06 categories: ---

Il CSS Working Group ha rilasciato due Working Draft aggiornati per Cascading Style Sheets Level 3 (CSS3). Generated Content for Paged Media descrive le caratteristiche come le referenze incrociate, le note a pié di pagina, e le intestazioni, spesso usate in pubblicazioni a stampa. Behavioral Extensions to CSS permette collegamenti come XBL per effettuare link da CSS style sheet, e permette che i collegamenti effettuati possano essere utilizzati con utenti, differenti style sheet e media selection. Maggiori informazioni sulla CSS home page. (Archivio News inglese, italiano)
