--- title: "Last Call: CSS Color Module Livello 3" date: 2008-07-21 layout: page categories: w3c --- Il Cascading Style Sheets (CSS) Working Group ha publicato il Last Call Working Draft di CSS Color Module Level 3. CSS (Cascading Style Sheets) è un linguaggio per descrivere il rendering di documenti HTML e XML su schermo, su carta, nella lingua parlata, etc. Utilizza proprietà correlate ai colori e valori corrispondenti per colorare il testo, gli sfondi, i bordi e altre perti di elementi in un documento. Queste specifiche descrivono i valori dei colori e le proprietà per i colori in primo piano e per il gruppo di opacità. Queste includono proprietà e valori derivate da CSS level 2 assieme a nuovi valori. Commenti sono bene accetti fino al primo settembre. Ulteriori informazioni su Style Activity.
