--- title: "Web Services Addressing WSDL Binding &egrave; W3C Candidate Recommendation" date: 2006-05-30 categories: --- W3C è lieto di annunciare l'avanzamento di Web Services Addressing 1.0 - WSDL Binding allo stato di Candidate Recommendation. I commenti sono benvenuti fino al 7 luglio. Il documento definisce come le proprietà in Web Services Addressing 1.0 - Core siano descritte con il Web Services Description Language (WSDL). Web Services Addressing fornisce i meccanismi transport-neutral ed è progettato per lavorare con entrambe le versioni di WSDL, 1.1 e 2.0. Maggiori informazioni sulla pagina dei Web service al W3C.
