--- title: "CSS Mobile Profile: Working Draft" date: 2006-12-08 categories: --- Il CSS Working Group ha rilasciato un Working Draft di CSS Mobile Profile 2.0. Questo subset di Cascading Style Sheets (CSS) 2.1 è il fondamento per l'implementazione di CSS su dispositivi vincolati come telefoni mobili, scritto per assicurare l'interoperabilità e per l'allineamento con la specifica OMA's Wireless CSS 1.1.
