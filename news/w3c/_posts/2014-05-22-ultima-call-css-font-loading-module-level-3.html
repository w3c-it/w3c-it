--- title: "Ultima Call: CSS Font Loading Module Level 3" date: 2014-05-22 categories: ---

Il Cascading Style Sheets (CSS) Working Group ha pubblicato un'ultima call per una Bozza di Lavoro su CSS Font Loading Module Level 3. Questo modulo CSS descrive eventi e interfacce usate per caricare dinamicamente risorse di font. Il CSS è un linguaggio per descrivere la rappresentazione di documenti strutturati (come HTML e XML) nello schermo, su carta, nel parlato, etc. I commenti sono benvenuti fino al 30 Giugno 2014. Per saperne di più vai a Style Activity.
