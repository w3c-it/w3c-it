--- title: "Rule Interchange Format Core Design: Working Draft" date: 2007-03-30 layout: page categories: --- The Rule Interchange Format (RIF) Working Group has published the First Public Working Draft of RIF Core Design. The document specifies the core design for a format that allows rules to be translated between rule languages and thus transferred between rule systems. The group invites comments through 27 April.
