--- title: "Pubblicata Draft su Web Alarms API Specification" date: 2013-02-05 categories: ---

Il System Applications Working Group ha pubblicato la prima Working Draft relativa a Web Alarms API Specification. Queste specifiche definiscono le System Level API che consentono l'accesso alle impostazioni di alarm di un dispositivo, in modo da schedulare una notifica o lanciare un'applicazione in istante temporale specifico. Ad esempio, in alcune applicazioni quali sveglie, calendario, aggiornarmenti automatici, è possibile usare le Alarm API per scatenare particolari comportamenti del dispositvo in istanti temporali specifici.
