--- title: "W3C Invita all'Implementazione di Linked Data Platform 1.0" date: 2014-06-19 layout: page categories: w3c ---

Il Linked Data Platform (LDP) Working Group invita all'implementazione della Raccomandazione Candidata su Linked Data Platform 1.0. Questo documento decrive un set di best practices e un approccio semplice per una lettura-scrittura dell'architettura Linked Data, basati sull'accesso HTTP a risorse web che descrivono il loro stato usando il modello di dati RDF. Per saperne di più circa la Data Activity.
