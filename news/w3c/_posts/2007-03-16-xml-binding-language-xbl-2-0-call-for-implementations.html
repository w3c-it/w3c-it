--- title: "XML Binding Language (XBL) 2.0: Call for Implementations" date: 2007-03-16 categories: --- W3C è lieto di annunciare l'avanzamento di XML Binding Language (XBL) 2.0 allo stato di Candidate Recommendation. XBL estende l'apparenza e il comportamento di elementi nei formati Web come HTML. Gli elementi possono essere mappati per script, event handlers, CSS, e modelli di cntenuti più complessi. I contenuti possono essere riordinato ed impacchettati in modo tale che complessi stili CSS possano essere applicati ad un semplice markup HTML o XHTML. XBL può essere utilzzata sia per implementare nuove interfacce DOM, sia con altre specifiche, per implementare tag set arbitrari come congegni.
