--- title: "Aggiornato il Web Services Internationalization Usage Scenarios" date: 2003-05-16 layout: page categories: ---

La Web Services Task Force dell' Internationalization Working Group ha rilasciato un Working Draft aggiornato dal titolo Web Services Internationalization Usage Scenarios. Il documento prende in esame come il linguaggio, la cultura ed i problemi relativi interagiscano con l' architettura e la tecnologia dei Web service. La task force invita a inviare commenti. Per maggiori informazioni visitate la Internationalization home page.
