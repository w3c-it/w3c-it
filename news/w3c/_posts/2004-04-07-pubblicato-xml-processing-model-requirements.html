--- title: "Pubblicato: XML Processing Model Requirements" date: 2004-04-07 layout: page categories: w3c ---

XML Core Working Group ha pubblicato XML Processing Model Requirements come Working Group Note. Il modello e il linguaggio di elaborazione di XML che vengono delineati rappresentano un modo interoperabile per le applicazioni per descrivere l' ordine in cui i processi dovrebbero essere applicati ai documenti XML. Per ulteriori informazioni: XML home page.
