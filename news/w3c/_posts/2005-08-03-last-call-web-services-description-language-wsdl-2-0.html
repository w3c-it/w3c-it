--- title: "Last Call: Web Services Description Language (WSDL) 2.0" date: 2005-08-03 layout: page categories: --- Il Web Services Description Working Group ha pubblicato il Last Call Working Drafts di Web Services Description Language (WSDL) Versione 2.0 che contiene: Primer, Part 1: Core Language, Part 2: Adjuncts e SOAP 1.1 Binding. Linguaggio XML, WSDL descrive servizi di rete ed è utilizzato per documentare sistemi distribuiti e comunicazioni automatizzate tra le applicazioni. I commenti sono benvenuti fino al 19 settembre. Maggiori informazioni sulla home page dei Web service. (Archivio News inglese, italiano)
