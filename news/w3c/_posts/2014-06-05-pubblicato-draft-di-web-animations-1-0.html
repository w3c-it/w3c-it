--- title: "Pubblicato Draft di Web Animations 1.0" date: 2014-06-05 layout: page categories: w3c ---

Il Cascading Style Sheets (CSS) Working Group e l'SVG Working Group hanno pubblicato un Working Draft di Web Animations 1.0. Questa specifica definisce un modello per la sincromizzazione e la temporizzazione di cambiamenti nella presentazione di una pagina Web. Questa specifica definisce anche una API per interagire con questo modello, e ci si aspetta che ulteriori specifiche definiranno metodi dichiarativi per esporre queste caratteristiche. Per ulteriori informazioni consulta Style Activity e Graphics Activity.
