--- title: "Pubblicata nota sui Web Cryptography API Use Cases " date: 2013-09-10 layout: page categories: ---

Il Web Cryptography Working Group ha pubblicato una nota di gruppo sui Web Cryptography API Use Cases. Questi casi d'uso, descritti come scenari, rappresentano alcuni set di funzioanlità che dovrebbe essere gestiti dalla Web Cryptography API che fornisce una API per le operazioni crittografiche, come encryption e decryption, e dalla Key Discovery API che riguarda, nello specifico, l'abilità di accesso alle chiavi crittografiche. Dato che entrambe le API sono in fase di implememtazione, il lettore deve consultare ogni specifica per eventuali cambiamenti e deve considerare gli esempi di codici forniti al solo scopo illustrativo. Gli scenari presentati sono quelli primari e rappresentano ciò che il working group vorrebbe fosse realizzato in maniera prioritaria.

Questo documento NON è un recommendation track document e deve essere letto come una presentazione generale degli use case per le API crittografiche per il Web. Maggiori informazioni su Security Activity.
