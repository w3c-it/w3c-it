--- title: "Il W3C incoraggia l'implementazione di JSON-LD 1.0" date: 2013-09-10 layout: page categories: w3c ---

L'RDF Working Group ed il JSON-LD Community Group hanno pubblicato la Candidate Recommendation su JSON-LD 1.0, e JSON-LD 1.0 Processing Algorithms and API. Ciò segna l'inizio della call for implementations per JSON-LD 1.0.

JSON-LD armonizza la rappresentazione dei Linked Data in JSON descrivendo un formato comune di rappresentazione JSON per esprimere grafi orientati, combinando Linked Data and non-Linked Data in un unico documento. La sintassi è progettata per non disturbare i sistemi già in uso che sfruttano JSON, ma fornisce un percorso di miglioramento dolce da JSON to JSON-LD. Principalmente è inteso come un modo per usare Linked Data in ambienti di programmazione basati sul Web, per costruire servizi Web Linked Data interoperabili, e per salvare Linked Data in motori basati su JSON.

La specifica JSON-LD 1.0descrive il linguaggio JSON-LD in un modo che sia utile per gli autori. Fornisce anche una grammatica di base del linguaggio per gli sviluppatori. La specifica "JSON-LD 1.0 Algorithms and API" descrive algoritmi utili per lavorare con i dati JSON-LD. Specifica anche una Application Programming Interface che può essere usata per trasformare documenti JSON-LD per rendere più facile lavorare in ambienti di programmazione come JavaScript, Python, and Ruby
