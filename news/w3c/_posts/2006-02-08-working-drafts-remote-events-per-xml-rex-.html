--- title: "Working Drafts: Remote Events per XML (REX)" date: 2006-02-08 layout: page categories: --- Sforzo congiunto di SVG e Web API Working Groups, il REX Task Force ha rilasciato il primo Public Working Draft di Remote Events for XML (REX) 1.0 e dei suoi Requisiti. Utilizzando la grammatica REX, gli endpoint possono interagire con gli eventi DOM in remoto come se fossero nello stesso sito. Maggiori informazioni su: Scalable Vector Graphics (SVG) e Web APIs.
