--- title: "Pubblicato il Draft Web MIDI API" date: 2013-11-26 layout: page categories: w3c ---

Il Audio Working Group ha pubblicato uno Working Draft su Web MIDI API.Questa specifica definisce un API che supporta il protocollo MIDI, consentendo alle applicazioni web di enumerare e selezionare l'ingresso MIDI e i dispositivi di output sul sistema client e di inviare e ricevere messaggi MIDI. E' fatto per abilitare le applicazioni MIDI non musicali, oltre a quelle musicali, fornendo l'accesso a basso livello ai dispositivi MIDI disponibili sui sistemi degli utenti. L'API Web MIDI non è destinata a descrivere musica o controllare imput semanticamente; è progettata per esporre il meccanismo delle interfacce MIDI di input e output, e gli aspetti pratici di inviare e ricevere messaggi MIDI, senza identificare che cosa queste azioni significano semanticamente. L'API Web MIDI dovrebbe inoltre essere utilizzata in combinazione con altre API ed elementi della piattaforma Web, in particolare il Web Audio API. Questa API è inoltre destinata ad essere familiare agli utenti di MIDI API su altri sistemi, come CoreMIDI di Apple e Windows MIDI API di Microsoft. Per saperne di più su Rich Web Client Activity.
