--- title: "Website Accessibility Conformance Evaluation Methodology (WCAG-EM): Draft di Revisione Finale" date: 2014-01-30 layout: page categories: ---

Un Working Draft completo su Website Accessibility Conformance Evaluation Methodology (WCAG-EM) 1.0 è stato pubblicato oggi dal Web Content Accessibility Guidelines Working Group (WCAG WG) e dal Evaluation and Repair Tools Working Group (ERT WG), attraverso l'unione WCAG 2.0 Evaluation Methodology Task Force (Eval TF). WCAG-EM descrive un approccio per valutare come i siti web, incluse le applicazioni Web e i siti web per dispositivi mobili, siano conformi a WCAG 2.0. I commenti sono benvenuti fino al 28 Febbraio 2014. Dopo i commenti arrivati in questo periodo di revisione, WAI prevede di pubblicare questo come un'informativa (cioè, non-normativa) W3C Working Group Note. Per saperne di più circa la WCAG-EM Overview e circa la Web Accessibility Initiative (WAI).
