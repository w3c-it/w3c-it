--- title: "Web Services Choreography Description Language 1.0 &egrave; Candidate Recommendation" date: 2005-11-09 layout: page categories: w3c --- W3C è lieto di annunciare l'avanzamento di Web Services Choreography Description Language Version 1.0 (WS-CDL) allo stato di Candidate Recommendation. Questo linguaggio XML-based descrive le collaborazioni peer-to-peer tra i partecipanti di Web service per definire il loro comportamento da un punto di vista globale. Gli scambi ordinati del messaggio compiono così un comune obiettivo-business. I commenti sono benvenuti fino al 31 marzo Maggiori informazioni sulla Web services home page.
