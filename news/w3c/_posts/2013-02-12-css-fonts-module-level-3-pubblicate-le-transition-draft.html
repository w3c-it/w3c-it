--- title: "CSS Fonts Module Level 3, pubblicate le Transition Draft" date: 2013-02-12 layout: page categories: w3c ---

12 febbraio 2013

Il Gruppo di Lavoro Cascading Style Sheets (CSS) ha publicato oggi due working draft:

  • CSS Fonts Module Level 3. Questo modulo CSS3 descrive come sono specificate le proprietà dei font e come le risorse correlate ai font sono caricate dinamicamente.
  • CSS Transitions. Le CSS Transition permettono ai valori delle proprietà del CSS di cambiare gradualmente in una specifica durata temporale.

Per saperne di più vai a Style Activity.
