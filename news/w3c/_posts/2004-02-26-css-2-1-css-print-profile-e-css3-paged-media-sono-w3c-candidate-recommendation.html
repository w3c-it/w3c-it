--- title: "CSS 2.1, CSS Print Profile e CSS3 Paged Media Sono W3C Candidate Recommendation" date: 2004-02-26 categories: ---

W3C è lieto di annunciare tre Candidate Recommendation per il linguaggio Cascading Style Sheets (CSS). CSS 2.1 aggiunge nuove caratteristiche altamente richieste, corregge gli errori e porta CSS2 in linea con le implementazioni. CSS Print Profile funziona insieme a XHTML-Print per la stampa su dispositivi di basso costo. CSS3 Paged Media Module aggiunge paginazione, margini, intestazioni e piè di pagina, riferimenti incrociati con numeri di pagina. Si attendono commenti entro il 25 agosto 2004. Per ulteriori informazioni: CSS home page.
