--- title: "Ultima chiamata: CSS Shapes Module Level 1" date: 2013-12-03 layout: page categories: ---

Il CSS Working Group ha pubblicato un Last Call Working Draft per CSS Shapes Module Level 1. CSS Shapes descrive le forme geometriche che possono essere usate in CSS. Nel livello 1 le forme CSS possono essere applicate ai float. Applicare una forma circolare ad un float farà sì che il contenuto inline si collochi intorno alla forma circolare invece che alla bounding box del float. Si accettano commenti fino al 7 gennaio 2014. Ulteriori informazioni sulla Style Activity.
