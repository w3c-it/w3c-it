--- title: "Ultima chiamata: CSS Counter Styles Level 3 " date: 2013-07-18 layout: page categories: ---

Il Cascading Style Sheets (CSS) Working Group ha pubblicato una Last Call Working Draft del CSS Counter Styles Level 3.Questo modulo introduce la regola ‘@counter-style’, che permette agli autori di definire i loro propri counter styles da usare con i marcatori di lista CSS e i contatori di contenuto generato [CSS3LIST]. Inoltre predefinisce un set di stili di contatori comuni, inclusi quelli presenti nel CSS2 e CSS2.1. Commenti sono benvenuti fino al 18 agosto. Per saperne di più circa la Style Activity.
