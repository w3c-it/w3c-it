--- title: "Last Call: W3C XML Schema Definition Language (XSD) 1.1" date: 2009-02-02 layout: page categories: --- L'XML Schema Working Group ha pubblicato la Last Call Working Drafts di W3C XML Schema Definition Language (XSD) 1.1 Part 1: Structures e Part 2: Datatypes. La prima specifica XML Schema Definition Language, che offre agevolazioni per descrivere la struttura e i vincoli dei contenuti dei documenti XML, inclusi quelli con XML Namespace. Lo schema del linguaggio, che è rappresentato mediante un vocabolario XML e usa namespace, sostanzialmente ricostruisce ed estende considerevolmente le capacità che si trovano nelle document type definitions (DTDs) di XML. La seconda definisce agevolazioni per la definizione di datatype da usare con XML Schema ma anche altre specifiche XML. Il linguaggio di datatype, che è rappresentato in XML, fornisce un sovrainsieme di capacità che si possono trovare nelle document type definitions (DTDs) di XML per specificare datatype su elementi e attributi. I commenti sono bene accetti fino al 20 febbraio. Ulteriori informazioni su Extensible Markup Language (XML) Activity. (Archivio News Inglese, Italiano)
