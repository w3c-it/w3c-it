--- title: "Last Call Working Draft per Internationalization Tag Set (ITS) Versione 2.0" date: 2012-12-06 categories: ---

Il MultilingualWeb-LT Working Group ha pubblicato un Last Call Working Draft per Internationalization Tag Set (ITS) Version 2.0. Questo documento fornisce le data category e la loro implementazione come insieme di elementi e attributi chiamato Internationalization Tag Set (ITS) 2.0. ITS 2.0 è il successore di ITS 1.0; è progettato per promuovere la creazione di contenuto Web multilingue, concentrandosi sui formati basati su HTML e XML, e per sfruttare i workflow di localizzazione basati sull'XML Localization Interchange File Format (XLIFF). Si accettano commenti fino al 10 di Gennaio. Ulteriori informazioni sulla Internationalization Activity.
