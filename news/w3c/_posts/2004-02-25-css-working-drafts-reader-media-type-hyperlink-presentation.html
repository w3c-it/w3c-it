--- title: "CSS Working Drafts: 'Reader' Media Type, Hyperlink Presentation" date: 2004-02-25 layout: page categories: w3c ---

Il CSS Working Group ha pubblicato due First Public Working Draft come parti del linguaggio Cascading Style Sheets (CSS). CSS 'Reader' Media Type istruisce i dispositivi a restituire un documento in forma visuale e parlata, oppure in forma visuale e braille. CSS3 Hyperlink Presentation Module descrive la presentazione di link e la loro attivazione. Si attendono commenti su entrambi i documenti. Per ulteriori informazioni: CSS home page.
