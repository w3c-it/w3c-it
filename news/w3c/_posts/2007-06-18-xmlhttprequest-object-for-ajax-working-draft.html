--- title: "XMLHttpRequest Object for Ajax: Working Draft" date: 2007-06-18 layout: page categories: w3c ---

Documentando le modifiche sino al Last Call, il Web API Working Group ha rilasciato un Working Draft aggiornato di The XMLHttpRequest Object. Componente base di Ajax, l'oggetto XMLHttpRequest è un'interfaccia che permette agli script di effettuare funzioni HTTP client, quali la presentazione di form data oppure il caricamento di dati da un sito Web remoto. Maggiori informazioni sulla pagina della Rich Web Clients Activity. (Archivio News inglese, italiano)
