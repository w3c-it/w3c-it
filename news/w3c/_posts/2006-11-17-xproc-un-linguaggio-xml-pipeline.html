--- title: "XProc: un linguaggio XML Pipeline" date: 2006-11-17 categories: --- L'XML Processing Model Working Group ha rilasciato un Working Draft aggiornato di XProc: An XML Pipeline Language. Utilizzato per controllare ed organizzare il flusso di documenti, il linguaggio XProc standardizza interazioni ed input ed output per le trasformazioni, per il grande gruppo di specifiche come XSLT, XML Schema, XInclude e Canonical XML che operano con o producono documenti XML. Maggiori informazioni sulla XML home page. (Archivio News inglese, italiano)
