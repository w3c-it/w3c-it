--- title: "W3C invita all'implementazione di XProc: un XML Pipeline Language" date: 2008-11-26 categories: --- L'XML Processing Model Working Group invita all'implementazione della Candidate Recommendation di XProc: An XML Pipeline Language. Questa specifica descrive la sintassi e la semantica di XProc, un linguaggio per descrivere le operazioni da svolgere sui documenti XML. Una pipeline è fatta di passi. Come le pipeline, i passi richiedono zero o più documenti XML come input e producono zero o più documenti XML com output. L'input di un passo viene dal Web, dal documento di pipeline, dallo stesso input della pipeline o dall'output di altri passi della pipeline. Ulteriori informazioni su Extensible Markup Language (XML) Activity. (Archivio News Inglese, Italiano)
