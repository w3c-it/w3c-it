--- title: "Il W3C incoraggia l'implementazione di HTML Media Capture" date: 2013-05-09 layout: page categories: w3c ---

Il Device APIs Working Group incoraggia l'implementazione della Candidate Recommendation per HTML Media Capture. La specifica di HTML Media Capture definisce un'estensione del form HTML che facilita l'accesso degli utenti ai meccanismi di acquisizione dati da dispositivi come macchine fotografiche o microfoni, dall'interno del controllo di file upload. Ulteriori informazioni sulla Ubiquitous Web Applications Activity.
