--- title: "Ultima chiamata: File API" date: 2013-09-12 layout: page categories: w3c ---

IL Web Applications Working Group ha pubblicato un Last Call Working Draft per File API. Questa specifica fornisce una API per rappresentare oggetti file nelle applicazioni Web, selezionarli programmaticamente e accedere ai loro dati. Si accettano commenti fino al 24 ottobre. Ulteriori informazioni sulla Rich Web Client Activity.
