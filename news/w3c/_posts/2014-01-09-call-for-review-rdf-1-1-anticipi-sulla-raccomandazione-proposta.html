--- title: "Call for Review: RDF 1.1 anticipi sulla Raccomandazione Proposta" date: 2014-01-09 layout: page categories: w3c ---

L' RDF Working Group oggi ha pubblicato un set di otto specifiche Resource Description Framework (RDF) 1.1 (6 Proposed Recommendations e 2 Proposed Edited Recommendations). RDF 1.1 Concetti e Abstract Syntax definisce un abstract syntax (un data model) che serve per collegare tutti i linguaggi e le specifiche RDF-based.L'abstract yntax ha due strutture di dati: grafi RDF sono set di soggetto-predicato-oggetto tripli, in cui l'elemento può essere IRIs, nodi bianchi o datatyped letterali. Essi sono usati per esprimere descrizioni di risorse. I dataset RDF sono usati per organizzare collezioni di grafi RDF e comprendono un grafo definito o più grafi denominati.

  • RDF 1.1 Semantics descrive una precisa semantica per il Resource Description Framework 1.1 e l' RDF Schema, e definisce una serie di regimi implicanti e corrispondenti modelli di implicazione.
  • RDF 1.1 Turtle definisce una sintassi testuale per l'RDF chiamato Turtle che permette ad un grafo RDF di essere scritto completamente in una forma di testo compatta e naturale, con abbreviazioni per modelli e datatype di uso comune. Turtle fornisce livelli di compatibilità con il formato N-Triples  così come per la sintassi a modello triplo della Raccomandazione SPARQL W3C.
  • RDF 1.1 TriG – RDF Dataset Language definisce una sintassi testuale per l'RDF chiamato TriG che consente ad un dataset RDF di essere scritto completamente in una forma di testo compatta e naturale, con abbreviazioni per modelli e datatype di uso comune. TriG è un'estensione del formato Turtle.
  • RDF 1.1 N-Triples è un formato di testo normale, line-based, per la codifica di un grafo RDF.
  • RDF 1.1 N-Quads è un formato di testo normale, line-based, per la codifica di un dataset RDF.
  • RDF Schema 1.1 (Proposed Edited Recommendations) fornisce un vocabolario di data-modelling per dati RDF. RDF Schema è un'estensione del vocabolario basico RDF.
  • RDF 1.1 XML Syntax (Proposed Edited Recommendations) definisce una sintassi XML per RDF chiamata RDF/XML in termini di Namespaces in XML, l' XML Information Set e l' XML Base.

Commenti sono benvenuti fino al 9 Febbraio 2014. Per saperne di più circa la Data Activity.
