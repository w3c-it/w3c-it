--- title: "Last Call: Mobile Web Best Practice" date: 2006-04-12 layout: page categories: --- Il Mobile Web Best Practices Working Group ha rilasciato un secondo Last Call Working Draft di Mobile Web Best Practices 1.0. Il draft descrive come produrre Web content e Web site predisposti per la distribuzione su dispositivi mobili e small-screen. Scritto per tutti i partecipanti della catena produttiva del mobile, il documento è progettato per migliorare la user experience. I commenti sono benvenuti fino al 3 maggio. Maggiori informazioni sulla pagina della W3C Mobile Web Initiative, sforzo congiunto di fornitori di authoring tool, content provider, produttori di microtelefoni, fornitori di browser ed operatori del mobile.
