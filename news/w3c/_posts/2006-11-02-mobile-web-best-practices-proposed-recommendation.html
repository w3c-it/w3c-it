--- title: "Mobile Web Best Practices è Proposed Recommendation" date: 2006-11-02 categories: ---

W3C è lieto di annunciare l'avanzamento di Mobile Web Best Practices 1.0 allo stato di Proposed Recommendation. I commenti sono benvenuti fino all'11 dicembre. Scritte per programmatori di siti Web e sistemi di content management, queste linee-guida descrivono come produrre contenuti Web che lavorino bene su dispositivi mobili. Trenta organizzazioni che partecipano alla Mobile Web Initiative hanno ottenuto consenso ed incoraggiano l'adozione e l'implementazione di queste linee-guida, per migliorare l'esperienza utente e raggiungere l'obiettivo di "un solo Web".
