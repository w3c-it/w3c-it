--- title: "Language Bindings for DOM Specifications: Working Draft" date: 2007-10-17 categories: ---

Il Web API Working Group ha rilasciato il First Public Working Draft di Language Bindings for DOM Specifications. La bozza definisce le specifiche del linguaggio IDL per l'utilizzo secondo lo standard del W3C per l'interfaccia DOM che ne definisce le conformità ed i requisiti per il collegamento con gli script ECMAScript e Java binding. Questa guida è rivolta agli implementatori esperti delle specifiche DOM ma è adatta anche ai meno esperti ed è stata scritta per garantire che tutte le implementazioni conformi delle interfacce DOM siano interoperabili fra loro. Maggiori informazioni sulla Rich Web Clients home page. (Archivio News inglese, italiano)
