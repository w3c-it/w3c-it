--- title: "Web Services Description Language (WSDL) 2.0 Is a Proposed Recommendation" date: 2007-05-23 layout: page categories: --- W3C è lieto di annunciare l'avanzamento di Web Services Description Language (WSDL) Versione 2.0 allo stato di Proposed Recommendation in tre parti: Part 0: Primer, Part 1: Core Language e Part 2: Adjuncts. I commenti sono benvenuti fino al 20 giugno. WSDL modella e descrive servizi modulari Web ed è usato per documentare sistemi distribuiti ed automatizzare le comunicazioni tra le applicazioni. WSDL Additional MEPs, RDF Mapping e SOAP 1.1 Binding sono Working Draft aggiornati. Maggiori informazioni sulla pagina dei Web service. (Archivio News inglese, italiano)
