--- title: "CSV on the Web: pubblicata draft su Use Cases e Requirements " date: 2014-07-01 layout: page categories: w3c ---

Il CSV on the Web Working Group ha pubblicato una Working Draft su CSV on the Web: Use Cases and Requirements. Una grossa percentuale di dati pubblicati sul Web è in forma tabellare, tipicamente pubblicata in formato Comma Separated Values (CSV). Il CSV on the Web Working Group si occupa di individuare e specificare tecnologie che forniscano una migliore interoperabilità per le applicazioni Web data dependent che lavorano su dataset tabellari composti da file singoli o multiipli in formato CSV o simili. Questo documento elenca una serie di use case giudicati rappresentativi di come i dati tabellari sono comunemente usati nelle applicazioni data dependent. Il documento fornisce inoltre un set di requisiti derivati dagli use case utilizzati in fase di design delle specifiche. Maggiori informazioni su Data Activity.
