--- title: "Web Intents Pubblicata Nota" date: 2013-05-23 categories: ---

Il Gruppo di Lavoro Device APIs e il Gruppo di Lavoro Web Applications (WebApps) hanno pubblicato una nota di Lavoro sui Web Intents. Questa nota informativa definisce un "service discovery" e un meccanismo RPC  a basso peso pensato per applicazioni web chiamato Web Intents. Questo documento definisce le  interfacce DOM e linguaggio di markup usato dal client e dalle service pages per creare,ricevere e rispondere a messaggi di tipo Web Intents , e definisce anche le procedure che i vari User Agents portano avanti volta per volta per facilitare il processo. Per saperne di più vai a Ubiquitous Web Applications Activity
