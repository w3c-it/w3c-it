Pubblicato il Draft Clipboard API and events

11 Aprile 2013

Il Web Applications Working Group ha pubblicato un Working Draft su Clipboard API and events. Questo documento descrive le API per le clipboard operations nelle applicazioni Web, come: copia, taglia e incolla . Per saperne di più vai al link Rich Web Client Activity.
