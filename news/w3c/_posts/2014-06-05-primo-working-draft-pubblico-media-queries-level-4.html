--- title: "Primo Working Draft pubblico: Media Queries Level 4" date: 2014-06-05 layout: page categories: ---

Il Cascading Style Sheets (CSS) Working Group ha pubblicato un Working Draft di Media Queries Level 4. Le Media Query permettono agli autori di testare o richiedere valori o caratteristiche degli user agent o dei dispositivi, in maniera indipendente dal documento che deve essere renderizzato. Sono usate nella regola @media del CSS per applicare degli stili ai documenti in base a delle condizioni, e in altri contesti e linguaggi, come l'HTML e il Javascript. Media Queries Level 4 descrive il meccanismo e la sintassi di media query, media type e media feature. La specifica estende e sostituisce le caratteristiche definite in Media Queries Level 3. CSS è un linguaggio per descrivere la renderizzazione di documenti strutturati (come HTML e XML) su schermo, su carta, come speech, ecc. Per ulteriori informazioni consulta Style Activity.
