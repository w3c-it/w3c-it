--- title: "Pubblicata Bozza su Web Telephony API" date: 2013-06-20 layout: page categories: ---

Il Gruppo di Lavoro su System Applications  ha pubblicato una Bozza di Lavoro su Web Telephony API. Questa specifica definisce un API per la gestione delle telefonate. Uno uses case tipico del Web Telephony API è l'implementazione di un'applicazione 'Dialer' che supporti chiamate multiple e molteplici servizi di telefonia. Per saperne di più vai a Ubiquitous Web Applications Activity.
