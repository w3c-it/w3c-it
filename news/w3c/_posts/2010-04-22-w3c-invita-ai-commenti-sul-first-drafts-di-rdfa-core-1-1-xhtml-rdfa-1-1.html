--- title: "W3C Invita ai Commenti sul First Drafts di RDFa Core 1.1, XHTML+RDFa 1.1" date: 2010-04-22 layout: page categories: w3c ---

L' RDFa Working Group ha pubblicato i First Public Working Drafts di RDFa Core 1.1 e XHTML+RDFa 1.1. RDFa Core è una specifica per attributi affinché esprimano dati strutturati in qualsiasi linguaggio di markup. XHTML+RDFa 1.1 è un'applicazione di RDFa Core 1.1 per XHTML. Entrambi questi documenti sostituiranno la specifica RDFa in XHTML (RDFa 1.0). Insieme, queste specifiche garantiscono che il markup di persone, luoghi, eventi, prodotti, ricette, social network,  e molti altri concetti che sono pubblicati frequentemente sul web sia human-readable e machine-readable. Tali documenti migliorano RDFa 1.0 poiché aggiungono una serie di richieste della comunità del Web per facilitare la creazione. Per saperne di più si consulti Semantic Web.
