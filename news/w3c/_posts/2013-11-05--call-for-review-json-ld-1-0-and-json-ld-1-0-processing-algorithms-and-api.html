--- title: " Call for Review: JSON-LD 1.0, and JSON-LD 1.0 Processing Algorithms and API" date: 2013-11-05 layout: page categories: w3c ---

L'RDF Working Group ha pubblicato oggi due Proposed Recommendations:

  • JSON-LD 1.0. JSON è un utile formato di messaging e data serialization. Queste specifiche definsicono JSON-LD, un formato JSON-based utile per i Linked Data. La sintassi è stata progettata per consentire una semplice integrazione con i sistemi che già usano JSON e fornisce un path di aggiornamento da JSON a JSON-LD. Lo scopo principale è la programmazione di Web Services interoperabili che usino i Linked Data in un ambiente Web-based. Si accettano commenti fino al 5 Dicembre.
  • JSON-LD 1.0 Processing Algorithms and API. Questa specifica definisce un set di algoritmi per programmare le trasformazioni di documenti JSON-LD. La ristrutturazione dei dati secondo trasformazioni predefinite ne semplifical'utilizzo. Inoltre, questo documento propone una Application Programming Interface (API) per consentire agli sviluppatori di implementare gli algoritmi. Si accettano commenti fino al 5 Dicembre.
