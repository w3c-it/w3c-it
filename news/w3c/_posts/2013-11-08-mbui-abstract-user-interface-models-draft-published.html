--- title: "MBUI Abstract User Interface Models Draft Published" date: 2013-11-08 categories: ---

Il Model-Based User Interfaces Working Group ha pubblicato un Working Draft su MBUI – Abstract User Interface Models. Model-Based User Interface Design facilita l'interscambio di design attraverso un approccio a livelli che separa i diversi livelli di astrazione nella progettazione delle interfacce utente. Questo documento copre la specifica di Abstract User Interface Models, definendo la sua semantica tramite un meta-modello e una sintassi di interscambio (espressa come Schema XML) per lo scambio di Abstract User Interface Model tra diversi ambienti di sviluppo di interfacce. Ulteriori informazioni sulla Ubiquitous Web Applications Activity.
