--- title: "Pubblicato il First Draft dei selettori API Livello 2" date: 2010-01-19 layout: page categories: ---

Il Web Applications Working Group ha pubblicato il First Public Working Draft di Selectors API Level 2. I selettori, che sono ampiamente usati nei CSS, sono modelli che corrispondono agli elementi in una struttura ad albero. La specifica dei selettori API definisce i metodi per il recupero dei nodi Element dal DOM confrontandoli con un gruppo di selettori, e verifica se un dato elemento corrisponde a un dato selettore. È spesso opportuno eseguire le operazioni DOM su uno specifico insieme di elementi in un documento. Questi metodi semplificano il processo di acquisizione e collaudo di elementi specifici, soprattutto in confronto con tecniche più prolisse definite e utilizzate in passato. Per saperne di più si consulti Rich Web Client Activity.
