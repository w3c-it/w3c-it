--- title: "Pubblicato Primo Working Draft Pubblico su CSS Font Loading Livello 3, CSS Display Livello 3" date: 2014-02-20 layout: page categories: ---

Il Cascading Style Sheets (CSS) Working Group oggi ha pubblicato due Primi Working Drafts Pubblici:

  • CSS Font Loading Module Level 3. Questo modulo CSS descrive eventi e interfacce usate per caricare dinamicamente le risorse di caratteri.
  • CSS Display Module Level 3. Questo modulo contiene le caratteristiche del CSS relative alla proprietà di visualizzazione e altri dettagli sulla generazione di box. Il CSS è un linguaggio per la descrizione del rendering di documenti strutturati (come HTML e XML) su schermo, carta, discorsi, etc.
