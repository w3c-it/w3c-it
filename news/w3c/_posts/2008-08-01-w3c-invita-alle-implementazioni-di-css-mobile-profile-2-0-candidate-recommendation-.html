--- title: "W3C Invita alle implementazioni di CSS Mobile Profile 2.0 (Candidate Recommendation)" date: 2008-08-01 layout: page categories: w3c --- The CSS Working Group has published a Candidate Recommendation for CSS Mobile Profile 2.0., which defines a common baseline of CSS support that even constrained mobile devices can provide. This effort is part of W3C's ongoing efforts to make the Web easier to use from a mobile devices (see related news). For the CSS Mobile Profile 2.0, W3C has worked closely together with OMA to remove the differences between W3C's and OMA's previous CSS-mobile profiles. An "alpha" quality test suite is available for the mobile profile. The Working Group will track implementations during the Candidate Recommendation phase. Implementers are invited to send feedback before February 2009. Learn more about the Style Activity. (Archivio News Inglese, Italiano)
