--- title: "Il W3C lancia la nuova Data Activity" date: 2013-12-11 layout: page categories: w3c ---

Oggi il W3C ha lanciato la nuova Data Activity per guidare il Web verso un nuovo livello di interoperabilità dei dati. L'attività comprende due nuovi gruppi:

  • il CSV on the Web Working Group, la cui missione è fornire tecnologie per mezzo delle quali le applicazioni Web basate dui dati possano fornire una maggiore interoperabilità quando lavorano con dataset che usano il formato CSV (Comma-Separated Values) o formati simili.
  • il Data on the Web Best Practices Working Group, la cui missione è 1) sviluppare l'ecosistema open data, facilitando la comunicazione tra sviluppatori ed editori; 2) fornire una guida agli editori, così che vi sia maggiore uniformità nel modo in cui sono gestiti i dati, promuovendo il loro riuso; 3) promuovere tra gli sviluppatori la fiducia nei dati, qualsiasi tecnologia decidano di usare, aumentando le possibilità di reale innovazione.

La Data Activity incorpora ed espande il lavoro fatto dalla Semantic Web e eGovernment Activity. Il W3C continuerà a completare e migliorare il Semantic Web alla luce delle sempre crescenti esperienze e richieste derivanti dal mondo reale.

Per ulteriori informazioni consultare Data Activity.
