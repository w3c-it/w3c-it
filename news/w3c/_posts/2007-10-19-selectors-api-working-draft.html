--- title: "Selectors API: Working Draft" date: 2007-10-19 layout: page categories: --- Il Web API Working Group ha rilasciato un Working Draft aggiornato di Selectors API. Ampiamente usati in CSS, i selettori sono pattern che accoppiano tra loro elementi contrastanti in una struttura ad albero. Questi metodi sono definiti per recuperare elementi nodali dal DOM per abbinare contro un gruppo di selettori e semplificare il processo di acquisizione di specifici elementi, specialmente comparati con le tecniche più verbose utilizzate in passato. Maggiori informazioni sulla Web API home page. (Archivio News inglese, italiano)
