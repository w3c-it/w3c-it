--- title: "Working Draft: XMLHttpRequest Object per AJAX" date: 2006-04-05 categories: ---

Il Web API Working Group ha rilasciato il Primo Public Working Draft di The XMLHttpRequest Object. Il draft documenta le caratteristiche dell'oggetto XMLHttpRequest, fondamento di AJAX. L'interfaccia consente script per lo svolgimento di funzioni HTTP client, quali la presentazione di form data o il caricamento di dati da un Web site remoto. Maggiori informazioni sulla pagina della Rich Web Clients Activity. (Archivio News inglese, italiano)
