--- title: "XML Security Working Group rilascia otto Working Drafts" date: 2009-02-26 layout: page categories: --- L'XML Security Working Group ha pubblicato un insieme di otto Working Draft. L'XML Signature 1.1 e XML Encryption 1.1 First Public Working Drafts cambia l'insieme di default degli algoritmi crittografici in entrambe le specifiche. XML Security Use Cases and Requirements e XML Signature Transform Simplification: Requirements and Design sono documenti che ci si aspetta guidino l'operato dei gruppi di lavoro per le future versioni delle specifiche di XML Security che dovrebbero rendere più radicali i cambiamenti rispetto a quelli della serie 1.1 di queste specifiche. Il Working Group vorrebbe ricevere feedback presto riguardo a queste quattro draft.

In aggiunta, le specifiche di XML Security Derived Keys introducono mark-up per la derivazione delle chiavi, perchè sia usato sia con XML Signature che con XML Encryption. XML Signature Properties definisce le proprietà usate più comunemente. XML Security Algorithms è un cross-reference per gli algoritmi e i loro identificatori usati con le specifiche di XML security, mettendo in un unico posto le informazioni sparse in diversi documenti. XML Signature Best Practices è una Working Draft rivisitata per Best Practices nel'uso delle specifiche di XML Signature. (Archivio News Inglese, Italiano)
