--- title: "Pubblicato il Draft CSS Generated Content for Paged Media Module" date: 2010-06-08 categories: ---

Il Cascading Style Sheets (CSS) Working Group ha pubblicato un Working Draft di CSS Generated Content for Paged Media Module. Questo modulo descrive le caratteristiche spesso usate in pubblicazioni a stampa. La maggior parte delle funzionalità specificate coinvolgono alcuni tipi di contenuto generato, in cui il contenuto estratto dal documento è abbellito, duplicato o spostato nella presentazione finale del documento. Insieme a due altri moduli CSS3 - layout multi-colonna e "paged media" - questo modulo offre funzionalità avanzate per la presentazione di documenti strutturati in "paged media". Per saperne di più si consulti Style Activity.
