--- title: "Last Call: Accessible Rich Internet Applications (WAI-ARIA); Best Practices e Implementation Guide Drafts pubblicate" date: 2009-02-24 categories: --- Il Protocols and Formats Working Group ha pubblicato la Last Call Working Draft di Accessible Rich Internet Applications (WAI-ARIA). WAI-ARIA definisce un modo di rendere i contenuti e le applicazioni Web più accessibili alle persone con disabilità. In particolare fornisce aiuto per i contenuti dinamici e le interfacce avanzate sviluppate con Ajax, HTML, JavaScript, ed altre tecnologie.

Il Working Group ha pubblicato anche una First Public Working Draft di WAI-ARIA User Agent Implementation Guide che fornisce linee guida su come i browser e altri user agent dovrebbero esporre le caratteristiche WAI-ARIA all'API della piattaforma per l'accessibilità. La Working Draft aggiornata di WAI-ARIA Best Practices, che è stata pubblicata oggi, descrive come gli sviluppatori di contenuti Web possano sviluppare applicazioni ricche ed accessibili usando WAI-ARIA. Questi documenti WAI-ARIA sono descritti in WAI-ARIA Overview. Ulteriori informazioni su details in the review announcement, e su Web Accessibility Initiative (WAI). (Archivio News Inglese, Italiano)
