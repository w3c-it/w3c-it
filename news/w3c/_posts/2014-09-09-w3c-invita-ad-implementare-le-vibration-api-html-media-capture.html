--- title: "W3C invita ad implementare le Vibration API; HTML Media Capture" date: 2014-09-09 categories: ---

Il Device APIs Working Group invita ad implementare due Candidate Recommendations:

  • Vibration API. Questa specifica definisce le API per accedere ai meccansismi di vibrazione degli hosting device. La vibrazione è una forma di feedback tattile.
  • HTML Media Capture. Questa specifica definisce l'estensione di HTML form per facilitare l'accesso ai meccanismi di media capture quali, ad esempio, fotocamera o microfono, attraverso un file di controllo.
