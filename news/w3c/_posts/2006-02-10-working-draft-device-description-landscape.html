--- title: "Working Draft: Device Description Landscape" date: 2006-02-10 layout: page categories: --- Il Device Description Working Group ha rilasciato il primo Public Working Draft di Device Description Landscape, integrativo di Device Description Ecosystem. Questo draft descrive lo stato attuale delle numerose opzioni che esistono per fornire una Device Description che abiliti le applicazioni device-aware. Maggiori informazioni sulla W3C Mobile Web Initiative, sforzo congiunto di fornitori di authoring tool, content provider, produttori di microtelefoni, browser vendor ed operatori del mobile.
