--- title: "POWDER Description Resources, Datatypes and Vocabulary: Working Drafts" date: 2007-09-28 categories: ---

Il POWDER Working Group ha pubblicato First Public Working Drafts di Protocol for Web Description Resources (POWDER): Description Resources e Protocol for Web Description Resources (POWDER): Web Description Resources (WDR) Vocabulary e Protocol for Web Description Resources (POWDER): Web Description Resources Datatypes (WDRD). POWDER è un metodo per collegare annotazioni piccole e facilmente producibili, ad una vasta collezione di Web content. Le risorse Web possono essere recuperate, personalizzate e consegnate in una grande varietà di contesti per venire incontro sia ai bisogni sociali di classificare i contenuti, sia alle esigenze commerciali per il loro adattamento.
