--- title: "Pubblicati i Draft di RDFa Core 1.1 e XHTML+RDFa 1.1 " date: 2010-08-03 categories: ---

Il RDFa Working Group ha appena pubblicato due Working Draft: RDFa Core 1.1 e XHTML+RDFa 1.1. RDFa Core 1.1 è una specifica per gli attributi per esprimere dati strutturati in qualsiasi linguaggio di markup. I dati incorporati già disponibili nel linguaggio di markup (ad esempio, XHTML) sono riutilizzati dal markup RDFa, cosicché gli editori non abbiano bisogno di replicare dati significativi del contenuto del documento. XHTML+RDFa 1.1 è un linguaggio di markup della famiglia XHTML. Questo estende il linguaggio di markup XHTML 1.1 con attributi definiti in RDFa Core 1.1. Questo documento è rivolto agli autori che vogliono creare documenti della famiglia XHTML che incorporano un ricco markup semantico. Per saperne di più si consulti Semantic Web Activity.
