--- title: "Working Draft: State Chart XML (SCXML)" date: 2006-01-24 layout: page categories: w3c --- Il Voice Browser Working Group ha rilasciato un Working Draft aggiornato di State Chart XML (SCXML): State Machine Notation for Control Abstraction 1.0. SCXML è un ambiente di esecuzione basato su UML Harel State Tables e CCXML . SCXML è candidato per il controllo del linguaggio con VoiceXML 3.0, CCXML 2.0, e l'authoring language sotto lo sviluppo del Multimodal Interaction Working Group. Maggiori informazioni sulla Voice Browser home page. (Archivio News inglese, italiano)
