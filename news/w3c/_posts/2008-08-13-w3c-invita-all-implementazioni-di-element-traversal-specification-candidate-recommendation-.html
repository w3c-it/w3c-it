--- title: "W3C Invita all implementazioni di Element Traversal Specification (Candidate Recommendation)" date: 2008-08-13 layout: page categories: --- The Web Applications Working Group invites implementation of the Candidate Recommendation of Element Traversal Specification. This specification defines the ElementTraversal interface, intended to provide a more convenient alternative to existing Document Object Model (DOM) navigation interfaces, with a low implementation footprint. It does so by allowing script navigation of the elements of a DOM tree, excluding all other nodes in the DOM, such as text nodes. It also provides an attribute to expose the number of child elements of an element. See the disposition of Last Call Comments and learn more about the Rich Web Client Activity.
