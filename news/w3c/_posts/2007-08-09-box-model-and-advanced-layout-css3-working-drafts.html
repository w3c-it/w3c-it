--- title: "Box Model and Advanced Layout: CSS3 Working Drafts" date: 2007-08-09 layout: page categories: --- Il CSS Working Group ha rilasciato due Working Draft aggiornati per il linguaggio Cascading Style Sheets (CSS) Level 3. CSS basic box model descrive il layout-base di documenti di testo in media visuali. Il CSS3 Advanced Layout Module definisce un ordine visuale indipendente dell'ordine dei documenti, dalla posizione e dall'allineamento dei congegni di interfaccia utente e griglie di pagine e finestre. Maggiori informazioni sulla CSS home page. (Archivio News inglese, italiano)
