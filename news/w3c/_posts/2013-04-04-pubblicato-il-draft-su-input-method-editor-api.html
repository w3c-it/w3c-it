--- title: "Pubblicato il draft su Input Method Editor API" date: 2013-04-04 layout: page categories: w3c ---

Il Web Applications Working Group ha pubblicato un Working Draft dell'Input Method Editor API. Questa specifica definisce una “IME API” che fornisce alle applicazioni Web un accesso ad un IME (input-method editor) associato ad uno user agent ospitante. Questa IME API include: un'interfaccia InputMethodContext, che fornisce dei metodi per ottenere dati dettagliati da una composizioni IME in corso; un Composition dictionary, che rappresenta degli attributi in sola lettura sulla composizione corrente, come ad esempio il testo attuale ed il suo stile. Questa API è progettata per essere utilizzata insieme agli eventi DOM. Ulteriori informazioni sulla Rich Web Client Activity.
