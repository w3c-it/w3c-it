--- title: "Pubblicato il last Call CSS3 Basic User Interface" date: 2003-07-03 categories: ---

Il CSS Working Group ha pubblicato un Last Call Working Draft di Basic User Interface, un modulo del linguaggio Cascading Style Sheets (CSS). Il ducumento tratta gli stati e le caratteristiche dell' interfaccia utente, frammenti di elemento, form, attributi stilistici in HTML, focus navigation, e elemeti di stile come icone per l' accessibilità. Si invita ad inviare commenti entro il 31 luglio. Per ulteriori informazioni: CSS home page.
