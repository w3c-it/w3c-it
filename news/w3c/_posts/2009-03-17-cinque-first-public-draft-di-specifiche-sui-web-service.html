--- title: "Cinque First Public Draft di specifiche sui Web Service" date: 2009-03-17 layout: page categories: --- Il Web Services Resource Access Working Group ha pubblicato cinque First Public Working Draft: Web Services Enumeration (WS-Enumeration), Web Services Eventing (WS-Eventing), Web Services Resource Transfer (WS-RT), Web Services Transfer (WS-Transfer) e Web Services Metadata Exchange (WS-MetadataExchange). La prima descrive un protocollo generale basato su SOAP per l'enumerazione di sequenze di elementi XML usati per log, code di messaggi, o altri modelli lineari di informazione. Il secondo descrive un protocollo che permette ai Web service di iscriversi o accettare sottoscrizioni per la notifica di eventi. Il terzo definisce estensioni di WS-Transfer che riguardano principalmente l'accesso fragment-based a risorse per soddisfare i requisiti comuni di WS-ResourceFramework e WS-Management. Il quarto descrive un protocollo generale basato su SOAP per accedere alla rappresentazione XML di risorse Web basate su servizi. Il quinto definisce come i metadati associati al Web service endpoint possano essere rappresentati come risorse, come i metadati possano essere inclusi negli endpoint reference, e come i metadati possano essere recuperati da un Web service endpoint. Ulteriori informazioni su Web Services Activity. (Archivio News Inglese, Italiano)
