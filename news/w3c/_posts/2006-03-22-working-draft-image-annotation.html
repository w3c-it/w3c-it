--- title: "Working Draft: Image Annotation" date: 2006-03-22 layout: page categories: --- Il Semantic Web Best Practices and Deployment Working Group ha pubblicato il Primo Public Working Draft di Image Annotation on the Semantic Web. Prodotto dal Gruppo Multimedia Annotation all'interno della Semantic Web Task Force, il draft descrive creazione, archiviazione, manipolazione, interscambio ed elaborazione di metadati immagini. Sono forniti linee-guida, una descrizione di tool e vocabolari RDF ed OWL. Maggiori informazioni sulla Semantic Web home page. (Archivio News inglese, italiano)
