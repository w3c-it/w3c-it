--- title: "Enabling Read Access: Working Draft" date: 2007-06-18 layout: page categories: --- Il Web Application Formats (WAF) Working Group ha rilasciato un Working Draft aggiornato di Enabling Read Access for Web Resources. Con questo meccanismo, le restrizioni sandbox sull'accesso cross-site ai browser possono essere selettivamente distese. Un'intestazione HTTP o un'istruzione di processo XML o entrambe, possono indicare che è permesso l'accesso in lettura. Maggiori informazioni sulla pagina della Rich Web Clients Activity.
