--- title: "Il W3C incoraggia l'implementazione di State Chart XML (SCXML): State Machine Notation for Control Abstraction" date: 2014-03-13 layout: page categories: w3c ---

Il Voice Browser Working Group incoraggia l'implementazione della Candidate Recommendation di State Chart XML (SCXML): State Machine Notation for Control Abstraction. Questo documento descrive l'SCXML, “State Chart extensible Markup Language” L'SCXML  fornisce un ambiente di esecuzione a macchine a stati basato su CCXML e Harel State Tables. Per ulteriori informazioni consulta Voice Browser Activity.
