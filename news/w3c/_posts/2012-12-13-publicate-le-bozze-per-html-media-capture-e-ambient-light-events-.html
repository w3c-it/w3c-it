Publicate le bozze per HTML Media Capture e Ambient Light Events

13 dicembre 2012 | Archivio

Il Device APIs Working Group ha pubblicato oggi due documenti:

  • Una bozza di lavoro su HTML Media Capture. Le specifiche HTML Media Capture definiscono una estensione di form HTML che facilita l'accesso di un utente al meccanismo di un dispositivo che cattura multimediali, come una videocamera, o un microfono, all'interno di un controllo per l'upload di file.
  • Una bozza di lavoro 'last call' per Ambient Light Events. Queste specifiche definiscono modalità per ricevere eventi che corrispondono ad un sensore che avverte la presenza di luce. I commenti sono benvenuti fino al 26 gennaio.

Per saperne di più vai a Ubiquitous Web Applications Activity.
