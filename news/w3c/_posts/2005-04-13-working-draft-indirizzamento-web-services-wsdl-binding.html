--- title: "Working Draft: Indirizzamento Web Services WSDL Binding" date: 2005-04-13 layout: page categories: --- Il Working Group Web Services Addressing ha rilasciato un Working Draft aggiornato di Indirizzamento Web Services 1.0 - WSDL Binding. Il documento definisce come le proprietà in Indirizzamento Web Services 1.0 - Core siano descritte nel linguaggio: Web Services Description (WSDL). L'indirizzamento Web Services fornisce i meccanismi transport-neutral ed è progettato per funzionare con entrambe le versioni WSDL 1.1 e 2.0. Maggiori informazioni su: Web services.
