--- title: "Paged Media, Values and Units: CSS3 Working Draft" date: 2006-09-19 categories: ---

Il CSS Working Group rilasciato due Working Draft aggiornati per Cascading Style Sheets Level 3 (CSS3). Generated Content for Paged Media descrive le caratteristiche come le referenze incrociate, le note a pié di pagina, e le intestazioni, spesso usate in pubblicazioni a stampa. Values and Units spiega, specifica, computa, valuta e definisce i valori comuni e le unità in una specifica che può essere riferita anche ad altri moduli CSS3. Maggiori informazioni sulla CSS home page. (Archivio News inglese, italiano)
