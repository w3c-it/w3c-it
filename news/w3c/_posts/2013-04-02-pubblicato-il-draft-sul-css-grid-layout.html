--- title: "Pubblicato il Draft sul CSS Grid Layout" date: 2013-04-02 categories: ---

Il Cascading Style Sheets (CSS) Working Group ha pubblicato un working draft sul CSS Grid Layout. Questo modulo CSS definisce un sistema di layout bi-dimensionale, ottimizzato per il design di interfacce utente. Nel modello grid layout i figli di un contenitore grid possono essere posizionati in slot arbitrari in un layout a griglia flessibile o fisso.  Ulteriori informazioni sulla Style Activity.
