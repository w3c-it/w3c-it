--- title: "W3C mobileOK Basic Tests 1.0 Advances to Candidate Recommendation" date: 2007-11-13 layout: page categories: --- Il Mobile Web Best Practices Working Group ha pubblicato la Candidate Recommendation di W3C mobileOK Basic Tests 1.0. Questo documento definisce i test che forniscono le basi per pretendere la conformità al W3C mobileOK Basic e sono basate sulle Mobile Web Best Practices W3C. W3C invita ad utilizzare la versione alpha del W3C mobileOK Checker per testare i contenuti. Maggiori informazioni sulla press release e sulla home page della Mobile Web Initiative Activity.
