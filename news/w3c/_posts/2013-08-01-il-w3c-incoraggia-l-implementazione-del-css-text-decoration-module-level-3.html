--- title: "Il W3C incoraggia l'implementazione del CSS Text Decoration Module Level 3" date: 2013-08-01 layout: page categories: w3c ---

Il Cascading Style Sheets (CSS) Working Group incoraggia l'implementazione della Candidate Recommendation su CSS Text Decoration Module Level 3. Questo modulo contiene le caratteristiche del CSS riguardanti text decoration, come la sottolineatura, le ombre del testo, e i segni di enfasi. Il CSS è un linguaggio per la descrizione e il rendering di documenti strutturati (come HTML e XML) su schermo, su carta, orale, ecc.. Ulteriori informazioni sulla Style Activity.
