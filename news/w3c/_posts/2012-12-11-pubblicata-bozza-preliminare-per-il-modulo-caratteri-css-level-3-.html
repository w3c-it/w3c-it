Pubblicata bozza preliminare per il modulo caratteri CSS Level 3

11 Dicembre 2012

Il gruppo di lavoro sul CSS ha pubblicato una bozza preliminare del modulo caratteri CSS Level 3. Questo modulo CSS3 descrive come devono essere specificate le proprietà dei caratteri e come le risorse vengono caricate dinamicamente. Questa specifica unisce i contenuti precedentemente suddivisi nei moduli Caratteri CSS3 e Caratteri Web CSS3. Per saperne di più vai a Style Activity.
