--- title: "Last Call: Web Services Addressing WSDL Binding" date: 2006-02-16 layout: page categories: w3c --- Il Web Services Addressing Working Group ha rilasciato un Last Call Working Draft di Web Services Addressing 1.0 - WSDL Binding. I commenti sono benvenuti fino al 31 marzo. Il documento definisce come le proprietà in Web Services Addressing 1.0 - Core siano descritte nel Web Services Description Language (WSDL). L'indirizzamento Web Service fornisce meccanismi transport-neutral ed è progettato per funzionare con entrambe le versioni WSDL 1.1 e 2.0. Maggiori informazioni sulla pagina dei Web service al W3C.
