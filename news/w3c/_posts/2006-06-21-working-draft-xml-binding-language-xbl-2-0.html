--- title: "Working Draft: XML Binding Language (XBL) 2.0" date: 2006-06-21 categories: --- Il Web Application Formats Working Group ha rilasciato il First Public Working Draft di XML Binding Language (XBL) 2.0, tecnologia per estendere l'aspetto e il comportamento degli elementi nel formati Web come HTML. Questo draft è stato prodotto dalla specifica XBL 2.0, sviluppata dalla Mozilla Foundation, e adesso è stato inserito nel W3C Recommendation Track. Maggiori informazioini sulla pagina della Rich Web Clients Activity. (Archivio News inglese, italiano)
