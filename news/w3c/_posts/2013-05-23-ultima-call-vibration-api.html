--- title: "Ultima Call:Vibration API" date: 2013-05-23 layout: page categories: ---

Il Gruppo di Lavoro Device APIs ha pubblicato un'ultima Call per la Bozza di Lavoro su  Vibration API. Questa specifica definisce un API che fornisce accesso al meccanismo di vibrazione del dispositivo ospitante. La vibrazione è una forma di feedback tattile. I commenti sono benvenuti fino al 13 Giugno. Per saperne di più vai a  Ubiquitous Web Applications Activity.
