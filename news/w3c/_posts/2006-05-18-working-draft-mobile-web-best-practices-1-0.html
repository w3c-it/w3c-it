--- title: "Working Draft: Mobile Web Best Practices 1.0" date: 2006-05-18 layout: page categories: w3c --- Il Mobile Web Best Practices Working Group ha pubblicato un Working Draft aggiornato di Mobile Web Best Practices 1.0 con inclusi i commenti provenienti dal loro ultimo Last Call Working Draft del 18 aprile 2006. Il documento cerca di migliorare la user experience, descrivendo come produrre Web content e siti Web progettati per dispositivi mobili e small-screen. Maggiori informazioni sulla Mobile Web Best Practices Working Group home page. (Archivio News inglese, italiano)
