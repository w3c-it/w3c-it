--- title: "Last Call: W3C mobileOK Basic Tests 1.0" date: 2007-05-25 layout: page categories: --- Il Mobile Web Best Practices Working Group ha rilasciato un secondo Last Call Working Draft di W3C mobileOK Basic Tests 1.0. I commenti sono benvenuti fino al 22 giugno. Questi test forniscono le basi per la richiesta di uniformità al W3C mobileOK Basic e sono basati sulle Mobile Web Best Practices del W3C. Maggiori informazioni sulla pagina della Mobile Web Initiative. (Archivio News inglese, italiano)
