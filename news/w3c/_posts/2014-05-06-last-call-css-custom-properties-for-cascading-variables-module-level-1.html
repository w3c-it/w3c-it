--- title: "Last Call: CSS Custom Properties for Cascading Variables Module Level 1" date: 2014-05-06 layout: page categories: ---

Il Cascading Style Sheets (CSS) Working Group ha pubblicato un Last Call Working Draft di CSS Custom Properties for Cascading Variables Module Level 1. Questo modulo presenta le variabili cascading come un nuovo tipo primitivo che viene accettato da tutte le proprietà CSS, e proprietà particolari per definirle. CSS è un linguaggio per la descrizione e il rendering di documenti strutturati (come HTML e XML) su schermo, su carta, ecc. Si accettano commenti fino al 3 giugno 2014. Per ulteriori informazioni consulta Style Activity.
