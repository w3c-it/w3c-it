--- title: "Call for Review: Device Description Repository Simple API Proposed Recommendation" date: 2008-09-22 layout: page categories: w3c --- Il Mobile Web Initiative Device Description Working Group ha pubblicato la Proposed Recommendation di Device Description Repository Simple API. I contenuti del Web per gli strumenti mobile di solito beneficiano dall'essere ritagliati per tener conto di una gamma di fattori quali la dimensione dello schermo, il supporto per linguaggi di markup e per i formati delle immagini. Queste informazioni vengono registrate nei "Device Description Repositories" (DDRs). Questo documento descrive una semplice API per l'accesso ai DDRs, in modo da semplificare e promuovere lo sviluppo di contenuti Web che si adattano al loro Delivery Context. Commenti sono bene accetti fino al 31 ottobre. Ulteriori informazioni su Mobile Web Initiative Activity. (Archivio News Inglese, Italiano)
