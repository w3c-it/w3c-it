--- title: "Widgets 1.0: API ed eventi" date: 2009-02-10 layout: page categories: w3c --- Il Web Applications Working Group ha pubblicato la First Public Working Draft di Widgets 1.0: APIs and Events. Questa specifica definisce un insieme di API ed eventi per la famiglia di specifiche di Widgets 1.0 che permettono le funzionalità base per i widget. L'API e eventi definiti da questa specifica stabiliscono, fra le altre cose, il significato di:
  • accesso ai metadati dichiarati nel documento di configurazione del widget,
  • ricezione di eventi collegati ai cambiamenti nello stato visibile del widget,
  • determinazione del luogo in cui un widget viene eseguito,
  • avere notifica degli eventi collegati all'aggiornamento del widget,
  • invocare un widget per aprire un URL sul browser di default del sistema,
  • catturare l'attenzione dell'utente in un modo indipendente dallo strumento,
  • e controllare che ogni API addizionale richiesta attraverso il documento di configurazione venga caricata con successo.
