--- title: "Il W3C invita ad implementare la Indexed Database API" date: 2013-07-04 categories: ---

Il Web Applications Working Group invita ad implementare la Candidate Recommendation della Indexed Database API. Questo documento definisce una API per un database contenente semplici valori o oggetti gerarchici. Ogni record consiste in una chiave ed un certo valore. Inoltre il database mantiene degli indici sui record che contiene. Uno sviluppatore di applicazioni usa direttamente la API per individuare i record o in base alla chiave o usando un indice. Un linguaggio di query può essere lo strato successivo a questa API. Un database indicizzato può essere implementato usando una struttura a B-tree persistente.
