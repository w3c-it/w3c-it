--- title: "\"DOM Level 3 Core\" e \"DOM Level 3 Load and Save\" Sono W3C Candidate Recommendation" date: 2003-11-07 layout: page categories: w3c ---

W3C è lieto di annunciare l' avanzamento di due specifiche del Document Object Model (DOM) a Candidate Recommendations. Con DOM Level 3 Core, gli sviluppatori di software e gli autori di script Web possono avere accesso e manipolare contenuti HTML e XML. DOM Level 3 Load and Save consente a programmi e script di caricare in maniera dinamica il contenuto di un documento XML in un documento DOM e di seralizzare un documento DOM in un documento XML. Si attendono commenti entro il 30 novembre. DOM Level 3 Events è stato pubblicato come Working Group Note. Per ulteriori informazioni: DOM home page.
