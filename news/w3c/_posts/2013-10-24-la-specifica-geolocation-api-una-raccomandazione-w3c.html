--- title: "La Specifica Geolocation API è una Raccomandazione W3C" date: 2013-10-24 categories: ---

Il Geolocation Working Group ha pubblicato una Raccomandazione W3C su Geolocation API Specification.Questa specifica definisce un API che consente l'accesso script a informazioni sulla posizione geografica associato al dispositivo di hosting. Per saperne di più: Ubiquitous Web Applications Activity.
