--- title: "Ultima Chiamata: Direttive di Sicurezza Interfaccia Utente per Politica di Sicurezza dei Contenuti" date: 2014-03-18 categories: ---

Il Web Application Security Working Group ha pubblicato un'Ultima Chiamata per Working Draft su Direttive di Sicurezza Interfaccia Utente per Politica di Sicurezza dei Contenuti. Questo documento definisce le direttive per il meccasismo di Politica di Sicurezza dei Contenuti di dichiarare una serie di protezioni input per l'interfaccia utente di una risorsa web, definisce un insieme non-normativo di euristiche per user agent Web per implementare queste protezioni in ingresso, e un meccanismo di segnalazione per quando sono innescate. I commenti sono benvenuti fino al 18 Giugno. Per saperne di più circa la Security Activity.
