--- title: "Pubblicato State Chart XML (SCXML) Draft" date: 2010-05-17 categories: ---

Il Voice Browser Working Group ha pubblicato un Working Draft di State Chart XML (SCXML): State Machine Notation for Control Abstraction. SCXML è un linguaggio di stato della macchina general-purpose basato su eventi che può essere utilizzato in molti modi. Può essere utilizzato come linguaggio di dialogo ad alto livello per controllare i moduli vocali. incapsulati in VoiceXML 3.0.  Può anche essere utilizzato come un metalinguaggio per applicazioni vocali, dove, oltre alle funzionalità di VoiceXML 3.0, può anche controllare l'accesso ai database e ai moduli di logica business. Per ulteriori informazioni si consulti Voice Browser Activity.
