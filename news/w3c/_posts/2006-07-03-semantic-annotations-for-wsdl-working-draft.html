--- title: "Semantic Annotations for WSDL: Working Draft" date: 2006-07-03 layout: page categories: --- Il Semantic Annotations for Web Services Description Language (SAWSDL) Working Group ha rilasciato il First Public Working Draft di Semantic Annotations for WSDL. Gli attributi definiti in questo draft sono referenze dagli elementi dentro documenti Web Services Description Language (WSDL) o XML Schema per concetti in ontologie all'esterno dei documenti. Maggiori informazioni sulla pagine dei Web services. (Archivio News inglese, italiano)
