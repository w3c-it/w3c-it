--- title: "POWDER Semantica formale, prima Working Draft pubblicata" date: 2008-07-09 layout: page categories: --- Il Protocol for Web Description Resources (POWDER) Working Group ha pubblicato il primo Working Draft pubblico di Protocol for Web Description Resources (POWDER): Formal Semantics. Questo documento sostiene il Protocol for Web Description Resources (POWDER). Descrive come il formato operativo, relativamente semplice, di un documento POWDER puo' essere trasformato attraverso due fasi, in primo luogo in un formato XML piu' restrittivo (POWDER-BASE), e poi in una codifica RDF/OWL (POWDER-S) che puo' essere utilizzata dai Semantic Web tools. Si capisce meglio la semantica formale di POWDER informandosi sui documenti Description Resources e Grouping of Resources. Ulteriori informazioni su Semantic Web Activity.
