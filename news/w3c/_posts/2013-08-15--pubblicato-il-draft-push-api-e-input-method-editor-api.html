--- title: " Pubblicato il Draft Push API e Input Method Editor API" date: 2013-08-15 categories: ---

L'Web Applications Working Group ha pubblicato due Working Drafts:

  • Push API. Questa specifica definisce un “Push API” che fornisce applicazioni web con accesso tramite script per le notifiche inviate dal server, per semplicità di cui qui come notifiche push, come consegnate da servizi push. I servizi push sono un modo per applicazioni server di mandare messaggi alle webapp, sia che la webapp sia attiva in una finestar del browser sia che non lo sia.
  • Input Method Editor API. Questa specifica definisce un “IME API” che fornisce applicazioni Web con accesso tramite script ad un IME (input-method editor) associato a un hosting user agent.
