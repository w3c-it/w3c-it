--- title: "Ultima chiamata: HTML5 Image Description Extension (longdesc) " date: 2013-07-16 layout: page categories: ---

L' HTML Working Group ha pubblicato una Last Call Working Draft del HTML5 Image Description Extension. Questa specifica definisce l'attributo "longdesc" che abilita gli autori web a fornire descrizioni testuali più lunghe per immagini complesse ed è parte del lavoro del W3C assicurare che l'Open Web Platform sia accessibile a persone con disabilità. È sviluppato dall'HTML Accessibility Task Force in coordinamento con il HTML Working Group and il Gruppo di Lavoro Protocolli e Formati WAI (PFWG). Vedi maggiori informazioni sull'HTML5 Image Description Extension Last Call e-mail. Commenti sono benvenuti fino al 16 Settembre 2013. Maggiorni informazioni circa l'HTML Activity e la Web Accessibility Initiative (WAI).
