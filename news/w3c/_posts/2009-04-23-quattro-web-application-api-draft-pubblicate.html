--- title: "Quattro Web Application API Draft pubblicate" date: 2009-04-23 layout: page categories: w3c --- Il Web Applications Working Group ha pubblicato quattro First Public Working Draft di specifiche per API che migliorano la piattaforma Web open come un runtime environment per applicazioni complete:
  • Web Storage fornisce una API per la persistenza dei dati client-side per le applicazioni Web.
  • Web Workers definisce una API per abilitare operazioni simili ai thread (usando lo scmabio di messaggi) nelle applicazioni Web, così che certi compiti delle applicazioni possano essere eseguite in parallelo.
  • Web Sockets API fornisce una API per la comunicazione full-duplex fra un'applicazione Web e un host remoto.
  • Server-Sent Events definisce una API per aprire una connessione HTTP per la ricezione di notifiche dal server sotto forma di eventi del DOM.

La specifiche Web Storage, Web Sockets API, e Server-Sent Events erano state pubblicate in precedenza come parte della specifica di HTML 5, ma ora saranno ciascuna una Recommendation-track da usare nel Web Applications Working Group.
