--- title: "Best Practice Recipes per pubblicare vocabolari RDF Nota pubblicata" date: 2008-08-29 layout: page categories: w3c --- The Semantic Web Deployment Working Group has published the Group Note of Best Practice Recipes for Publishing RDF Vocabularies. This document describes best practice recipes for publishing vocabularies or ontologies on the Web (in RDF Schema or OWL). It is intended for the creators and maintainers of vocabularies in RDFS and OWL (vocabulary and ontology are used interchangeably in the context of this specification). It provides step-by-step instructions for publishing vocabularies on the Web, giving example configurations designed to cover the most common cases. Learn more about the Semantic Web Activity.
