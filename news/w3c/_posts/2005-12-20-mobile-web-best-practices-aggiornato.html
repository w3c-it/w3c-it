--- title: "Mobile Web Best Practices aggiornato" date: 2005-12-20 layout: page categories: --- Il Mobile Web Best Practices Working Group ha rilasciato un Working Draft aggiornato di Mobile Web Best Practices 1.0. Il draft descrive come produrre Web content e siti Web destinati al trasferimento su dispositivi mobili o dotati di small-screen. Obiettivo del Mobile Web Best Practices era pubblicare il draft come Working Group Note. Maggiori informazioni sulla pagina della W3C Mobile Web Initiative, sforzo congiunto di fornitori di authoring tool, content provider, produttori di microtelefoni, fornitori di browser e operatori del mobile.
