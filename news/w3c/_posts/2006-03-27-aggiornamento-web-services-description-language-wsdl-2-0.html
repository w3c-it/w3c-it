--- title: "Aggiornamento: Web Services Description Language (WSDL) 2.0" date: 2006-03-27 layout: page categories: w3c --- Il Web Services Description Working Group ha aggiornato tre Candidate Recommendations per il Web Services Description Language (WSDL) Versione 2.0: Part 0: Primer, Part 1: Core Language e Part 2: Adjuncts. I commenti sono benvenuti fino all'1 luglio. WSDL RDF Mapping e SOAP 1.1 Binding sono Working Draft aggiornati. WSDL 2.0 modella e rinnova i Web service modulari ed è utilizzato per documentare sistemi distribuiti ed automatizzare la comunicazione tra le applicazioni. Maggiori informazioni sulla pagina dei Web service. (Archivio News inglese, italiano)
