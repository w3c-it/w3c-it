--- title: "Last Call: Web Services Description Language (WSDL) 2.0" date: 2007-03-27 layout: page categories: --- Il Web Services Description Working Group ha rilasciato tre Last Call Working Draft per il Web Services Description Language (WSDL) Versione 2.0: Part 0: Primer, Part 1: Core Language e Part 2: Adjuncts. I commenti sono benvenuti fino al 15 aprile su questo breve Last Call per i cambiamenti da effettuare fino alla revisione della Candidate Recommendation. WSDL RDF Mapping e SOAP 1.1 Binding sono Working Draft aggiornati. WSDL 2.0 modella e descrive Web service modulari ed è usato per documentare sistemi distribuiti ed automatizzare la comunicazione tra le applicazioni. Maggiori informazioni sulla pagina dei Web service. (Archivio News inglese, italiano)
