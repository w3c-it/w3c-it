--- title: "La Guida di Implementazione WAI-ARIA 1.0 User Agent è una Candidate Recommendation" date: 2013-12-17 layout: page categories: w3c ---

Il Protocols and Formats Working Group (PFWG) oggi ha pubblicato la Guida di Implementazione di WAI-ARIA 1.0 User Agent come Candidate Recommendation.Essa descrive come i browser ed altri user agent dovrebbero supportare WAI-ARIA (the Accessible Rich Internet Applications specification), in particolare come offrire le caratteristiche di WAI-ARIA a piattaforme di accessibilità APIs. Commenti e implementazioni sono benvenuti fino al 17 Gennaio 2014. Per saperne di più circa Web Accessibility Initiative (WAI).
