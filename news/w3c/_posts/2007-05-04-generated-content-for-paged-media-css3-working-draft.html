--- title: "Generated Content for Paged Media: CSS3 Working Draft" date: 2007-05-04 layout: page categories: ---

Il CSS Working Group ha rilasciato un Working Draft aggiornato per Cascading Style Sheets Level 3 (CSS3). Generated Content for Paged Media descrive le caratteristiche come le referenze incrociate, le note a pié di pagina, e le intestazioni, spesso usate in pubblicazioni a stampa. CSS è il linguaggio di stile più ampiamente implementato, usato per ottenere documenti strutturati come HTML and XML su schermo, carta e in sintesi vocale. Maggiori informazioni sulla CSS home page.
