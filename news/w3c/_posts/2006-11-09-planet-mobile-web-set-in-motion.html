--- title: "Planet Mobile Web Set in Motion" date: 2006-11-09 categories: --- W3C è lieto di presentare Planet Mobile Web. Questa comunità di servizio serve per discutere attraverso lo strumento dei blog sull'utilizzo del mobile Web e sarà utile per veicolare nuove idee. Il Planet trasmetterà le notizie in digest HTML e RSS/Atom feed. Maggiori informazioni sulla pagina della W3C Mobile Web Initiative, sforzo congiunto di fornitori di authoring tool, content provider, produttori di microtelefoni, fornitori di browser ed operatori del mobile.
