--- title: "Last Call: XML Schema Patterns for Databinding" date: 2006-11-22 layout: page categories: --- L'XML Schema Patterns for Databinding Working Group ha rilasciato un Last Call Working Draft di Basic XML Schema Patterns for Databinding Versione 1.0 ed il First Public Working Draft di Advanced Patterns. Il pattern descrive rappresentazioni XML 1.0 di strutture dati comunemente usate, indipendenti da linguaggi di programmazione, database o modelling environment. Il basic set è noto per essere interoperabile fra le esecuzioni databinding avanzate, mentre i modelli avanzati sono d'uso comune, ma sono utilizzati per effettuare le pubblicazioni. I commenti al Last Call sono benvenuti fino al 12 gennaio. Maggiori informazioni sulla pagina dei Web service. (Archivio News inglese, italiano)
