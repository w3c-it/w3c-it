--- title: "Pubblicati Cinque Working Draft su XQuery e XPath" date: 2003-08-22 layout: page categories: w3c ---

L' XML Query Working Group e l' XSL Working Group hanno pubblicato cinque Working Draft. Si attendono commenti su tutti questi documenti, in particolare sui problemi aperti. Per maggiori informazioni si visiti: XML home page. I Working Draft pubblicati sono:

 

  • XML Path Language (XPath) 2.0
  • XQuery 1.0: An XML Query Language
  • XQuery 1.0 and XPath 2.0 Formal Semantics
  • XML Query Use Cases
  • XPath Requirements Version 2.0
