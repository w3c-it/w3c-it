--- title: "Internationalization Tag Set Is a Proposed Recommendation" date: 2007-02-26 layout: page categories: --- W3C is pleased to announce the advancement of Internationalization Tag Set (ITS) Version 1.0 to Proposed Recommendation. Comments are welcome through 26 March. Organized by data categories, the ITS set of elements and attributes supports the internationalization and localization of schemas and documents. Implementations are provided for DTDs, XML Schema and Relax NG, and can be used with new or existing vocabularies like XHTML, DocBook and OpenDocument. Visit the Internationalization home page.
