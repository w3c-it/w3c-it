--- title: "Ultima Call: CSS Counter Styles Level 3" date: 2014-08-26 layout: page categories: w3c ---

Il Cascading Style Sheets (CSS) Working Group ha pubblicato una ultima Call per la Bozza di Lavoro su CSS Counter Styles Level 3. Questo modulo introduce la regola counter-style@, che permette agli autori di definire i propri stili contatore personalizzati per l'utilizzo con i CSS list-marker e contatori generated-content. Esso predefinisce anche una serie di stili di contatori comuni, tra cui quelli presenti nei CSS2 e CSS 2.1. I commenti sono benvenuti fino al 23 September. CSS è un linguaggio per descrivere il rendering di documenti strutturati (come HTML e XML) sullo schermo, sulla carta, nel parlato, ecc. Per saperne di più vai a Style Activity.
