--- title: "Versioning XML Languages Using XML Schema 1.1" date: 2006-09-28 layout: page categories: --- L'XML Schema Working Group ha rilasciato il First Public Working Draft di Guide to Versioning XML Languages using XML Schema 1.1. XML Schema 1.1 introduce nuove feature che rendono più semplice definire i linguaggi XML flessibili per tollerare revisioni successive in un modo forward-compatible. Scritta per applicazioni e sviluppatori di schemi, la guida mostra nuovi meccanismi ed illustra numerose tecniche. Il gruppo richiede commenti su questo draft che potrebbe diventare Working Group Note. Maggiori informazioni sulla XML home page. (Archivio News inglese, italiano)
