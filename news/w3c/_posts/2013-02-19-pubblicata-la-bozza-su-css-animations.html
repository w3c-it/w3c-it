--- title: "Pubblicata la Bozza su CSS Animations" date: 2013-02-19 layout: page categories: ---

Il Cascading Style Sheets (CSS) Working Group ha pubblicato un Working Draft sulle CSS Animations. Questo modulo CSS descrive il modo con cui gli autori possono animare (specificare cambiamenti) nel tempo le proprietà CSS, usando keyframes. Il comportamento di queste animazioni keyframe può essere controllato specificandone la durata, il numero di ripetizioni e il comportamento ripetitivo da tenere. Per saperne di più vai a Style Activity.
