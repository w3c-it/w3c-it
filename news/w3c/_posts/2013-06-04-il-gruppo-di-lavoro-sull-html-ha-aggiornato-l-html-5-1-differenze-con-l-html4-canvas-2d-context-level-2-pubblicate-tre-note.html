--- title: "Il gruppo di lavoro sull'HTML ha aggiornato l'HTML 5.1; Differenze con l'HTML4; Canvas 2D Context Level 2; Pubblicate tre Note" date: 2013-06-04 layout: page categories: ---

L'HTML Working Group ha pubblicato tre Working Drafts e tre Group Notes:

  • un Working Draft su HTML 5.1. Questa specifica definisce la quinta versione principale, prima revisione minore, del linguaggio fondamentale del World Wide Web: l'Hypertext Markup Language (HTML). In questa versione continuano ad essere introdotte nuove caratteristiche per aiutare gli autori di applicazioni Web, e nuovi elementi basati sulla ricerca riguardante le pratiche di authoring prevalenti. Continua inoltre ad essere data attenzione speciale alla definizione di criteri chiari di conformità per gli user agent, con lo scopo di migliorare l'interoperabilità.
  • un Working Draft su Differences from HTML4. L'HTML è il linguaggio fondamentale del World Wide Web. Il W3C pubblica l'HTML e l'HTML 5.1. Il WHATWG pubblica l'HTML, che è un soprainsieme dell'HTML 5.1 del W3C. "Differences from HTML4" descrive le differenze tra la specifica dell'HTML e quella dell'HTML4, e mette in risalto i casi in cui sono in disaccordo.
  • un Working Draft su HTML Canvas 2D Context, Level 2. Questa specifica definisce il 2D Context per l'elemento canvas dell'HTML. Il 2D Context fornisce oggetti, metodi e proprietà per disegnare e manipolare grafiche su una superfice di disegno di tipo canvas.
  • una Group Note su HTML5: Edition for Web Authors. Questo documento, ora interrotto e reso disponibile per motivi storici, è indirizzato agli autori Web e ad altri che non siano sviluppatori di User Agent che vogliono una visione della specifica dell'HTML, più focalizzata sui dettagli rilevanti per l'uso del linguaggio HTML per la creazione di documenti e pagine Web.
  • una Group Note su main element - an HTML5 extension specification. Questa specifica, ora integrata in quella dell'HTML e non più sviluppata separatamente, definisce un elemento da usare per l'identificazione del contenuto principale di un documento. Definisce anche un elemento HTML che incarna la semantica e la funzione del WAI-ARIA landmark role=main
  • una Group Note su HTML: The Markup Language (an HTML language reference). Questo documento descrive il linguaggio di markup HTML e fornisce dettagli per aiutare i produttori di contenuto HTML a creare documenti conformi al linguaggio. Questo documento è stato interrotto e viene reso disponibile per motivi storici. Per un riferimento aggiornato agli elementi HTML (e molto di più) consultate Web Platform Docs.
