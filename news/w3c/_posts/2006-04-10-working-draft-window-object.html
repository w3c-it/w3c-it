--- title: "Working Draft: Window Object" date: 2006-04-10 layout: page categories: ---

Il Web API Working Group ha rilasciato il First Public Working Draft di Window Object 1.0. Il draft definisce il Window object, standard de facto long-standing. Window fornisce il namespace globale per i linguaggi di scripting Web, l'accesso a documenti differenti in un documento composto per referenze, timer e navigazione per altre location. Maggiori informazioni sulla home page della Rich Web Clients Activity. (Archivio News inglese, italiano)
