--- title: "Call for Review: XQuery 3.0, XPath 3.0, Data Model, Functions e Operators e XSLT e XQuery Serialization 3.0" date: 2013-10-22 layout: page categories: ---

L' XML Query Working Group e il XSLT Working Group hanno pubblicato oggi cinque Proposed Recommendations:

  • XQuery 3.0: An XML Query Language. XQuery è un linguaggio domain-specific per interrogare collezioni di alberi XDM.
  • XML Path Language (XPath) 3.0. XPath è un linguaggio domain-specific per fare riferimento a parti di un albero rappresentato come un'istanza di XPath e XQuery Data Model (XDM).
  • XQuery and XPath Data Model 3.0. Questa specifica definisce l' XPath e l' XQuery Data Model (XDM).
  • XPath and XQuery Functions and Operators 3.0. La specifica Functions e Operators definisce strutture per XPath e XQuery.
  • XSLT and XQuery Serialization 3.0. Serializzazione specifica come rappresentare un'istanza XDM come XML, HTML o come text. L' XDM è più spesso usato per rappresentare documenti XML come alberi, ma può anche rappreentare altri formati di dati come JSON, RDF o databases relazionali.

I commenti sono benvenuti fino al 19 Novembre. Per saperne di più: Extensible Markup Language (XML) Activity.
