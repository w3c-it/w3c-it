--- title: "Pubblicato Proposal for XML Fragment Identifier Syntax" date: 2003-09-12 layout: page categories: w3c ---

L' XML Linking Working Group ha pubblicato Proposal for XML Fragment Identifier Syntax 0.9 come W3C Working Group Note. Il documento suggerisce una sintassi per gli identificatori di frammento che utilizza XPointer Framework e raccomanda che gli XML media type (RFC 2376 e RFC 3023) la adottino per usarla con risorse XML. Si invita a fare commenti. Per maggiori informazioni: XML home page.
