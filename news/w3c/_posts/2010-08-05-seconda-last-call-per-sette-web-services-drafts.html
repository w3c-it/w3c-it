--- title: "Seconda Last Call per sette Web Services Drafts" date: 2010-08-05 categories: ---

Il Web Services Resource Access Working Group ha pubblicato sette Second Last Call Working Drafts per Web Services: Enumeration (WS-Enumeration), Event Descriptions (WS-EventDescriptions), Eventing (WS-Eventing), Fragment (WS-Fragment), Metadata Exchange (WS-MetadataExchange), SOAP Assertions (WS-SOAPAssertions), e Transfer (WS-Transfer). I commenti sono benvenuti fino al 17 settembre 2010. Per saperne di più si veda Web Services Activity.
