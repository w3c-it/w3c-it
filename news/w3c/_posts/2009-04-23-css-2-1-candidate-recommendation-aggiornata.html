--- title: "CSS 2.1 Candidate Recommendation aggiornata" date: 2009-04-23 layout: page categories: --- Il Cascading Style Sheets (CSS) Working Group ha aggiornato la Candidate Recommendation di Cascading Style Sheets Level 2 Revision 1 (CSS 2.1) Specification. CSS 2.1 è un linguaggio per style sheet che permette a autori e utenti di aggiungere stile (ad esempio, font e spaziatura) a documenti strutturati (ad esempio documenti HTML e applicazioni XML). CSS 2.1 corregge alcuni errori in CSS2 (il più importante riguarda una nuova deinizione di height/width di posizionamento assoluto, più influenza per l'attributo "style" di HTML e un nuovo calcolo della proprietà 'clip'), e aggiunge alcune caratteristiche molto richieste che sono state già largamente implementate. Ma soprattutto CSS 2.1 rappresenta uno "snapshot" dell'uso di CSS: consiste di tutte le caratteristiche di CSS che sono implementate in modo interoperabile. Questa draft incorpora un errata risultato dell'esperienz di miplementazione già dalla precedente pubblicazione. Ulteriori informazioni su Style Activity. (Archivio News Inglese, Italiano)
