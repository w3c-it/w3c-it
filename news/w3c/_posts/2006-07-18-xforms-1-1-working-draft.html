--- title: "XForms 1.1: Working Draft" date: 2006-07-18 layout: page categories: --- The XForms Working Group has released an updated Working Draft of XForms 1.1. Designed to refine and strengthen the XML processing platform introduced by XForms 1.0, version 1.1 adds several submission capabilities, a more powerful action processing facility, the ability to manipulate data arbitrarily and to access event context information, and adds numerous helpful data types, utility functions, user interface improvements, and action event handlers. Visit the XForms home page. (Archivio News inglese, italiano)
