--- title: "Authoring Tool Accessibility: Techniques Working Draft" date: 2007-04-23 layout: page categories: --- L'Authoring Tool Accessibility Guidelines Working Group ha pubblicato un Working Draft di Implementation Techniques for Authoring Tool Accessibility Guidelines 2.0. Queste tecniche, strategie-campione e risorse sono un aiuto per gli sviluppatori che vogliano soddisfare i checkpoint nelle Authoring Tool Accessibility Guidelines 2.0 (ATAG). ATAG aiuta gli sviluppatori a progettare tool ed interfacce di authoring che siano accessibili agli utenti e producano contenuti Web accessibili, con il risultato di ottenere contenuti Web veramente accessibili a tutti.
