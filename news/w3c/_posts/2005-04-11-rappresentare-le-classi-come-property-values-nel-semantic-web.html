--- title: "Rappresentare le classi come Property Values nel Semantic Web" date: 2005-04-11 layout: page categories: w3c --- La OEP Task Force del Working Group Semantic Web Best Practices and Deployment ha rilasciato come Nota del gruppo: Representing Classes As Property Values on the Semantic Web. La nota illustra l'approccio diretto per rappresentare le classi come property values nel OWL Web Ontology Language completo e in RDF Schema, meccanismi per OWL DL e OWL Lite e considerazioni per gli utenti. Maggiori informazioni sulla Semantic Web home page. (Archivio News inglese, italiano)
