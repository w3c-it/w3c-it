--- title: "Efficient XML Interchange Measurements: Working Draft" date: 2007-07-25 categories: --- L'Efficient XML Interchange Working Group ha rilasciato un Working Draft aggiornato di Efficient XML Interchange Measurements Note. Analisi delle performance caratteristiche di un potenziale formato encoding Efficient XML Interchange (EXI), il draft copre le proprietà "compattezza," "efficienza di processo" e "roundtrip support" e piani di massima per futuri aggiornamenti. Maggiori informazioni sulla XML home page. (Archivio News inglese, italiano)
