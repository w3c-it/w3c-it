--- title: "Ultima Call: Internationalization Tag Set (ITS) Versione 2.0" date: 2013-05-21 layout: page categories: w3c ---

Il MultilingualWeb-LT Working Group ha pubblicato un'ultima chiamata per la Bozza di Lavoro riguardo l'Internazionalizzazione dei Tag Set (ITS) Versione 2.0. L'Internazionalizzazione dei Tag Set (ITS) Versione 2.0 migliora le basi dell'integrazione dell'elaborazione automatica del linguaggio umano in tecnologie web di base. ITS 2.0 fornisce metadati per favorire l'adozione del web multilingue. Il progetto riflette tutti i cambiamenti dopo la pubblicazione dell'ultima Call. I commenti sono i benvenuti fino all'11 giugno. Per saperne di più sull'Attività di Internazionalizzazione.
