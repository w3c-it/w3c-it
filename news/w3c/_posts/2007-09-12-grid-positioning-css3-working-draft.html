--- title: "Grid Positioning: CSS3 Working Draft" date: 2007-09-12 categories: --- Il CSS Working Group ha rilasciato il First Public Working Draft del CSS Grid Positioning Module per Level 3 del linguaggio Cascading Style Sheets (CSS). Il Modulo applica il tradizionale grid systems utilizzato nei libri e nei giornali al contenuto online, in complemento al differente approccio definito nel CSS Advanced Layout Module. Le griglie possono essere generate esplicitamente o sottointese e combinate con le Media Query. Maggiori informazioni sulla CSS home page. (Archivio News inglese, italiano)
