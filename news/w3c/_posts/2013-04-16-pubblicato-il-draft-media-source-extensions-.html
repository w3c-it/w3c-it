Pubblicato il Draft Media Source Extensions

16 Aprile 2013

Il HTML Working Group ha pubblicato un Working Draft sulle Media Source Extensions. Questa specifica estende gli HTMLMediaElement per consentire a JavaScript di generare flussi multimediali per il playback. Consentire a JavaScript di generare stream facilita una varietà di use case come lo streaming adattivo e lo stream live time shifting. Per saperne di più vai al link HTML Activity.
