--- title: "Widgets 1.0 Requirements: Working Draft" date: 2007-07-05 layout: page categories: --- The Web Application Formats Working Group has released an updated Working Draft of Widgets 1.0 Requirements. These design goals are the requirements for device-independent standards for scripting, digitally signing, securing, packaging and deploying client-side Web applications (widgets). Also known as gadgets or modules, widgets are small programs like clocks, stock tickers, news casters, games and weather forecasters that display and update remote data and run on the Web browser environment.
