Call per Review: Pubblicata la Proposed Recommendation su Selectors API Level 1

13 dicembre 2012 | Archivio

Il Web Applications Working Group ha publicato una Proposed Recommendation su Selectors API Level 1. I selettori, ampiamente usati in CSS, sono pattern che si applicano ad elementi di una struttura ad albero. La specifica sulle API dei selettori definisce metodi per ottenere nodi Element dal DOM confrontandoli rispetto a un gruppo di selettori. Spesso é preferibile eseguire operazioni DOM su uno specifico insieme di elementi in un documento. Questi metodi semplificano il processo di acquisizione di elementi specifici, specialmente se comparati con tecniche più verbose definite e usate in passato. I commenti sono benvenuti fino al 25 gennaio. Per saperne di più vai a Rich Web Client Activity.
