--- title: "Pubblicato Draft su API Orientamento dello Schermo" date: 2014-02-20 categories: ---

Il Web Applications Working Group ha pubblicato un Working Draft sull' API Orientamento dello Schermo. L'API Orientamento dello Schermo offre la possibilità di leggere lo stato di orientamento dello schermo, per essere informato quando questo stato cambia e poter bloccare l'orientamento dello schermo in uno stato specifico. Per saperne di più circa la Rich Web Client Activity.
