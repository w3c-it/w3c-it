Publicata la bozza di Content Security Policy 1.1

13 dicembre 2012

Il Web Application Security Working Group ha publicato la prima bozza di lavoro su Content Security Policy 1.1. Questo documento definisce un linguaggio di policy usato per dichiarare un insieme di restrizioni di contenuto per una risorsa Web, e un meccanismo per trasmettere la policy da un server a un client, nel caso in cui viene imposta una policy. Per saperne di più vai a Security Activity.
