--- title: "Minor Update to W3C mobileOK Basic Tests 1.0 Candidate Recommendation" date: 2007-11-30 layout: page categories: --- Il Mobile Web Best Practices Working Group ha pubblicato un aggiornamento minore alla Candidate Recommendation di W3C mobileOK Basic Tests 1.0. L'update corregge il mobileOK User-Agent String. Il documento definisce i test che forniscono le basi per rivendicare la conformità mobileOK Basic del W3C e basata sulle W3C Mobile Web Best Practices. Maggiori informazioni sulla pagina Mobile Web Initiative Activity. (Archivio News inglese, italiano)
