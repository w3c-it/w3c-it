--- title: "Last Call: Authoring Tool Accessibility Guidelines (ATAG) 2.0, Implementing ATAG 2.0" date: 2010-07-08 layout: page categories: w3c ---

L'Authoring Tool Accessibility Guidelines Working Group ha pubblicato il Last Call Working Drafts di Authoring Tool Accessibility Guidelines (ATAG) 2.0 e il documento correlato Implementing ATAG 2.0. L'ATAG definisce come gli strumenti di authoring dovrebbero aiutare gli sviluppatori a creare contenuti Web accessibili conformi  alle Web Content Accessibility Guidelines (WCAG) 2.0. I documenti ATAG descrivono altresì le modalità per rendere gli strumenti di authoring accessibili, in modo che le persone con disabilità possano usarli. Commenti sono benvenuti entro il 2 settembre 2010. Si prega di leggere l' invitation to review the ATAG 2.0 Last Call Working Draft e la Web Accessibility Initiative (WA).
