--- title: "Pubblicato il Draft RDFa API " date: 2010-09-23 categories: ---

Il RDFa Working Group ha pubblicato il First Public Working Draft di RDFa API. RDFa permette agli autori di pubblicare un'informazione strutturata che è sia umana - che leggibile dalla macchina. I concetti che sono stati tradizionalmente difficili per le macchine da individuare, come le persone, luoghi, eventi, musica, film, e le ricette, sono ora facilmente marcati nei documenti Web. Mentre la pubblicazione di questi dati è vitale per la crescita dei Linked Data, il vero scopo è usare l'informazione per migliorare l'utilità collettiva del Web per l'umanità. Per raggiungere questo obiettivo, per gli sviluppatori Web deve essere semplice estrarre e utilizzare informazioni strutturate dal documento Web. Questo documento descrive un tale meccanismo; un'applicazione RDFa Programming Interface (RDFa API) che permette una semplice estrazione e l'utilizzo delle informazioni strutturate del documento Web. Per saperne di più si veda Semantic Web Activity.
