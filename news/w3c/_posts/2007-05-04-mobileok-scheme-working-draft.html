--- title: "mobileOK Scheme: Working Draft" date: 2007-05-04 categories: --- Il Mobile Web Best Practices Working Group rilascia un Working Draft aggiornato di W3C mobileOK Scheme 1.0. I mobileOK mark sono etichette machine-readable che indicano Web content e consegnano il lasciapassare alla test suite Mobile Web Best Practices. Progettato per creare una esperienza utente reale, mobileOK è scritto per autori di contenuti, sviluppatori di tool e content provider. Maggiori informazioni sulla W3C Mobile Web Initiative, sforzo congiunto di fornitori di authoring tool, content provider, produttori di microtelefoni, fornitori di browser ed operatori del mobile.
