--- title: "Use Cases e Requirements per Ontology e API per Media Object 1.0" date: 2009-01-20 categories: --- Il Media Annotations Working Group ha pubblicato la prima Public Working Draft di Use Cases e Requirements per Ontology e API per Media Object 1.0. Questo documento specifica i casi d'uso e i requisiti come input per lo sviluppo dell'"Ontology for Media Object 1.0" e l'"API for Media Object 1.0". L'ontologia sarà una semplice ontologia per supportare l'integrazioni di dati cross-community di informazioni collegate a media object sul Web. L'API fornirà accesso in lettura e potenzialmente in scrittura ai media objects, facendo affidamento sulle definizioni dell'ontologia. Ulteriori informazioni su Video in the Web Activity.
