--- title: "Call for Review: Web Security Context: Pubblicato User Interface Guidelines Proposed Recommendation" date: 2010-06-22 categories: ---

Il Web Security Context Working Group ha pubblicato una Proposed Recommendation di Web Security Context: User Interface Guidelines. Questa specifica riguarda le decisioni di fiducia ("trust") che gli utenti devono effettuare online, e i modi per aiutarli a prendere decisioni sicure e informate dove possibile. Al fine di raggiungere questo obiettivo, questa specifica include raccomandazioni sulla presentazione di informazioni identificative da parte degli utenti nonché raccomandazioni nel caso di errore nei protocolli di sicurezza. Commenti sono benvenuti fino al 20 luglio. Per saperne di più si veda Security Activity.
