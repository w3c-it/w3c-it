--- title: "Note: Rich Web Application Backplane" date: 2006-11-16 layout: page categories: --- L'Hypertext Coordination Group ha rilasciato Rich Web Application Backplane come Coordination Group Note. La nota descrive infrastrutture comuni per linguaggi di programmazione Web dichiarativi ed imperativi. La particella elementare comune per le applicazioni Web come submissioni, modelli di dati, fissagigo e comportamento di model-view, e componenti Web, può anche essere utilizzata per formati multipli di markup. Maggiori informazioni sulla pagina dell' Hypertext Coordination Group. (Archivio News inglese, italiano)
