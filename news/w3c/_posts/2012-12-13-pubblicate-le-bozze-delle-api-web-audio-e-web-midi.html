Pubblicate le bozze delle API Web Audio e Web MIDI

13 dicembre 2012

L' Audio Working Group ha pubblicato due bozze di lavoro oggi:

  • API Web Audio . Questa specifica descrive API JavaScript di alto livello per processare e sintetizzare audio in applicazioni Web. Il paradigma principale è un grafo di routing audio, dove oggetti AudioNode sono interconnessi in modo da definire il rendering audio complessivo. Il processo effettivo avra' luogo principalmente nell'implementazione sottostante (tipicamente Assembly ottimizzato/ C / C++),  ma sono anche supportati processing JavaScript diretto e sintesi.
  • API Web MIDI . Questa specifica descrive API che supportano il protocollo MIDI, consentendo alle applicazioni Web di enumerare e selezionare dispositivi MIDI input e output sul sistema client e spedire e ricevere messaggi MIDI. Lo scopo é quello di abilitare applicazioni MIDI non musicali analogamente a quelle musicali, fornendo accesso di basso livello ai dispositivi MIDI disponibili sul sitema dell'utente. Allo stesso tempo, gli API Web MIDI non intendono diventare una piattaforma di controllo semantico; sono progettati per esporre la meccanica delle interfacce input e output e le modalità di invio e ricezione di messaggi MIDI, senza identificare il significato semantico di queste azioni.
