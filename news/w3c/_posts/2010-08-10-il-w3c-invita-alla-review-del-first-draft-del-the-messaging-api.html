--- title: "Il W3C Invita alla Review del First Draft del The Messaging API" date: 2010-08-10 layout: page categories: w3c ---

Il Device APIs and Policy Working Group ha pubblicato un First Public Working Draft di The Messaging API. Il Messaging API definisce un'interfaccia high-level per la funzionalità Messaging, inclusi SMS, MMS ed Email. Esso include APIs per creare, inviare e ricevere messaggi. Per saperne di più si legga Ubiquitous Web Applications Activity.
