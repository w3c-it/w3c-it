--- title: "SOAP su Java Message Service 1.0 Prima Draft Pubblicata" date: 2008-07-24 categories: --- Il SOAP-JMS Binding Working Group ha pubblicato la prima Working Draft pubblica di SOAP over Java Message Service 1.0. Il lavoro descritto in questo e nei documenti associati sono rivolti ad un insieme di standard per il trasporto dei messaggi SOAP su Java Message Service (JMS). Lo scopo principale è assicurare l'interoperabilità fra le implentazioni di diversi distributori di Web services. Dovrebbe anche permettere ai clienti ad implementare i loro Web services per parte della loro infrastruttura, e permettere di interoperare con i Web services forniti dai distributori. Questo documento specifica i legami sia per SOAP 1.1 che per SOAP 1.2, usando SOAP 1.2 Protocol Binding Framework. Ulteriori informazioni su Web Services Activity. (Archivio News Inglese, Italiano)
