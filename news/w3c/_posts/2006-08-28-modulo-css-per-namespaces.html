--- title: "Modulo CSS per Namespaces" date: 2006-08-28 layout: page categories: --- Il CSS Working Group ha riasciato un Working Draft aggiornato di CSS Module: Namespaces. La regola @namespace è utilizzata per dichiarare il namespace di default e per collegare i namespace ai prefissi. La sintassi definisce che altre specifiche possano essere adottate, per l'uso di quei prefissi in nomi namespace-qualified. Maggiori informazioni sulla CSS home page. (Archivio News inglese, italiano)
