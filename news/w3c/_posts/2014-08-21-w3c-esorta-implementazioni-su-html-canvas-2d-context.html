--- title: "W3C Esorta implementazioni su HTML Canvas 2D Context" date: 2014-08-21 categories: ---

L' HTML Working Group esorta implementazioni della Candidate Recommendation su HTML Canvas 2D Context. Questa specifica definisce il 2D Context per l'elemento canvas dell'HTML. Il 2D Context fornisce oggetti, metodi e proprietà per disegnare e manipolare oggetti grafici su una superficie da disegno (canvas). Per saperne di più vai a HTML Activity.
