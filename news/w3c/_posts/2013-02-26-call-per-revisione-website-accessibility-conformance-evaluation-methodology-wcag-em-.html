--- title: "Call per Revisione: Website Accessibility Conformance Evaluation Methodology (WCAG-EM) " date: 2013-02-26 layout: page categories: w3c ---

Oggi il Web Content Accessibility Guidelines Working Group (WCAG WG) e il Evaluation and Repair Tools Working Group (ERT WG) hanno pubblicato un aggiornamento del Working Draft sulla Website Accessibility Conformance Evaluation Methodology (WCAG-EM) 1.0, tramite l'iniziativa comune WCAG 2.0 Evaluation Methodology Task Force (Eval TF). WCAG-EM descrive un approccio per valutare come i siti Web, incluse le applicazioni Web e i siti per dispositivi mobili, si conformano al WCAG 2.0. Per saperne di più vai a WCAG-EM Overview e Web Accessibility Initiative (WAI).
