--- title: "Aggiornato il Candidate Recommendation per CSS Values e Units Module Livello 3 " date: 2013-07-30 categories: --- Il Cascading Style Sheets (CSS) Working Group invita all'implementazione del Candidate Recommendation del CSS Values e Units Module Livello 3. La nuova versione riflette aggiornamenti minori. Questo modulo CSS3 descrive i valori comuni e le unità che le proprietà CSS accettano e la sintassi usata per desciverli nelle definizioni delle proprietà CSS. Per saperne di più su Style Activity.
