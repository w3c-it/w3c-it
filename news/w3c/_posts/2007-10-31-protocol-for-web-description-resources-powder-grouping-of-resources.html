--- title: "Protocol for Web Description Resources (POWDER): Grouping of Resources" date: 2007-10-31 layout: page categories: --- Il Protocol for Web Description Resources (POWDER) Working Group ha pubblicato il First Public Working Draft di Protocol for Web Description Resources (POWDER): Grouping of Resources. Il Protocol for Web Description Resources (POWDER) facilita la pubblicazione di descrizioni di risorse multiple, quali tutte quelle disponibili su un sito Web. Questo documento descrive come insiemi di risorse possano essere definiti, o per l'uso in Description Resources oppure in altri contesti. Maggiori informazioni sulla pagina della Semantic Web Activity. (Archivio News inglese, italiano)
