--- title: "Efficient XML Interchange (EXI) Format: Working Draft" date: 2007-07-16 layout: page categories: --- L'Efficient XML Interchange Working Group ha pubblicato il First Public Working Draft di Efficient XML Interchange (EXI) Format 1.0. EXI è una rappresentazione molto compatta per l'eXtensible Markup Language (XML) Information Set, pensato per ottimizzare simultaneamente le performance e per l'utilizzo di risorse computazionali. Utilizzando un algoritmo relativamente semplice ed un piccolo set di tipi di dati, produce, in modo affidabile, codifiche efficienti di flussi di eventi XML.
