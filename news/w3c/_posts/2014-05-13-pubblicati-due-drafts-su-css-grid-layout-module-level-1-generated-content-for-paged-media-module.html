--- title: "Pubblicati due Drafts su CSS: Grid Layout Module Level 1, Generated Content for Paged Media Module" date: 2014-05-13 categories: ---

Il Cascading Style Sheets (CSS) Working Group ha pubblicato oggi due Working Draft:

  • CSS Grid Layout Module Level 1. Questo modulo CSS definisce un sistema di layout bidimensionale basato su una griglia, ottimizzato per il design di interfacce utente. Nel modello di layout a griglia, il figlio di un grid container può essere posizionato all'interno di slot arbitrari in un layout a griglia fisso o flessibile.
  • CSS Generated Content for Paged Media Module. I libri e altri media a pagine usano spesso delle tecniche speciali per visualizzare l'informazione. I contenuti possono essere mossi in o generati per aree speciali della pagina, come le intestazioni o le note a pie' di pagina. Contenuti generati all'interno delle pagine, come i tab leader o i rimandi, aiutano i lettori a navigare all'interno delle e tra le pagine.

Il CSS è un linguaggio per la descrizione ed il rendering di documenti strutturati (come HTML e XML) su schermo, su carta, in parlato, ecc.. Per ulteriori informazioni consultate Style Activity.
