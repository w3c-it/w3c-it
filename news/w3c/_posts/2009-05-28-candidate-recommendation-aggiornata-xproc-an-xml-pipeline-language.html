--- title: "Candidate Recommendation aggiornata: XProc: An XML Pipeline Language" date: 2009-05-28 layout: page categories: --- L'XML Processing Model Working Group ha pubblicato una Candidate Recommendation aggiornata di XProc: An XML Pipeline Language. Questa specifica descrive la sintassi e la semantica di un linguaggio per descrivere operazioni sui documenti XML. La sezione status del documento riassume la lista di cambiamenti dalla Candidate Recommendation precedentemente pubblicata. Ulteriori informazioni su Extensible Markup Language (XML) Activity. (Archivio News Inglese, Italiano)
