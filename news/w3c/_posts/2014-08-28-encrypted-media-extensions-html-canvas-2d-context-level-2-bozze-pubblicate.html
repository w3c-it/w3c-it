--- title: "Encrypted Media Extensions; HTML Canvas 2D Context, Level 2 Bozze Pubblicate" date: 2014-08-28 layout: page categories: w3c ---

L' HTML Working Group ha pubblicato oggi due documenti:

  • Una Bozza di Lavoro su Encrypted Media Extensions. Tale proposal estende HTMLMediaElement per fornire le APIs per controllare la riproduzione di contenuto protetto. L'API supporta i casi d'uso che vanno dalla semplice decrittografia di chiave al video di alto valore (stante un'implementazione di user-agente appropriato).Lo scambio Licenza /chiavi è controllato dall'applicazione. Questa specifica non definisce un sistema di protezione del contenuto o un sistema di Digital Rights Management. 
  • Una Bozza pubblica su HTML Canvas 2D Context, Level 2. Questa specifica definisce il 2D-context  per l'elemento canvas HTML. Il context 2D fornisce oggetti, metodi e proprietà per disegnare e manipolare la grafica su una superficie di disegno.

Per saperne di più vai a HTML Activity.
