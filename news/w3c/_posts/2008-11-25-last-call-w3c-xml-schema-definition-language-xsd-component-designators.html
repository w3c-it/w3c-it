--- title: "Last Call: W3C XML Schema Definition Language (XSD): Component Designators" date: 2008-11-25 layout: page categories: w3c --- L'XML Schema Working Group ha pubblicato la Last Call Working Draft di W3C XML Schema Definition Language (XSD): Component Designators. XML Schema: Component Designators definisce uno chema per individuare componenti di XML Schema come specificato da XML Schema Part 1: Structures e XML Schema Part 2: Datatypes. I commenti sono bene accetti fino al 19 gennaio. Ulteriori informazioni su Extensible Markup Language (XML) Activity. (Archivio News Inglese, Italiano)
