Il W3C Pubblica le Best Practices per Creare Applicazioni Mobile Web più intelligenti

In inglese | W3C Press Release Archive | Archivio dei comunicati stampa tradotti in italiano

http://www.w3.org/ -- 14 Dicembre 2010 -- Il W3C ha oggi annunciato uno standard che renderà più facile per sviluppatori e provider di contenuti creare applicazioni mobile Web. Il Mobile Web Application Best Practices, pubblicato come una W3C Recommendation, offre consigli pratici di molte persone interessate al mobile Web per un facile sviluppo e per la distribuzione di applicazioni del mobile Web che funzionano su diverse piattaforme.

“Sono veramente entusiasta di questo documento, che ho già usato per garantire che il W3C Cheat Sheet, un'applicazione mobile Web, funzioni non solo sui dispositivi mobili, ma su qualsiasi dispositivo”, ha dettoDominique Hazael-Massieux,Responsabile della W3C Mobile Web Initiative. Queste linee guida includono saggi consigli provenienti dal mondo reale di sviluppatori di applicazioni Web, aziende di telecomunicazioni, e sviluppatori di browser.

Le Applicazioni Mobile Web avranno un futuro splendente

I designer di applicazioni danno molto valore alla capacità di scrivere del codice una volta sola e di riutilizzarlo in ambienti multipli. La rapida crescita del mercato delle applicazioni mobile ha aumentato la tendenza ad usare il Web come una piattaforma di sviluppo su questo tipo di dispositivi; questo punto è dimostrato in una relazione tecnica da un analista del GIA. Le applicazioni Web stanno già sostituendo le applicazioni native in diversi computer, e si spera di raggiungere nel prossimo futuro una tendenza simile nei dispositivi mobili dal momento che la piattaforma Web risolve i problemi di frammentazione spesso familiari agli sviluppatori di dispositivi mobili. Il Web rende anche più veloce e più facile implementare e aggiornare le applicazioni senza la necessità di alcun intervento da parte dell'utente, e permette un' integrazione trasparente di servizi attraverso la rete (cloud-based). Gli utenti, inoltre, potranno riconoscere il valore delle applicazioni Web-based.

Linee guida chiare per la piena integrazione nella piattaforma Open Web del W3C


The Mobile Web Application Best Practices Cards summarize the Guidelines

Con i lavori attualmente in corso nei Working Group di HTML5, CSS3, SVG, e WAI-ARIA, il Web sta recuperando rapidamente terreno nella realizzazione di un'interfaccia utente superiore. Questo lavoro di squadra promuove lo sviluppo di innovazioni e nuove applicazioni che possono essere visualizzate solo in una piattaforma Web. Il W3C sta creando una piattaforma ancora più potente per gli sviluppatori, che include una migliore integrazione di hardware e sensori. I Working Group di Web Applications, Web Notifications, Web Events, Device APIs e Policy, e Geolocation stanno sviluppando una gran quantità di API JavaScript che ridurranno il divario tra il Web e le applicazioni native nei prossimi mesi e anni.

Le Mobile Web Application Best Practices offrono una guida su quali tecnologie Web sono particolarmente rilevanti sui dispositivi mobili. Le linee guida indicano anche come progettare appplicazioni web che si adattano al loro contesto d'uso, senza sovraccaricare la rete e ottimizzando i tempi di risposta per rafforzare significativamente l'esperienza generale dell'utente nel Web mobile.

Il World Wide Web Consortium

Il World Wide Web Consortium (W3C) è un consorzio industriale internazionale in cui i membri, uno staff a tempo pieno e il pubblico lavorano congiuntamente per sviluppare gli standard Web. Il W3C persegue la sua missione essenzialmente mediante la creazione di standard Web e linee guida progettate per assicurare la crescita a lungo termine del Web. Oltre 300 organizzazioni sono Membri del Consorzio. Il W3C è guidato congiuntamente dal MIT Computer Science and Artificial Intelligence Laboratory (MIT CSAIL) negli USA, European Research Consortium for Informatics and Mathematics (ERCIM) con sede amministrativa in Francia e Keio University in Giappone, ed ha Uffici in tutto il mondo.
