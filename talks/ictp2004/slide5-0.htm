The Recommendation Track

Driven by the consensus principle

  • setting up a Working Group (members and possibly external experts only)
  • Working Draft (technical document)
  • Last Call Working Draft (public technical document)
  • Candidate Recommendation ("Call for implementations")
  • Proposed Recommendation (following implementation experience)
  • W3C Recommendation (final and stable document)

W3C is not a standard organization, but a Recommendation has:

  • proof of the concept
  • proof of implementation

W3C Recommendations are:

  • technical specifications agreed by international groups of experts
  • de facto standards not deriving from dominant position in the market
