Web Services: role of ontologies

  • This usage scenario assumes that all the services are using common concepts (e.g. flight, economy class, room, etc)
  • For the travel agent service to understand the airline services and to be able to send meaningful information to them, a travel industry ontology needs to exist and be used by the Web services taking part in this scenario
  • An ontology is a formal description of a set of concepts and their relationships to each other
  • By associating a name with each concept, an ontology defines a standard vocabulary that can be used to communicate those concepts
