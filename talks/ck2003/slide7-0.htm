Searching the web

The web is probably the most used knowledge repository

Searching the web is a pain:

  • many search engines
  • lacking semantics both in indexing (<meta> tags often ignored, spiders can't capture the semantics) and in searching (keyword searching and context operators)

Richer expressions allow to distinguish between:

  • aspirine cures headache
  • aspirine causes headache
documents when searching for:
aspirine AND headache

Can we trust in the information coming from the web?
