The levels in the Semantic Web ...

  • Basic role played by XML
  • RDF and RDF Schema describe Resources and their types
  • Ontologic level:
    • describes relationships between element types
    • non information about computational usage of these relationships
  • Digital signature:
    • until now "coarse grained"
    • "fine grained" can be used to authenticate ontologies, inferences, data
  • Logical level:
    • derive new knowledge from existing assertions
    • present deductive systems are not interoperable
    • a possibility: an universal language to represent demonstrations, that can be exported on the web with their digital signature
