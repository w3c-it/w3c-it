What is RDF?

  • Effective use of metadata requires appropriate conventions for:
    • semantics (defined by single disciplinary communities)
    • syntax (organization of data elements for automatic processing)
    • structure (formal constraint on the syntax)
  • RDF:
    • basis for coding, exchanging and reusing structured metadata
    • allows interoperability among applications exchanging machine-understandable information on the web
  • RDF is for the Semantic Web what HTML has been for the web
