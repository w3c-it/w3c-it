RDF Data Model RDF: properties

Directed relations between two resources

  • A property is a specific aspect, characteristic, attribute, or relation used to describe a resource
  • Each property:
    • has a specific meaning

    defines:

    • its permitted values
    • the types of resources it can describe
    • its relationship with other properties
  • Each property is identified by a name, and takes some values
