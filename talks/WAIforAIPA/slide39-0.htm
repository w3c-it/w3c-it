MathML

  • La matematica compare ovunque nella documentazione tecnica
  • La matematica sul web è essenziale per i documenti digitali!
  • Le soluzioni attuali presentano dei problemi:
    • Utilizzare HTML (per es.., ai (a<sub>i</sub>))
      • HTML non è abbastanza ricco
      • La composizione di formule matematiche non è semplice!
    • Utilizzare embedded images (mediante un editor specializzato):
      • Difficile posizionare correttamente le immagini
      • Il contenuto della formula non è ricercabile
      • Problemi di accessibilità (come dovrebbe comportarsi un voice browser?)
  • Andrebbe convertita una gran mole di documenti esistenti (la maggior parte in TEX)
  • Per questo esiste MathML!
