Ontologie

  • Il Semantic Web ha bisogno del supporto delle ontologie:
    "definisce i concetti e le relazioni usate per rappresentare un' area di conoscenza"
  • Abbiamo bisogno di un Web Ontologies Language per definire:
    • la terminologia utilizzata in un contesto specifico
    • vincoli sulle proprietà
    • le caratteristiche logiche delle proprietà
    • l' equivalenza dei termini tra le varie ontologie
    • etc.
  • Il linguaggio deve essere un compromesso tra:
    • semantica ricca per applicazioni basate sulla conoscenza
    • fattibilità

Questo è il Web Ontology Language (OWL)

  • Uno strato sopra RDFS con alcune ulteriori possibilità
  • Risultato di vari progetti (DAML, OIL)
