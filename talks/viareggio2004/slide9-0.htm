W3C e accessibilità

"La potenza del web risiede nella sua universalità.
L'accesso per tutti indipendentemente dalle disabilità ne costituisce un aspetto essenziale."
(Tim Berners-Lee, Direttore del W3C e inventore del World Wide Web)
  • Web Content Accessibility Guidelines (WCAG 1.0 Recommendation, 1999)
    Creare contenuti Web accessibili
  • Authoring Tool Accessibility Guidelines ( ATAG 1.0 Recommendation, 2000)
    Progettare strumenti di authoring utilizzabili da autori disabili, e che producono contenuti Web accessibili
  • User Agent Accessibility Guidelines (Browser, UAAG 1.0 Recommendation, 2002)
    Come migliorare l' accessibilità dei mainstream browser a dei multimedia player per consentire un migliore accesso alle risorse Web per le persone affette da disabilità uditive, cognitive, visive, fisiche

Oreste Signore, Manager of W3C Office in Italy
