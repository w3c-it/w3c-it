WAI (Web Accessibility Initiative)

Tre linee guida:
  • Web Content Accessibility Guidelines
    (WCAG 1.0, 1999, Recommendation)
    Illustra ai creatori di pagine Web come creare contenuti Web accessibili
    e ora ...
    WCAG 2.0 (Working Draft)
  • Authoring Tool Accessibility Guidelines
    (ATAG 1.0, 2000, Recommendation)
    Illustra agli sviluppatori come progettare strumenti di authoring utilizzabili da autori portatori di handicap, e che producono contenuti Web accessibili
  • User Agent Accessibility Guidelines
    (Browser, UAAG 1.0, 2002, Recommendation)
    Illustra cosa devono fare gli sviluppatori di software per migliorare l' accessibilità dei mainstream browser a dei multimedia player per consentire un migliore accesso alle risorse Web per le persone affette da handicap uditivi, cognitivi, visivi, fisici
