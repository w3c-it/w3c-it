Linguaggi XML per l' architettura XML

  • RDF - per descrivere le risorse sul Web
  • Xlink - una sintassi per i link
  • XSLT - un linguaggio di trasformazione
  • XML Schema (definizione dei linguaggi)
  • Xforms - moduli (forms) per l' interazione, con separazione di scopo e presentazione
  • XML Events - interazioni (clic, pressione di un tasto, ...)
