La trasformazione delle informazioni

  • Si ha sempre un doppio processo:
    1. trasformazione dal formato interno a quello da utilizzare nella distribuzione
    2. ulteriore trasformazione per la restituzione (rendering)
  • Meglio legare i dati alla forma finale il più tardi possibile
  • Nella trasformazione di un documento XML:
    • trasformazione dell' istanza di vocabolario XML in una nuova istanza coerente con il vocabolario della semantica di restituzione
    • formattazione dell' istanza del vocabolario di restituzione nello user agent
