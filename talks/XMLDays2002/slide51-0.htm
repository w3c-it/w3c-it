W3C e Web Services

  • XML, XML Namespace, XML Schema sono strumenti utili per fornire meccanismi per scambiarsi informazione strutturata in un ambiente distribuito
  • W3C fornisce una piattaforma di discussione per la pianificazione e creazione di una varietà di tecnologie per progettare Web service
  • Web Service Activity lanciata nel gennaio 2002, proseguendo ed estendendo il lavoro del XML Protocol Working Group (SOAP ).

    E' composta da:

    • Web Services Architecture Working Group (identificazione delle tecnologie)
    • XML Protocol Working Group
    • Web Services Description Working Group (linguaggio per descrivere le interfacce con i Web Service e come interagire con essi)
    • Web Services Coordination Group (coordinamento dei gruppi e con l' attività Semantic Web)
