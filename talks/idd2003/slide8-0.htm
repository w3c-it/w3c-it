La W3C Recommendation Track

  • Sono il risultato di un processo cooperativo
    • costituzione di un Working Group (aperto solo ai membri ed eventualmente a esperti qualificati)
    • Working Draft (primo documento tecnico, sottoposto a revisioni e raffinamenti nel WG)
    • Last Call Working Draft (rapporto tecnico pubblico)
    • Candidate Recommendation (prevede una "Call for implementation")
    • Proposed Recommendation (dopo adeguata esperienza implementativa)
    • W3C Recommendation (documento definitivo, distribuito al vasto pubblico)
  • Quindi:
    • proof of the concept
    • proof of implementation
  • Le W3C Recommendation sono standard internazionali?
