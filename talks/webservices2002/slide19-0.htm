Un esempio di Web service

Travel agent usage scenario
  • L' agente di viaggio vende pacchetti vacanza
  • Utilizza altri Web Service: WS carta di credito, WS hotel, WS linea aerea
  • Nota: l' agente di viaggio è un operatore umano, tutti gli altri elementi sono applicazioni
I vantaggi
  • Costi inferiori per le nuove linee aeree, hotel, compagnie che gestiscono carte di credito, per vendere i loro prodotti
  • L' agente di viaggio ha un ventaglio di possibilità più ampio
  • I clienti ricevono un servizio migliore
