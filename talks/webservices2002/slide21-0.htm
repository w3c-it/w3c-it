Il Web ha alcuni principi architetturali

Insieme di risorse che sono:

  • identificate (URI)
  • reperite mediante certe regole (es. HTTP GET)
  • strutturate (es. XML)

Le tecnologie per i Web Service devono essere progettate in modo che siano consistenti con il web
