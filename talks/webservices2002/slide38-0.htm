Il Web Services Description Working Group

  • Descrivere l' interfaccia, cioè l' aspetto esterno mediante il quale comunicano le applicazioni (Web Service e User Agent)
  • I componenti dell' interfaccia:
    • messaggio

      definizione delle strutture dati e dei loro tipi

    • modello di scambio del messaggio

      dscrizione delle sequenze di operazioni supportate dal Web Service

    • protocol binding

      meccanismo per trasportare informazione XML-based su un protocollo sottostante
