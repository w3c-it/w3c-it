XForms

  • A recent W3C Recommendation (14th October)
  • A case for tecnological interoperability towards different devices (device independence) and integration in web services
  • The new generation of the 10 years old HTML Forms
  • Separates
    • description of the purpose of the form
    • presentation of the form
    • how the results (instance data) are written in XML
  • Advantages:
    • reuse
    • device independence
    • accessibility
  • XML at the core of XForm model
  • Several implementations yet available
  • Explicitly referenced in the "United Kingdom e-government interoperability framework"
  • XForms Basic (Candidate Recommendation) is the XForms mobile profile
    (Native implementation of XForms on mobile devices will be possible)
