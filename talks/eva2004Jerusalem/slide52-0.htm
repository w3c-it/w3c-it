RDF Data Model: resources

  • A resource is any thing that can be described using a RDF expression
  • A resource can be:
    • a web page
    • a part of a web page
    • a XML element in the source document
    • a web pages collection
    • an object that can't be accessed directly on the web (book, painting, object, ...)
  • Any resource is idenfied by a Uniform Resource Identifier (URI)
