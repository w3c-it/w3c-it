  • Never forget the basic principles of the World Wide Web :
    • Universal Information Space
    • Interoperability
    • Trust
    • Universal Access
    • Decentralization
  • Adaptive and intelligent systems:
    • agents
    • information filtering
    • user profiling
  • Document structure:
    • light tagging
    • weighted information items
    • weighted and typed links
  • User interface must support:
    • several metaphors (mapp, time, classification)
    • query formulation
    • dynamic reconfiguration of content
    • accessibility
    • usability
