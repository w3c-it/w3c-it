Searching vs. linking

  • Big emphasis on searching
  • Users don't use "advanced search" facilities
  • A much more common behaviour:
    • a generic search
    • browsing result set
    • once an interesting node has been found ...
    • follow the links pertinent to his/her interests
  • Much similar to the process in finding scientific literature
  • Much more effective if can take advantage from the stored knowledge:
    • links are semantically typed
    • user interests can be formalized
    • an intelligent software agent is capable of reasoning about data and follow the pertinent paths
