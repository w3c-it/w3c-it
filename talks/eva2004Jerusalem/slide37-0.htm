From HTML to XML

  • HTML (V1):
    • was one language
    • was easy to learn and write by hand
    • allowed images to be include in text
    • allowed hyperlinks to be put onto text or images.
    • combined content and presentation
    • had no data typing, no continuous media, no clear semantics
  • XML
    • New Data-centric Web applications emerging
    • Better tag semantics for specialist information
    • Transaction processing using the Web
    • Database exchange
    • Different views of the same data
    • Customisation of information by agents
    • Management of document collections
    • International publishing: platform, language and media independent
