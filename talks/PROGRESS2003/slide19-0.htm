HTML was the past

  • Remember:
    • it was one language
    • it was easy to learn and write by hand
    • it allowed images to be include in text
    • it allowed hyperlinks to be put onto text or images.
  • Example - an address list:
     <html/>
        <head/>
        </head/>
        <body/>
                <h2/> Ivan Herman </h2/>
                        <p/>
                                W3C Offices, CWI, Netherlands
        </body/>
</html/>

    which is displayed as:

Ivan Herman

W3C Offices, CWI, Netherlands

  • HTML V1:
    • combined content and presentation
    • had no data typing, no continuous media, no clear semantics
