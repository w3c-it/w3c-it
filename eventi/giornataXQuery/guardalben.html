Bridging Relational Technology and XML

Giovanni Guardalben
Hit Software

Abstract

E' una review dello stato dell'arte in quattro settori:
  1. Techniques for Storing/Modeling XML in Relational Repositories
  2. Materializing Relational Data as XML Documents
  3. Querying XML Views of Relational Data
  4. XML for Data Integration (Mapping Technologies)

Nell'ambito di questi settori vengono descritte le funzionalità di Allora e come si colloca rispetto alle altre proposte.
