Giuseppe Amato

Abstract

Processing XML queries mainly involves matching the structure of the queries against the structure of XML documents in addition to content matching. Techniques developed for traditional databases, such as relational database systems, have shown high inefficiency when dealing with hierarchical structures. In fact, path traversal and containment queries are translated into long sequences of joins, which make simple query processing impracticable. In addition, boolean retrieval, both for structure and content match, is also not sufficient in this context. Users should be provided with query languages that offer the possibility of expressing uncertainty and to retrieve information also by similarity. We will discuss techniques developed at CNR for efficient XML query processing and for XML search by similarity.
